# 💊 Health & Supplement Writing Blueprint
## ปรับสูตร BestWinbiz → สายสุขภาพ & อาหารเสริม
### Version: กระชับ พอดีอ่านจบ ครอบคลุม Key Points

---

## 🔄 PART 1: สิ่งที่ต้องปรับ vs สิ่งที่คงไว้

### ✅ คงไว้เหมือนเดิม (DNA หลัก)

| สิ่งที่คง | เหตุผล |
|---|---|
| 1 บรรทัด = 1 ความคิด | อ่านง่ายบนมือถือ |
| เส้นคั่น ━━━ ทุก 10-20 บรรทัด | แบ่งส่วนชัด |
| (วงเล็บ Insight) | คมๆ คนอยากแชร์ |
| [ คำคมในกรอบ ] | หยุดอ่าน + แคปหน้าจอ |
| ไม่ใช้ Emoji | รักษาภาพ Authority |
| "ครับ" ทุกประโยค | สุภาพ น่าเชื่อถือ |
| Before vs After | พิสูจน์ผลลัพธ์ |
| Reframe ปัญหา | พลิกมุมมอง |
| Soft Sell ไม่กดดัน | ไม่ดูขายของ |

### 🔧 สิ่งที่ต้องปรับ

| สิ่งที่ปรับ | จากเดิม (ธุรกิจ) | ปรับเป็น (สุขภาพ) |
|---|---|---|
| **Persona** | พี่ชายนักธุรกิจ | พี่/คนรู้จักที่ดูแลสุขภาพจริง |
| **Audience** | เจ้าของกิจการ | คนวัยทำงาน 30-55 ที่ห่วงสุขภาพ |
| **Pain Point** | ขายไม่ดี ลูกค้าต่อราคา | เหนื่อยง่าย นอนไม่หลับ ผิวโทรม |
| **ความยาว** | 150-200 บรรทัด | **80-120 บรรทัด** ← กระชับลง 40% |
| **จำนวนเส้นคั่น** | 8-12 เส้น | **5-7 เส้น** |
| **จำนวน Maxim** | 2-4 ชุด | **1-2 ชุด** |
| **Application Steps** | 5-7 ข้อ | **3-5 ข้อ** |
| **Hashtag ท้ายโพสต์** | 8-15 ตัว | **5-8 ตัว** |
| **ศัพท์อังกฤษ** | Business terms | Health/Science terms |
| **Analogy** | หมอ/สร้างบ้าน | ร่างกายเหมือนรถ/แบตเตอรี่ |

## 🎭 PART 2: Persona & Tone สำหรับสายสุขภาพ

### 2.1 ตัวตนที่ต้องสร้าง

```
❌ ไม่ใช่: หมอ (ผิดกฎหมายถ้าไม่ใช่หมอจริง)
❌ ไม่ใช่: พ่อค้าอาหารเสริม (ดูขายของ)
❌ ไม่ใช่: Influencer รีวิว (ดูโฆษณา)

✅ คือ: "คนที่ศึกษาเรื่องสุขภาพจริงจัง 
       และแชร์สิ่งที่ตัวเองเรียนรู้มา"
✅ คือ: "เพื่อนที่ดูแลตัวเองดี แล้วอยากบอกต่อ"
```

### 2.2 Tone ที่ปรับ

| ช่วง | Tone เดิม (ธุรกิจ) | Tone ใหม่ (สุขภาพ) |
|---|---|---|
| **Hook** | ท้าทาย กระตุ้น | **ห่วงใย + ชี้ความจริง** |
| **Pain** | จี้จุดเจ็บธุรกิจ | **เข้าใจความลำบาก + สะท้อนชีวิตจริง** |
| **Reframe** | ตรงๆ มั่นใจ | **อ่อนโยน + ให้ข้อมูล** |
| **สอน** | ผู้รู้ สั่งสอน | **แชร์ประสบการณ์ + อ้างข้อมูล** |
| **สรุป** | หนักแน่น | **ให้กำลังใจ + ชวนลอง** |
| **CTA** | เชิญชวนติดตาม | **ชวนดูแลตัวเอง + เปิดทาง** |

### 2.3 🗣️ Tone v2: ภาษาพูดบ้านๆ (แนะนำ)

```
หลักการ: พูดตรงไปตรงมา กึ่งภาษาพูด เน้นคำบ้านๆ
         เข้าใจง่าย อ่านง่าย เห็นภาพทันที ไม่ต้องแปลซ้ำ
```

**ตาราง "คำทางการ → คำบ้านๆ":**

| ❌ คำทางการ/ซับซ้อน | ✅ คำบ้านๆ ตรงไปตรงมา |
|---|---|
| อ่อนเพลียเรื้อรัง | เหนื่อยทั้งวัน พักก็ไม่หาย |
| ได้รับไม่เพียงพอ | ได้ไม่พอ / ขาด |
| มีส่วนเกี่ยวข้อง | ร่างกายใช้มัน… |
| ส่งสัญญาณเตือน | ร่างกายกำลังบอกอะไรอยู่ |
| ผลลัพธ์อาจแตกต่าง | แต่ละคนไม่เหมือนกันนะครับ |
| ร่างกายกำลังส่งสัญญาณเตือนบางอย่าง | นี่ไม่ใช่เรื่องปกติครับ |
| จากการศึกษาของ NIH | ข้อมูลจาก NIH บอกว่า... |
| ควบคุมระบบประสาท | ช่วยให้สมองสงบ |
| กระบวนการ | อย่าง / เรื่อง |

**เทคนิคให้ภาษาบ้านๆ:**
```
✅ ใช้ "พูดง่าย ๆ คือ" ก่อนอธิบายเรื่องซับซ้อน
✅ ใช้ "ผมพูดตรง ๆ เลยนะครับ" เปิดเรื่อง
✅ ใช้ "ลองคิดตามนะครับ" ก่อนชวนให้เห็นภาพ
✅ ใช้ "มาเช็คกันเลยครับ" แทน "ลองสังเกตดูนะครับ"
✅ ใช้ "พอขาด" แทน "เมื่อร่างกายได้รับไม่เพียงพอ"
✅ ตัดคำฟุ่มเฟือยออก เหลือแต่แก่น
❌ อย่าใช้ภาษาวิชาการยาวๆ โดยไม่แปลเป็นคำง่าย
❌ อย่าใช้ศัพท์อังกฤษที่คนทั่วไปไม่รู้จัก โดยไม่อธิบาย
```

### 2.4 สรรพนาม & คำเรียก

```
ตัวเอง:    "ผม" + "ครับ" (ผู้ชาย) / "เรา" + "ค่ะ" (ผู้หญิง)
ผู้อ่าน:    "คุณ" / "เพื่อน ๆ" / "ใครที่..."
กลุ่มเป้า:  "คนที่ทำงานหนัก" / "คนวัย 30+" / "คนที่เริ่มรู้สึกว่าร่างกายไม่เหมือนเดิม"
```

## 👥 PART 3: Audience สายสุขภาพ

### 3.1 กลุ่มเป้าหมายที่ระบุในเนื้อหา

```
🎯 กลุ่มหลัก:
"คนวัยทำงานที่เริ่มรู้สึกว่าร่างกายส่งสัญญาณ"
"คนที่ดูแลตัวเองแต่ยังรู้สึกว่าไม่พอ"
"คนที่อยากเริ่มดูแลสุขภาพ แต่ไม่รู้จะเริ่มจากตรงไหน"

🎯 กลุ่มย่อย:
"คนนอนดึก ตื่นมาไม่สดชื่น"
"คนที่กินไม่ครบ 5 หมู่ทุกวัน"
"คนที่เริ่มมีปัญหาผิว ผม เล็บ"
"คนที่ป่วยบ่อย ภูมิไม่ค่อยดี"
"ผู้หญิงวัย 35+ ที่เริ่มเห็นความเปลี่ยนแปลง"
```

### 3.2 Pain Points สายสุขภาพ

```
😩 ระดับ Physical:
- เหนื่อยง่าย ทั้งที่ไม่ได้ทำอะไรหนัก
- นอนไม่หลับ หลับไม่สนิท ตื่นมาเพลีย
- ผิวหมองคล้ำ แก่กว่าวัย
- ป่วยบ่อย หายช้า
- ปวดตัว ปวดข้อ อ่อนเพลีย

😤 ระดับ Emotional:
- รู้สึกผิดที่ไม่ดูแลตัวเอง
- กลัวแก่ก่อนวัย
- อยากดูดี แต่ไม่มีเวลา
- ลองมาหลายอย่างแล้ว ไม่เห็นผล
- กลัวอาหารเสริมจะเป็นแค่หลอก
```

## 📝 PART 4: สูตรหลัก 3 แบบ (ปรับกระชับ)

### 🔴 Formula A: "สอน + เชื่อมสินค้า" (ใช้บ่อยที่สุด)

**ความยาว: 80-100 บรรทัด**

```
[1] HOOK: ชี้อาการที่คนเจอ (2-3 บรรทัด)
    ↓
[2] PAIN: อธิบายว่าทำไมถึงเป็นแบบนี้ (8-12 บรรทัด)
    ━━━━━━━━━━━━
[3] REFRAME: บอกว่าสาเหตุจริงไม่ใช่ที่คิด (8-12 บรรทัด)
    (วงเล็บ Insight)
    ━━━━━━━━━━━━
[4] KNOWLEDGE: ให้ความรู้ + อ้างข้อมูล (15-20 บรรทัด)
    ━━━━━━━━━━━━
[5] ACTION: สิ่งที่ทำได้ 3-5 ข้อ (10-15 บรรทัด)
    ━━━━━━━━━━━━
[6] BRIDGE: เชื่อมไปสินค้าเนียนๆ (8-12 บรรทัด)
    ━━━━━━━━━━━━
[7] SUMMARY + CTA (8-12 บรรทัด)
```

---

### 🔵 Formula B: "เคสจริง / ประสบการณ์ + สอน"

**ความยาว: 60-80 บรรทัด**

```
[1] HOOK: เล่าเคส / ประสบการณ์ (3-5 บรรทัด)
    ━━━━━━━━━━━━
[2] STORY: เล่ารายละเอียด Before (8-12 บรรทัด)
    ━━━━━━━━━━━━
[3] TURNING POINT: จุดเปลี่ยน (5-8 บรรทัด)
    ━━━━━━━━━━━━
[4] RESULT: ผลลัพธ์ After (8-10 บรรทัด)
    (วงเล็บ Insight)
    ━━━━━━━━━━━━
[5] LESSON: บทเรียนที่ได้ (5-8 บรรทัด)
[6] CTA (5-8 บรรทัด)
```

---

### 🟢 Formula C: "Soft Sell ขายตรง (แต่ไม่กดดัน)"

**ความยาว: 50-70 บรรทัด**

```
[1] HOOK: คำถาม / สถานการณ์ (2-3 บรรทัด)
    ━━━━━━━━━━━━
[2] PAIN + REFRAME: สั้นๆ (8-10 บรรทัด)
    ━━━━━━━━━━━━
[3] SOLUTION: แนะนำสินค้า + จุดเด่น (10-15 บรรทัด)
    [ Maxim ]
    ━━━━━━━━━━━━
[4] SOCIAL PROOF: รีวิว / ตัวเลข (5-8 บรรทัด)
    ━━━━━━━━━━━━
[5] OFFER + CTA (8-12 บรรทัด)
```

## 📖 PART 5: คลังคำศัพท์สายสุขภาพ

### 5.1 คำเปิดเรื่อง (Hook Words)

```
"ถ้าคุณเริ่มรู้สึกว่า..."
"ลองสังเกตตัวเองดูนะครับ"
"หลายคนไม่รู้ว่า..."
"เคยสงสัยไหมครับว่า ทำไม..."
"ร่างกายส่งสัญญาณมาตลอด แต่เราอาจไม่ได้ฟัง"
"นี่คือสิ่งที่ผมเพิ่งเข้าใจเรื่อง #สุขภาพ"
```

### 5.2 คำ Reframe (พลิกมุม)

```
"ปัญหาไม่ได้อยู่ที่ [อาการ] แต่อยู่ที่ [สาเหตุลึก]"
"หลายคนแก้ที่ปลายเหตุ แต่ยังไม่ได้แก้ที่ต้นตอ"
"ไม่ใช่ว่าร่างกายอ่อนแอ แต่เราอาจให้สิ่งที่มันต้องการไม่ครบ"
"เราไม่ได้ขาดยา เราขาดสารอาหารที่ร่างกายใช้ทุกวัน"
"ตรงนี้แหละครับที่หลายคนมองข้าม"
```

### 5.3 คำอธิบาย (Knowledge Words)

```
"งานวิจัยจาก [แหล่ง] พบว่า..."
"ร่างกายเราต้องการ [สารอาหาร] วันละ..."
"ถ้าเปรียบง่าย ๆ มันเหมือน..."
"พูดง่าย ๆ คือ"
"นี่คือเหตุผลว่าทำไม..."
```

### 5.4 คำเชื่อมไปสินค้า (Bridge Words)

```
"นั่นคือเหตุผลที่ผมเลือก [สินค้า]"
"ผมลองมาหลายตัว แต่ตัวที่เห็นผลจริงคือ..."
"ถ้าถามว่าผมดูแลตัวเองยังไง..."
"สิ่งที่ผมทำทุกเช้าตอนนี้คือ..."
"ผมไม่ได้บอกว่าต้องกิน [สินค้า] แต่ถ้าใครสนใจ..."
```

### 5.5 คำปิด (Closing Words)

```
"สุขภาพดี ไม่ได้เริ่มจากวันที่ป่วย แต่เริ่มจากวันที่เลือก"
"ดูแลตัวเองวันนี้ ดีกว่ารอแก้ปัญหาพรุ่งนี้ครับ"
"ร่างกายมีแค่หนึ่งเดียว ดูแลให้ดีครับ"
"ถ้าอ่านแล้วรู้สึกว่าตรงกับตัวเอง ลองเริ่มจาก..."
```

### 5.6 คำเปรียบเทียบ (Analogy Bank)

| Analogy | ใช้เมื่อ |
|---|---|
| **ร่างกาย = รถยนต์** | ต้องเติมน้ำมัน ต้องเข้าศูนย์ ต้องเช็ค |
| **สารอาหาร = วัตถุดิบสร้างบ้าน** | ถ้าวัตถุดิบไม่ครบ บ้านก็ไม่แข็งแรง |
| **ภูมิคุ้มกัน = ทหารรักษาการณ์** | ถ้าทหารอ่อนแอ ศัตรูก็บุกง่าย |
| **ร่างกาย = แบตเตอรี่** | ถ้าชาร์จไม่เต็ม ก็ใช้ได้ไม่เต็มที่ |
| **อาหารเสริม = ประกันสุขภาพ** | ไม่ได้ใช้ตอนป่วย ใช้ตอนป้องกัน |
| **การดูแลสุขภาพ = การลงทุน** | ยิ่งเริ่มเร็ว ดอกเบี้ยทบต้นยิ่งมาก |

## 🏗️ PART 6: โครงสร้างโพสต์กระชับ (Compact Structure)

### 6.1 ขึ้นต้น (OPENING) — 10-15 บรรทัด

```
[บรรทัด 1-3]  HOOK: อาการ/คำถามที่คนเจอจริง
[บรรทัด 4-8]  PAIN: ขยายอาการ + ชี้ว่าคนอื่นก็เจอ
[บรรทัด 9-12] CONTEXT: ทำไมเรื่องนี้สำคัญ
━━━━━━━━━━━━
```

**สูตร Hook สายสุขภาพ:**

| แบบ | ตัวอย่าง |
|---|---|
| **ชี้อาการ** | "ตื่นมาแล้วยังเหนื่อย ทั้งที่นอนไป 7-8 ชั่วโมง" |
| **คำถาม** | "เคยสงสัยไหมครับ ทำไมกินครบ 3 มื้อ แต่ยังรู้สึกอ่อนเพลีย" |
| **สถิติ** | "คนไทย 70% ได้รับวิตามินดีไม่เพียงพอ" |
| **ประสบการณ์** | "3 เดือนก่อน ผมเป็นคนที่เหนื่อยง่ายมากครับ" |

### 6.2 ส่วนกลาง (BODY) — 40-60 บรรทัด

```
━━━━━━━━━━━━
REFRAME: พลิกมุมสาเหตุจริง (8-12 บรรทัด)
(วงเล็บ Insight)
━━━━━━━━━━━━
KNOWLEDGE / ANALOGY: ให้ความรู้ (12-18 บรรทัด)
━━━━━━━━━━━━
ACTION / SOLUTION: สิ่งที่ทำได้ 3-5 ข้อ (10-15 บรรทัด)
━━━━━━━━━━━━
```

**กฎ Body กระชับ:**
```
✅ ตัด Analogy เหลือ 1 ตัว (เดิมมี 2-3)
✅ ตัด Before vs After เหลือ 1 ชุดสั้น
✅ ตัด Case Study ออก (ย้ายไป Formula B)
✅ Application Steps ไม่เกิน 5 ข้อ
✅ แต่ละข้อยาวไม่เกิน 2 บรรทัด
```

### 6.3 ปิดท้าย (CLOSING) — 15-25 บรรทัด

```
━━━━━━━━━━━━
BRIDGE TO PRODUCT (ถ้ามี): 5-8 บรรทัด
━━━━━━━━━━━━
SUMMARY: สรุป Big Idea (3-5 บรรทัด)
[ คำคมสรุป ]
VISION: ภาพอนาคต (3-5 บรรทัด)
CTA: ชวนทำ/ติดตาม/สั่งซื้อ (3-5 บรรทัด)
#Hashtag Set (5-8 ตัว)
```

## ⚠️ PART 9: ข้อควรระวัง — กฎหมายคอนเทนต์สุขภาพ

> [!CAUTION]
> โพสต์สายสุขภาพมีกฎหมายคุม ต้องระวังเรื่องเหล่านี้:

```
❌ ห้ามอ้างว่า "รักษาโรค" / "หายขาด" / "ป้องกันโรค"
❌ ห้ามใช้คำว่า "ยา" กับอาหารเสริม
❌ ห้ามอ้างชื่อหมอ/โรงพยาบาลโดยไม่ได้รับอนุญาต
❌ ห้าม Before/After ที่เกินจริง
❌ ห้ามใช้คำว่า "การันตีผล 100%"

✅ ใช้คำว่า "ช่วยเสริม" / "ช่วยดูแล" / "เสริมสร้าง"
✅ ใช้คำว่า "ส่วนตัวผมรู้สึกว่า..."
✅ ใช้คำว่า "ผลลัพธ์อาจแตกต่างกันในแต่ละบุคคล"
✅ อ้างอิงงานวิจัยเมื่อพูดถึงสารอาหาร (ไม่ใช่ตัวสินค้า)
✅ แยกชัดระหว่าง "ข้อมูลสุขภาพ" กับ "การแนะนำสินค้า"
```

**คำ Safe ที่ใช้แทน:**

| ❌ ห้ามใช้ | ✅ ใช้แทน |
|---|---|
| รักษา | ดูแล / เสริมสร้าง |
| หายขาด | รู้สึกดีขึ้น / เปลี่ยนแปลง |
| ป้องกันโรค | ช่วยดูแลร่างกาย |
| ต้องกิน | อาจลองดู / เป็นทางเลือก |
| ดีที่สุด | เป็นหนึ่งในตัวเลือกที่ดี |
| การันตี | ส่วนตัวผมรู้สึกว่า / ลูกค้าหลายคนบอกว่า |

## 🎯 PART 12: เทคนิคสำคัญที่ต้องใช้

### 12.1 Trick 08: Repetition with Variation (ซ้ำแต่เปลี่ยนมุม)

**หลักการ:**
```
ใช้โครงสร้างประโยคเดียวกัน ซ้ำ 3-5 ครั้ง
แต่เปลี่ยนเนื้อหาทุกครั้ง
ครั้งสุดท้ายต้องหนักที่สุด (Climax)
```

**สูตร:**
```
"[คำนำ] [A]"
"[คำนำ] [B]"
"[คำนำ] [C]"
"[คำนำ] [D ← หนักที่สุด]"
```

**ตัวอย่างสายสุขภาพ:**
```
Pattern 1: "ก็อาจเพราะ..."

เหนื่อยง่าย ก็อาจเพราะขาดแมกนีเซียม
นอนไม่หลับ ก็อาจเพราะขาดแมกนีเซียม
เป็นตะคริว ก็อาจเพราะขาดแมกนีเซียม
เครียดง่าย ก็อาจเพราะขาดแมกนีเซียม

Pattern 2: "ไม่ใช่เพราะ..."

ไม่ใช่เพราะคุณนอนไม่พอ
ไม่ใช่เพราะคุณทำงานหนักเกินไป
ไม่ใช่เพราะคุณอายุมากขึ้น
แต่เพราะร่างกายขาดสิ่งที่มันต้องการ

Pattern 3: "ก่อนที่จะ..."

ก่อนที่จะกินยาแก้ปวด → ลองเช็คดูว่าขาดอะไรหรือเปล่า
ก่อนที่จะซื้อวิตามินมั่วๆ → ลองเช็คดูว่าร่างกายต้องการอะไร
ก่อนที่จะยอมแพ้กับอาการเหล่านี้ → ลองอ่านให้จบก่อน
```

**ใช้ตรงไหน:**
```
✅ ช่วงสรุป — ย้ำ Big Idea ให้ฝังหัว
✅ ช่วง Reframe — พลิกมุมหลายครั้งซ้อน
✅ ช่วง Pain — ซ้อม "ทำไม" หลายครั้งให้คนพยักหน้า
```

### 12.2 Trick 09: Emotional Dialogue (บทสนทนาในหัว)

**หลักการ:**
```
ใส่ "เสียงในหัว" ของผู้อ่านลงไปในโพสต์
ให้เขารู้สึกว่า "นี่คือสิ่งที่เราคิดอยู่เลย!"
วางไว้หลัง Hook เพื่อขยี้ Pain Point ให้ลึก
```

**สูตร:**
```
ลองคิดตามนะครับ
"[ประโยคที่คนคิดในหัว 1]"
"[ประโยคที่คนคิดในหัว 2]"
"[ประโยคที่คนคิดในหัว 3]"
"[ประโยคที่คนคิดในหัว 4]"
เคยถามตัวเองแบบนี้ไหมครับ?
แต่ถามเสร็จแล้วก็ปล่อยมันไป
เพราะคิดว่า…
"[ข้อแก้ตัวที่คนชอบใช้ 1]"
"[ข้อแก้ตัวที่คนชอบใช้ 2]"
```

**ตัวอย่างสายสุขภาพ:**
```
ลองคิดตามนะครับ
"ทำไมนอนไปตั้ง 7 ชั่วโมง ตื่นมาก็ยังเหนื่อยอีก"
"ทำไมสมาธิมันหลุดทั้งวัน ทั้งที่ไม่ได้ทำอะไรหนัก"
"ทำไมเครียดง่าย หงุดหงิดง่าย ทั้งที่เรื่องเล็กนิดเดียว"
"ทำไมเป็นตะคริวตอนดึกบ่อย ทั้งที่ไม่ได้ออกกำลังกาย"
เคยถามตัวเองแบบนี้ไหมครับ?
แต่ถามเสร็จแล้วก็ปล่อยมันไป
เพราะคิดว่า…
"เราก็แค่เหนื่อย ใคร ๆ ก็เป็น"
"อายุมากขึ้นก็แบบนี้แหละ"
```

**ใช้ตรงไหน:**
```
✅ หลัง Hook — ขยี้ปัญหาให้คนรู้สึกว่า "ใช่เลย นี่คือเรา!"
✅ ก่อน Reframe — ให้คนรู้สึกถึงปัญหาก่อน แล้วค่อยพลิก
✅ ในช่วง Before — ให้คนเห็นตัวเองในสถานการณ์
```

**หลักการเลือก Dialogue:**
```
✅ ใช้คำที่ "คนคิดในหัวจริง ๆ" ไม่ใช่คำทางการ
✅ ใช้โครงสร้าง "ทำไม...ทั้งที่..." ซ้ำ → สร้าง Frustration
✅ ปิดด้วย "ข้อแก้ตัว" ที่คนชอบใช้ → ให้คนรู้สึก "ใช่ ฉันก็คิดแบบนี้"
✅ ต่อด้วย "ผมเคยคิดแบบนี้เหมือนกัน" → สร้าง Connection
```

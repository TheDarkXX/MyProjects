## 📊 PART 1: โพสต์ที่วิเคราะห์

### สรุปภาพรวม
- โพสต์ทั้งหมดใน XLS: **21 โพสต์**
- Long-form (≥50 บรรทัด): **1 โพสต์** → วิเคราะห์ทั้งหมด
- Medium-form (20-49 บรรทัด): **10 โพสต์** → วิเคราะห์ทั้งหมด (เนื่องจาก Long-form มี < 3 โพสต์)
- Short-form (5-19): **2 โพสต์** → สแกน pattern
- Micro (<5): **8 โพสต์** → ข้าม

## 🧬 PART 2: DNA ของสไตล์การเขียน

### 2.1 Writing Persona (ตัวตนในการเขียน)

| มิติ | ลักษณะ |
|---|---|
| **บทบาท** | **แพทย์ที่เป็นกันเอง** — วางตัวเป็นหมอที่อธิบายเรื่องยากให้เข้าใจง่าย ไม่ใช่อาจารย์สอน ไม่ใช่ Guru แต่เป็น "หมอที่คุยกับเรา" |
| **ไม่ใช่** | ไม่ใช่ Influencer ขายของ, ไม่ใช่หมอที่พูดศัพท์ยาก, ไม่ใช่คนที่ตัดสินพฤติกรรม — จะไม่บอกว่า "ผิด" แต่บอกว่า "อาจไม่เหมาะ" |
| **สรรพนาม** | ใช้ **"ผม"** ตลอด (ไม่เคยใช้ "เรา" หรือ "พี่") |
| **จุดยืน** | หมอที่ให้ความรู้โดยไม่กดดัน — มีจุดยืนชัดว่า "รู้ก่อน แก้ได้ก่อน" แต่ไม่เร่ง ไม่ขู่ |
| **ระดับภาษา** | **กึ่งทางการ** — ใช้ศัพท์แพทย์เมื่อจำเป็นแต่อธิบายทุกครั้ง + ลงท้ายด้วย "ครับ" เกือบทุกประโยค ทำให้รู้สึกสุภาพและเป็นกันเอง |

### 2.2 Signature Elements (ลายเซ็นที่จำได้)

| ลายเซ็น | ตัวอย่าง |
|---|---|
| **ลงท้าย "ครับ" ทุกประโยคสำคัญ** | "...ทำให้ท้องอืดง่าย**ครับ**", "...ลองเปลี่ยนตรงนี้**ครับ**" |
| **CTA ปิดท้ายเดิมทุกโพสต์** | "ใครมีคำถาม หรืออยากให้ผมเขียนความรู้เรื่องอะไรคอมเมนต์กันมาได้เลยนะครับ" |
| **เปิดด้วย "หลายคน..."** | ใช้ "หลายคน" เปิดต้นโพสต์เกือบทุกโพสต์ เป็น Signature ชัดมาก |
| **"ผมตอบให้เลย/ผมบอกให้เลย"** | ใช้ประโยคนี้เมื่อจะตอบคำถามหลัก — ให้ความรู้สึกตรงไปตรงมา |
| **Hashtag ท้ายโพสต์** | #หมอเจด #หมอเจดสุขภาพ #หมอเจดโคราช + Hashtag เฉพาะหัวข้อ |
| **จุด "." เดี่ยวคั่นระหว่างหมวด** | ใช้จุด "." เพียงตัวเดียวเป็นเส้นคั่นระหว่างหมวดย่อย |
| **Emoji หน้าหัวข้อย่อย** | ใช้ Emoji เดี่ยว (เช่น 🔵, 🟡) นำหน้าหัวข้อย่อยแต่ละอัน — แต่ไม่ใช้ Emoji ในเนื้อหาปกติ |

## 🎭 PART 3: TONE & EMOTION

### 3.1 Tone Spectrum

| ช่วงของโพสต์ | Tone | ตัวอย่างจริง |
|---|---|---|
| **Hook** | ท้าทายความเชื่อ / จี้จุด | "กินผักทุกวัน ระวังลำไส้พัง เพราะเลือกผักผิด!" |
| **ชี้ปัญหา** | อ่อนโยน + ไม่ตัดสิน | "หลายคนตั้งใจ 'กินสลัดทุกวัน' เพราะอยากลำไส้ดี... แต่ทำไมบางคนท้องอืด" |
| **ให้ข้อมูล** | มั่นใจ + เป็นกันเอง | "ถ้ามื้อไหนมีแต่แป้งขัดขาว น้ำหวาน หรือของหวานเยอะ น้ำตาลจะขึ้นเร็วมากครับ" |
| **Reframe** | หนักแน่น + กระตุ้นสติ | "ร่างกายไม่ควรต้องพังพาบทุกครั้งหลังกินข้าวครับ" |
| **ยกตัวอย่าง** | เป็นกันเอง + ติดตลก | "ไม่ใช่เช้ากาแฟ กลางวันขนม แล้วเย็นกินไก่ย่างทั้งตัวชดเชย" |
| **สรุป** | อบอุ่น + ให้กำลังใจ | "เริ่มจากผักสุกนิ่ม เพิ่มทีละน้อย เคี้ยวให้ดี..." |
| **CTA** | เปิดกว้าง + เชิญชวน | "ใครมีคำถาม หรืออยากให้ผมเขียนความรู้เรื่องอะไรคอมเมนต์กันมาได้เลยนะครับ" |

### 3.2 Emotional Arc

โพสต์ของหมอเจดมี Emotional Arc 5 ขั้นที่ซ้ำกันทุกโพสต์:

📉 **ขั้น 1: Recognition (จับอาการ)**
→ ผู้อ่านรู้สึก: "เอ๊ะ เรื่องนี้ฉันก็เป็น!"
→ Quote: "บางคนปิดไฟแล้ว แต่สมองยังไม่ยอมพักครับ"

📉 **ขั้น 2: Pinch (จี้ให้ตระหนัก)**
→ ผู้อ่านรู้สึก: "อ๋อ มันไม่ใช่เรื่องปกตินี่นา"
→ Quote: "อันนี้บางทีไม่ใช่แค่อิ่มครับ แต่มันอาจเป็นสัญญาณว่า 'ระบบเผาผลาญเริ่มรวน'"

📈 **ขั้น 3: Educate (สอนด้วยเหตุผล)**
→ ผู้อ่านรู้สึก: "โอ้ มีเหตุผลทางวิทยาศาสตร์ด้วย"
→ มีตัวเลข, กลไกทางการแพทย์, อ้างอิงงานวิจัย

📈 **ขั้น 4: Relief (ให้ทางออก)**
→ ผู้อ่านรู้สึก: "ทำได้ ไม่ยากเลย"
→ Quote: "ไม่ต้องจับเวลาจนมื้ออาหารหมดสนุกครับ"

📈 **ขั้น 5: Empowerment (ให้กำลังใจ)**
→ ผู้อ่านรู้สึก: "หมอเข้าใจเรา อยากลองทำ"
→ Quote: "ผมอยากให้ลองสังเกตร่างกายตัวเองดี ๆ ครับ"

### 3.3 Emotion Injection (เทคนิคฉีดอารมณ์)

| เทคนิค | วิธี | ตัวอย่างจริง |
|---|---|---|
| **เสียงคนไข้จริง** | ใส่ประโยคที่คนไข้พูดในเครื่องหมายคำพูด | "กินข้าวเสร็จทีไร หนังตาจะปิดทุกทีเลยหมอ" |
| **ขยายอาการให้เห็นภาพ** | บรรยายอาการละเอียดจนผู้อ่าน "เห็นตัวเอง" | "นอนคิดเรื่องงาน คิดถึงพรุ่งนี้ แล้วหยิบมือถือขึ้นมาดูอีกนิดจนเลยเวลานอนไปไกลแล้ว" |
| **เปรียบเทียบมุกตลก** | ใช้สถานการณ์ตลกแต่จริงเพื่อ "จุก" | "ไม่ใช่เช้ากาแฟ กลางวันขนม แล้วเย็นกินไก่ย่างทั้งตัวชดเชย" |
| **ใส่คำแซวตัวเอง** | ใช้มุกที่ผู้อ่านเคยแซวตัวเอง | "แซวกันเล่นว่า 'หนังท้องตึง หนังตาหย่อน'" |
| **Parenthetical Insight** | วงเล็บอธิบายเพิ่มเพื่อเน้น | "(ไม่ใช่แข็งแรงขึ้นครับ)" |

### 3.4 Tone Rules (กฎที่ไม่เคยเปลี่ยน)

**สิ่งที่ "ไม่เคยทำ":**
- ❌ ไม่เคยดุ / ตำหนิผู้อ่าน
- ❌ ไม่เคยใช้คำว่า "ผิด" ตรงๆ กับพฤติกรรมของผู้อ่าน — จะใช้ "อาจไม่เหมาะ", "ไม่จำเป็น", "ต้องระวัง"
- ❌ ไม่เคยขู่ให้กลัวตาย — จะ warn แต่ไม่ scare
- ❌ ไม่เคยอวดตัว / โชว์ความเก่ง
- ❌ ไม่เคยพูด "ห้าม" ตรงๆ — จะใช้ "ควรลด", "ไม่จำเป็น", "ระวัง"

**สิ่งที่ "ทำเสมอ":**
- ✅ ลงท้ายด้วย "ครับ" ในทุกจุดสำคัญ
- ✅ ใช้ "ไม่ได้แปลว่า..." ก่อน Reframe
- ✅ เปิดทางให้ผู้อ่านไม่ต้องเครียด: "ไม่ต้อง...", "ไม่จำเป็นต้อง..."
- ✅ ให้ทางเลือกเสมอ ไม่บังคับ: "ลอง...", "ถ้า...ก็ได้"
- ✅ ปิดท้ายด้วย CTA ที่เปิดกว้าง (ไม่ขาย ไม่กดดัน)

## 🎯 PART 4: AUDIENCE

### 4.1 กลุ่มเป้าหมายที่ระบุชัดในเนื้อหา

จากโพสต์ทั้งหมด หมอเจดเรียกกลุ่มเป้าหมายว่า:
- "หลายคน" (ใช้มากที่สุด — เปิดกว้าง ไม่ชี้หน้า)
- "บางคน" (เมื่อพูดถึงอาการเฉพาะ)
- "คนที่มีเบาหวาน / คนที่ดื้ออินซูลิน"
- "ผู้สูงอายุ"
- "คนลดน้ำหนัก / คนกำลังลดไขมัน"
- "คนเล่นเวท / คนออกกำลังกาย"
- "คนที่มีโรคไต"
- "คนที่มีลำไส้แปรปรวน"

**Audience หลัก:** คนทั่วไปวัย 30-60+ ที่สนใจสุขภาพ มีโรคประจำตัว หรือเริ่มมีอาการเตือน

### 4.2 วิธีเรียกผู้อ่าน (Addressing the Reader)

| วิธี | ตัวอย่าง | ใช้เมื่อ |
|---|---|---|
| **"หลายคน"** | "หลายคนกินโปรตีนแบบเดาสุ่มครับ" | เปิดโพสต์ — ไม่ชี้หน้า ให้ผู้อ่าน self-identify |
| **"บางคน"** | "บางคนไม่เคยแสบร้อนกลางอกเลยครับ" | พูดถึงอาการเฉพาะ |
| **ไม่มีสรรพนาม** | "เริ่มจากผักสุกนิ่ม" | ตอนสรุป/แนะนำ — กระชับ ตรง |
| **"ตัวเอง/ตัวเรา"** | "ลองสังเกตร่างกายตัวเองดี ๆ ครับ" | ตอนให้กำลังใจ |
| **"ใคร" (CTA)** | "ใครมีคำถาม..." | ปิดท้ายโพสต์ — เปิดกว้างสุด |

### 4.3 วิธีทำให้ผู้อ่านรู้สึก "เขาพูดกับเรา" (Identification)

| เทคนิค | ตัวอย่าง |
|---|---|
| **ลิสต์อาการที่คนอ่านเจอจริง** | "ท้องอืด จุกแน่น ขับถ่ายไม่ดีขึ้น แถมจุลินทรีย์ดีก็ไม่เพิ่ม?" |
| **ใช้ "อาจ" "บางครั้ง" ไม่กดดัน** | "อันนี้บางทีไม่ใช่แค่อิ่มครับ แต่มัน**อาจ**เป็นสัญญาณ..." |
| **ยกย่องผู้อ่านก่อนสอน** | "หลายคน**ตั้งใจ** 'กินสลัดทุกวัน' เพราะ**อยาก**ลำไส้ดี" |
| **ใช้สถานการณ์ที่คุ้นเคย** | "จากหาเสียงฝนหนึ่งคลิป เดี๋ยวก็หลุดไปดูข่าว ดูคลิปสั้น" |
| **อย่าเพิ่ง + ทำไม** | "อย่าเพิ่งเลิกผักทั้งหมด **แค่**เลือกให้ถูกกับท้องตัวเองก่อนครับ" |

## 📝 PART 5: VOCABULARY & WORD USAGE

### 5.1 Signature Vocabulary (คำที่ใช้ซ้ำ)

🔵 **คำเปิดเรื่อง (Opening Words):**
- "หลายคน..." (ใช้ทุกโพสต์)
- "บางคน..."
- "ผมตอบให้เลยว่า..."
- "ผมเลย..."

🟡 **คำพลิกมุม (Reframe Words):**
- "ไม่ได้แปลว่า..."
- "จริง ๆ อาจไม่ใช่..."
- "ความจริงคือ..."
- "แต่ต้องระวัง..."
- "ไม่ได้หมายความว่า..."
- "ทั้งที่..."

🟢 **คำยกตัวอย่าง (Example Words):**
- "เช่น..."
- "ตัวอย่าง..."
- "ผลคือ..."
- "เหมือน..."
- "แบบนี้..."

🔴 **คำย้ำประเด็น (Emphasis Words):**
- "ต้องระวัง..."
- "สิ่งสำคัญคือ..."
- "ที่สำคัญคือ..."
- "เพราะฉะนั้น..."
- "อย่ารีบ..."

🟣 **คำสรุป (Closing Words):**
- "สรุป..." / "สรุปรอบนี้..."
- "แบบนี้...ถึงจะ..."
- "เริ่มจาก..."
- "ใครมีคำถาม หรืออยากให้ผมเขียนความรู้เรื่องอะไรคอมเมนต์กันมาได้เลยนะครับ"

### 5.2 การผสมภาษา

| กฎ | รายละเอียด |
|---|---|
| ✅ ใช้ศัพท์แพทย์ภาษาอังกฤษ | CGM, LDL, HDL, BMI, HbA1c, FODMAP, White coat hypertension, Stroke, AF, ABPM, Resistant Starch, ASMR |
| ✅ อธิบายศัพท์ทุกครั้ง | "เบตากลูแคน" → "ใยอาหารละลายน้ำที่เรียกว่าเบตากลูแคน" |
| ✅ ใช้คำอังกฤษใน Hashtag | #longevity #healthylifestyle |
| ❌ ไม่ใช้ศัพท์อังกฤษแทรกในประโยคไทย | ไม่มี "คุณต้อง manage stress" แบบนี้ |
| ❌ ไม่ใช้สแลงอังกฤษ | ไม่มี "toxic", "red flag" แบบคนทั่วไป |

### 5.3 Word Frequency (คำที่ใช้บ่อยสุด TOP 10)

| อันดับ | คำ | ความถี่ | หน้าที่ |
|---|---|---|---|
| 1 | "ครับ" | สูงมาก (เกือบทุกประโยค) | ลงท้ายสุภาพ สร้างความเป็นกันเอง |
| 2 | "หลายคน" | ทุกโพสต์ | Opening + เชื่อมผู้อ่าน |
| 3 | "บางคน" | เกือบทุกโพสต์ | ชี้เฉพาะเจาะจง ไม่กดดัน |
| 4 | "ไม่ได้แปลว่า" | เกือบทุกโพสต์ | Reframe ความเข้าใจผิด |
| 5 | "อาจ" | ถี่มาก | ให้ทางเลือก ไม่ฟันธง |
| 6 | "ควร" | ถี่มาก | แนะนำแบบไม่บังคับ |
| 7 | "ลอง" | ถี่มาก | เปิดทาง ไม่สั่ง |
| 8 | "ไม่ต้อง/ไม่จำเป็น" | ทุกโพสต์ | ลดความเครียด ปลดล็อกผู้อ่าน |
| 9 | "เพราะ" | ถี่มาก | อธิบายเหตุผลทุกครั้ง |
| 10 | "เช่น" | ถี่มาก | ยกตัวอย่างประกอบเสมอ |

### 5.4 ลูกเล่นการใช้คำพิเศษ

#### Pattern 1: "ไม่ได้แปลว่า... แต่คือ..." (Negation-Revelation)
**สูตร:** X ไม่ได้แปลว่า [ความเข้าใจผิด] + แต่ [ความจริง]
**จากโพสต์จริง:** "ไขมันในเลือดสูง**ไม่ได้แปลว่า**ต้องกินแต่ของจืดจนไม่มีความสุขครับ"
> **จุดประสงค์:** พลิกความเชื่อเดิม — ทำให้ผู้อ่านเปิดใจรับข้อมูลใหม่

#### Pattern 2: "ต่อให้... ก็..." (Concession)
**สูตร:** ต่อให้ [ทำดี] + ก็/แต่ [ยังไม่พอ]
**จากโพสต์จริง:** "**ต่อให้**กินปลาแซลมอนทุกวัน **แต่**ยังดื่มชาเย็น ไตรกลีเซอไรด์**ก็**ไม่เกรงใจครับ"
> **จุดประสงค์:** ป้องกันผู้อ่าน cherry-pick ข้อมูล

#### Pattern 3: "ไม่ใช่... แต่คือ..." (Redefine)
**สูตร:** ไม่ใช่ [สิ่งที่คิด] + แต่คือ [สิ่งที่จริง]
**จากโพสต์จริง:** "คำว่า 'ไขมันอุดตันในช่องท้อง' **ไม่ได้หมายถึง**ไขมันไปอุดเหมือนท่อน้ำ **แต่มันคือ**ไขมันที่กองล้อมรอบตับ ลำไส้"
> **จุดประสงค์:** แก้ misconception ที่แพร่หลาย

#### Pattern 4: "อย่ารีบ... ก่อนที่..." (Don't Rush)
**สูตร:** อย่า [ทำ X] + จนลืม/ก่อน [Y ที่สำคัญกว่า]
**จากโพสต์จริง:** "อย่ารีบสรุปว่าลดไขมันสำเร็จครับ เพราะน้ำหนักที่หายไปอาจเป็นน้ำหรือกล้ามเนื้อ"
> **จุดประสงค์:** ชะลอความยินดีที่ไม่ครบ ให้ดูข้อมูลเพิ่ม

#### Pattern 5: "ถ้า... ก็ได้" / "ไม่ต้อง..." (Permission Pattern)
**สูตร:** ไม่ต้อง [กังวล/ทำ X หนัก] + แค่ [ทำ Y เบาๆ] ก็ได้
**จากโพสต์จริง:** "**ไม่ต้อง**จับเวลาจนมื้ออาหารหมดสนุกครับ", "**ไม่ต้อง**รวมทุกอย่างในจานเดียว"
> **จุดประสงค์:** ลดความเครียดให้ผู้อ่าน — ทำให้คำแนะนำรู้สึก "ทำได้จริง"

## 🎪 PART 6: WRITING TRICKS (เทคนิคลูกเล่น)

### Trick 1: "Reframe the Problem" (พลิกมุมปัญหา)
**สูตร:** [สิ่งที่คนเข้าใจ] → "จริง ๆ แล้ว..." → [มุมใหม่]
**จากโพสต์จริง:**
> "หลายคนคิดว่าในเมื่อกินข้าว ผัก และเนื้อในปริมาณเท่าเดิม สุดท้ายลงท้องเหมือนกัน น้ำตาลก็น่าจะขึ้นพอ ๆ กัน" → จริงๆ แค่สลับลำดับก็ต่างกัน 28 mg/dL

> **จุดประสงค์**: ท้าทายความเชื่อเดิม — ดึงให้อ่านต่อ

### Trick 2: "Before vs After" (เปรียบก่อน-หลัง)
**สูตร:** [วิธี A] → ผลลัพธ์ A VS [วิธี B] → ผลลัพธ์ B
**จากโพสต์จริง:**
> "กินข้าวก่อน: น้ำตาลพุ่ง 75 จุด, ถึงยอด 45-60 นาที, กลับ 3 ชม."
> "กินผักโปรตีนก่อน: น้ำตาลพุ่ง 48 จุด, ถึงยอด 60-75 นาที, กลับ 2.5 ชม."

> **จุดประสงค์**: ให้เห็นผลต่างชัดเจนด้วยตัวเลข

### Trick 3: "Analogy / Metaphor" (เปรียบเทียบ)
**สูตร:** [เรื่องยาก] = "เหมือน [เรื่องง่าย]"
**จากโพสต์จริง:**
> "เหมือนมีโรงงานเล็ก ๆ เปิดทำงานอยู่ในพุงทุกวันครับ" (ไขมันปล่อยสารอักเสบ)
> "เหมือนเปิดสัญญาณเตือนไฟไหม้ทิ้งไว้ทั้งวัน" (อักเสบเรื้อรัง)
> "ร่างกายเลยเหมือน 'มีพลังงาน แต่ใช้ไม่ได้'" (ดื้ออินซูลิน)

> **จุดประสงค์**: ทำเรื่องทางการแพทย์ให้เข้าใจได้ทันที

### Trick 4: "Cascading Short Lines" (ประโยคสั้นซ้อน)
**สูตร:** สร้างจังหวะเร็วด้วยการตัดบรรทัดสั้นๆ ติดๆ กัน
**จากโพสต์จริง:**
> "กินข้าวก่อน
> ก่อนกิน 96 mg/dL
> ยอดสูงสุด 171 mg/dL
> พุ่งขึ้น 75 จุด"

> **จุดประสงค์**: สร้าง visual impact — สแกนง่าย + จำได้

### Trick 5: "Internal Monologue Quote" (ใส่เสียงคนไข้/ผู้อ่าน)
**สูตร:** "..." (เสียงคนไข้/ผู้อ่าน)
**จากโพสต์จริง:**
> "กินข้าวเสร็จทีไร หนังตาจะปิดทุกทีเลยหมอ"

> **จุดประสงค์**: สร้างอารมณ์ร่วม — ผู้อ่านรู้สึก "ฉันก็เคยพูดแบบนี้"

### Trick 6: "ตลกจุก" (Relatable Humor)
**สูตร:** [พฤติกรรมที่คนทำจริง] + [แต่ไม่ค่อยจะดี]
**จากโพสต์จริง:**
> "ซื้อเวย์มาวางเต็มบ้าน แต่ข้าวแต่ละมื้อยังแทบไม่มีโปรตีนครับ"
> "กินอะโวคาโดหนึ่งมื้อ แล้วมื้อเย็นจัดหมูกระทะพร้อมของหวาน"

> **จุดประสงค์**: ตลกแต่จุก — ทำให้ผู้อ่านยิ้มแล้วยอมรับ

### Trick 7: "Gift + Comment CTA" (ของขวัญแลก Comment)
**สูตร:** ถ้าอยากให้ผมเขียนเรื่อง [X] → คอมเมนต์มา
**จากโพสต์จริง:**
> "ครั้งหน้าอยากให้ผมลองเปรียบเทียบ 'กินเร็ว VS กินช้า' หรือ 'เดินหลังอาหาร VS นั่งต่อ' คอมเมนต์ไว้ได้เลยครับ"

> **จุดประสงค์**: กระตุ้น engagement + สร้างหัวข้อถัดไป

## 🏗️ PART 7: โครงสร้างการเรียบเรียง (Post Structure)

### 7.1 Macro Structure

```
┌──────────────────────────────────┐
│ Opening — ~10-15%                │
│ Hook + Context + Pain            │
├──────────────────────────────────┤
│ Body — ~70-75%                   │
│ หมวดย่อย x 4-8 หมวด             │
│ (แต่ละหมวดมี Loop Structure)     │
├──────────────────────────────────┤
│ Closing — ~15-20%                │
│ สรุป Bullet + CTA + Hashtag     │
└──────────────────────────────────┘
```

### 7.2 Opening (ขึ้นต้น) — รายละเอียด

**แบบ A: Hook + Context + Pain (ใช้บ่อยสุด ~70%)**
1. [บรรทัด 1-2] HOOK — ท้าทายความเชื่อ / จี้อาการ
2. [บรรทัด 3-5] CONTEXT — อธิบายว่าทำไมคนถึงคิดแบบนั้น
3. [บรรทัด 5-7] PAIN — ลิสต์อาการที่เกิดขึ้นจริง
4. [บรรทัด 8] TRANSITION — "วันนี้เรามาดูกัน..." / "ผมเลย..."

**แบบ B: คำถาม + คำตอบทันที**
1. [บรรทัด 1] คำถามเปิด
2. [บรรทัด 2] "ผมตอบให้เลยว่า..." + คำตอบ
3. [บรรทัด 3-5] อธิบายเพิ่ม

**HOOK Types ที่ใช้:**

| แบบ | ตัวอย่าง |
|---|---|
| **ท้าทายความเชื่อ** | "กินผักทุกวัน ระวังลำไส้พัง เพราะเลือกผักผิด!" |
| **VS เปรียบเทียบ** | "กินข้าวก่อน VS ผักกับโปรตีนก่อน" |
| **คำถาม + ตอบทันที** | "อ้วนลงพุง = โรงงานอักเสบจริงไหม? ผมตอบให้เลยว่าจริงครับ" |
| **เสียงคนไข้** | ""กินข้าวเสร็จทีไร หนังตาจะปิดทุกทีเลยหมอ"" |
| **สถานการณ์ที่คุ้นเคย** | "บางคนปิดไฟแล้ว แต่สมองยังไม่ยอมพักครับ" |
| **ขัดแย้ง** | "วัดความดันที่บ้านปกติ แต่ไปโรงพยาบาลพุ่ง!" |

### 7.3 Body (ส่วนกลาง) — รายละเอียด

**มี Loop Structure ชัดเจน ซ้ำ 4-8 รอบ:**

```
┌──────────────────────┐
│  Emoji  หัวข้อย่อย    │
│  เนื้อหาอธิบาย        │
│  (ตัวอย่าง/ตัวเลข)    │
│  .                    │
└──────────────────────┘
     ↓ ซ้ำ 4-8 ครั้ง
```

**กฎจังหวะ:**
- ✅ ทุกหมวดย่อยมี **หัวข้อ 1 บรรทัด** + **เนื้อหา 3-6 ประโยค**
- ✅ ทุกหมวดย่อยจบด้วย **"."** (จุดเดี่ยว) หรือ **"ครับ"**
- ✅ แต่ละหมวดมี **ตัวอย่างจริง** หรือ **ตัวเลข** อย่างน้อย 1 อย่าง
- ✅ ระหว่างหมวด ใช้ **จุดเดี่ยว "."** เป็น separator

### 7.4 Closing (ปิดท้าย) — รายละเอียด

**โครงสร้าง:**
1. **[หัวข้อสรุป]** เช่น "สิ่งที่ช่วยลดอาการ..." / "ลดรอบเอวให้ลงจริง เริ่มจากตรงไหนดี?"
2. **[Bullet List]** สรุป action items 5-8 ข้อ ขึ้นต้นด้วย "•"
3. **[ย่อหน้าสรุปรวม]** 2-3 ประโยค สรุปใจความทั้งโพสต์
4. **[CTA]** "ใครมีคำถาม หรืออยากให้ผมเขียนความรู้เรื่องอะไรคอมเมนต์กันมาได้เลยนะครับ"

## 🔴 PART 8: MASTER FORMULAS (สูตรหลัก)

### Formula 1: "Myth Buster" (ทลายความเชื่อผิด)
**ใช้ใน:** โพสต์ผัก-ลำไส้, โพสต์ง่วงหลังอาหาร, โพสต์อ้วนลงพุง, โพสต์ไขมัน
```
OPENING:  Hook ท้าทายความเชื่อ → Pain list → "วันนี้เรามาดูกัน"
BODY:     หมวดย่อย x 5-7 (ผิด→ถูก) คั่นด้วย "."
CLOSING:  Bullet สรุป + ย่อหน้ารวม + CTA
```

### Formula 2: "VS Comparison" (เปรียบเทียบตัวต่อตัว)
**ใช้ใน:** โพสต์ CGM กินข้าวก่อน vs ผัก, โพสต์ความดันบ้าน vs รพ.
```
OPENING:  คำถาม VS → ตั้งสมมติฐาน → "ผมเลยลอง..."
BODY:     [วิธี A + ตัวเลข] → [วิธี B + ตัวเลข] → เปรียบเทียบ → อธิบาย
CLOSING:  ตาราง/Bullet สรุป + CTA เปรียบเทียบต่อ
```

### Formula 3: "How-To Guide" (สอนทำตาม)
**ใช้ใน:** โพสต์โปรตีน, โพสต์ ASMR, โพสต์วัดรอบเอว
```
OPENING:  Pain (ทำแบบนี้แต่ไม่ work) → "ปริมาณที่เหมาะต้องดู..."
BODY:     กลุ่ม x 5-7 (แต่ละกลุ่มมีสูตร/ตัวเลข) คั่นด้วย "."
CLOSING:  สูตรจำง่าย + Bullet list + CTA
```

### Formula 4: "Symptom Decoder" (แกะรหัสอาการ)
**ใช้ใน:** โพสต์กรดไหลย้อน, โพสต์ Stroke
```
OPENING:  อาการที่คนไม่รู้ → "ต้นเหตุอาจเป็น..."
BODY:     อาการ x 5-6 (แต่ละอันมีอธิบาย + เตือน) คั่นด้วย "."
CLOSING:  "ปรับยังไง?" + Bullet + CTA
```

## 📐 PART 10: POST FORMATTING RULES

### 10.1 ความยาวบรรทัด (Line Length)

| ช่วง | สัดส่วน | บทบาท |
|---|---|---|
| 1-15 ตัวอักษร | ~5% | หัวข้อ, ตัวเลขเดี่ยว, จุดคั่น |
| 16-40 ตัวอักษร | ~15% | หัวข้อย่อย, ประโยคสั้นเน้น |
| 41-100 ตัวอักษร | ~30% | ประโยคสรุป, bullet items |
| 101+ ตัวอักษร | ~50% | ย่อหน้าเนื้อหาหลัก |

**กฎทอง:**
- ✅ โพสต์หมอเจดเป็น **"ย่อหน้ายาว"** เป็นหลัก — ไม่ใช่สไตล์ตัดบรรทัดสั้นๆ
- ✅ แต่ละหมวดย่อยเป็น **ย่อหน้าเดียว** ยาวต่อเนื่อง 3-6 ประโยค
- ✅ ประโยคสั้นใช้เฉพาะตอน **สรุป/ตัวเลข/bullet**

### 10.2 การตัดบรรทัด & เว้นบรรทัด

| กฎ | ตัวอย่าง |
|---|---|
| ✅ หัวข้อย่อย = บรรทัดเดี่ยว | " สูตรพื้นฐาน คูณน้ำหนักตัวด้วย 0.8" |
| ✅ เนื้อหา = ย่อหน้ายาวต่อเนื่อง | ไม่ตัดบรรทัดภายในย่อหน้า — ปล่อย Facebook word-wrap |
| ✅ ระหว่างหมวด = จุดเดี่ยว "." | จุดหนึ่งตัวเป็น separator |
| ✅ Bullet list = ตัดบรรทัดทุกข้อ | "• ลดน้ำหวาน..." (แต่ละ bullet อาจยาว) |
| ❌ ไม่ตัดบรรทัดกลางประโยค | ไม่เขียนแยกบรรทัดกลางประโยค |

### 10.3 สัญลักษณ์คั่น (Separators)

| สัญลักษณ์ | จำนวนที่พบ | ใช้เมื่อ |
|---|---|---|
| **.** (จุดเดี่ยว) | ทุกโพสต์ 4-8 ครั้ง | คั่นระหว่างหมวดย่อย |
| **Emoji** | บางโพสต์ | นำหน้าหัวข้อย่อย |
| **•** (bullet) | ทุก Closing 5-8 ข้อ | สรุป action items |

### 10.4 Emoji Usage

- จำนวน Emoji ที่พบในโพสต์ยาว = **น้อยมาก (0-6 ตัว/โพสต์)**
- Emoji ที่ใช้: **Emoji สี** เช่น 🔵, 🟡 → ใช้นำหน้าหัวข้อย่อยเท่านั้น
- **ไม่ใช้ Emoji ในเนื้อหา** — นี่คือ Design Choice สำคัญ

**สัญลักษณ์ที่ "ไม่เคยใช้":**
- ❌ เส้นคั่นยาว (━, ═, ──)
- ❌ วงเล็บจีน (【】, 《》)
- ❌ สัญลักษณ์ตกแต่ง (★, ✦, ➤)
- ❌ Emoji อารมณ์ (🔥, 💪, ❤️)

### 10.5 Visual Rhythm (จังหวะทางสายตา)

**สูตรจังหวะ:**
```
หัวข้อสั้น (1 บรรทัด)
→ ย่อหน้ายาว (3-6 ประโยค)
→ จุดเดี่ยว "."
→ หัวข้อสั้น
→ ย่อหน้ายาว
→ จุดเดี่ยว "."
→ [ซ้ำ x 4-8 ครั้ง]
→ หัวข้อสรุป (1 บรรทัด)
→ Bullet list สั้น (5-8 ข้อ)
→ ย่อหน้าสรุป (2-3 ประโยค)
→ CTA
```

### 10.6 ความยาวรวมของโพสต์

| ประเภท | ความยาว | จำนวนหมวดย่อย | จำนวน Bullet |
|---|---|---|---|
| Long-form | ~56 บรรทัด | 6-8 หมวด | 6-8 bullet |
| Medium-form (ยาว) | ~30-39 บรรทัด | 5-7 หมวด | 5-7 bullet |
| Medium-form (สั้น) | ~21-22 บรรทัด | 4-6 หมวด | 5-6 bullet |

### 10.7 Hashtag Formatting

- จำนวน Hashtag: **5-10 ตัว/โพสต์** (อยู่ท้ายโพสต์เท่านั้น)
- ตำแหน่ง: **ท้ายสุดของโพสต์** (หลัง CTA)
- กฎ: ผสมไทย + อังกฤษ
- เรียงลำดับ: Brand → Series → Topic → General

**Hashtag ที่ใช้ประจำ:**
- #หมอเจด #หมอเจดโคราช #หมอเจดสุขภาพ
- #สุขภาพดี #longevity
- + Hashtag เฉพาะหัวข้อ เช่น #เบาหวาน #ไขมันในเลือด #กรดไหลย้อน

## ✅ PART 11: QUICK CHECKLIST

### เนื้อหา:
- ☐ **TONE:** สุภาพ + เป็นกันเอง + ไม่ตัดสิน + ลงท้าย "ครับ" ทุกจุดสำคัญ
- ☐ **EMOTION:** มี Emotional Arc 5 ขั้น (Recognition → Pinch → Educate → Relief → Empowerment)
- ☐ **AUDIENCE:** เปิดด้วย "หลายคน/บางคน" + ลิสต์อาการที่คนอ่านเจอจริง
- ☐ **TRICK:** มี Reframe ≥ 1, ตัวอย่างตลกจุก ≥ 1, "ไม่ได้แปลว่า..." ≥ 1
- ☐ **OPENING:** Hook ท้าทายความเชื่อหรือชี้อาการ + "หลายคน..." context
- ☐ **BODY:** หมวดย่อย 4-8 หมวด แต่ละหมวดมี: หัวข้อ + ย่อหน้าเนื้อหา + ตัวอย่าง/ตัวเลข
- ☐ **CLOSING:** Bullet list 5-8 ข้อ + ย่อหน้าสรุป + CTA "ใครมีคำถาม..."
- ☐ **ไม่ดุ ไม่ตัดสิน ไม่ขู่ ไม่อวด** — ใช้ "อาจ", "ลอง", "ไม่จำเป็นต้อง"
- ☐ มีตัวเลข/ข้อมูลจริง อ้างอิงแนวทางทางการแพทย์
- ☐ สรรพนาม "ผม" เท่านั้น (ไม่ใช้ "เรา", "พี่")

### Formatting:
- ☐ **LINE:** เนื้อหาเป็นย่อหน้ายาวต่อเนื่อง (ไม่ตัดบรรทัดสั้นๆ)
- ☐ **BREAK:** หัวข้อย่อย = บรรทัดเดี่ยว, เนื้อหา = ย่อหน้าเดียว
- ☐ **SEPARATOR:** ใช้จุดเดี่ยว "." คั่นระหว่างหมวดเท่านั้น
- ☐ **EMOJI:** ใช้ Emoji สีนำหน้าหัวข้อเท่านั้น (ไม่ใช้ในเนื้อหา)
- ☐ **SYMBOL:** ใช้ "•" ใน bullet, "–" ในช่วงตัวเลข, ไม่มีเส้นคั่นยาว
- ☐ **HASHTAG:** 5-10 ตัวท้ายโพสต์ (#หมอเจด #หมอเจดสุขภาพ + เฉพาะหัวข้อ)
- ☐ **RHYTHM:** สั้น(หัวข้อ) → ยาว(เนื้อหา) → จุดคั่น → สั้น → ยาว → ซ้ำ

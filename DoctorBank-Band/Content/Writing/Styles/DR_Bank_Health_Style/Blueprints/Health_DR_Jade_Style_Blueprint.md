# 🎯 Health DR Jade Style Blueprint
## ปรับจาก DR Jade Clone → สไตล์ หมอแบงค์ (The Realist)
### Version: 1.0 — เรียล ตรง เห็นภาพ เข้าถึงวัย 40+

---

## 🔄 PART 1: สิ่งที่ปรับ vs สิ่งที่คงไว้ (จาก DR Jade)

### ✅ คงไว้เหมือนเดิม (DNA ที่ยังใช้ได้)

| สิ่งที่คง | เหตุผล |
|---|---|
| Emoji นำหน้าหัวข้อย่อย (🔴 🟡) | ช่วยให้สแกนอ่านง่าย |
| Bullet List สรุปท้ายโพสต์ | ชัดเจน เอาไปทำตามได้ |
| Reframe ความเชื่อผิด ≥ 1 จุด | ทำให้คนเปิดใจรับข้อมูลใหม่ |
| Hashtag ท้ายโพสต์ 5-8 ตัว | SEO + ค้นหาง่าย |
| มีตัวอย่าง/ตัวเลขประกอบ | เพิ่มความน่าเชื่อถือ |
| "ครับ" ลงท้ายจุดสำคัญ | สุภาพ น่าเชื่อถือ |

### 🔧 สิ่งที่ต้องปรับ (จาก DR Jade → DR Bank)

| สิ่งที่ปรับ | DR Jade (เดิม) | DR Bank (ใหม่) |
|---|---|---|
| **สรรพนาม** | "ผม" (หมอ) | **"เรา"** (เพื่อน) |
| **Tone** | สุภาพ-วิชาการ-ไม่ตัดสิน | **ตรงไปตรงมา-ภาษาพูด-เห็นภาพทันที** |
| **ภาษา** | กึ่งทางการ มีศัพท์แพทย์ | **ภาษาบ้านๆ ตัดศัพท์ออกให้หมด** |
| **Pain Point** | ชี้อาการ + อธิบายทางวิทย์ | **ขยี้อาการให้เห็นเลือด + ใช้ "เสียงในหัว"** |
| **เปรียบเทียบ** | Analogy ทางการ (โรงงานเล็กๆ ในพุง) | **Analogy บ้านๆ (สวิตช์ปลดล็อค, เบรกมือ, หัวเทียน)** |
| **Separator** | จุดเดี่ยว "." | **ไม่ใช้ separator — เว้นบรรทัดธรรมดา** |
| **ย่อหน้า** | ย่อหน้ายาวต่อเนื่อง 3-6 ประโยค | **ย่อหน้าสั้น 1-3 ประโยค กระชับเป็นชิ้นๆ** |
| **CTA** | เปิดกว้าง "ใครมีคำถาม คอมเมนต์มา" | **Micro-Commitment "พิมพ์เลขข้อที่ตรงกับคุณ"** |
| **Failed Solutions** | ไม่มี | **ต้องมีเสมอ! (ลองมาหมดแล้วไม่หาย → เปิดทางรับ Solution ใหม่)** |
| **Confirmation Trigger** | ไม่มี | **ท้ายแต่ละข้อต้องมี ("นี่ใช่เลย", "ถ้าตรง นี่คือคำตอบ")** |
| **Emotional Close** | ไม่มี | **ต้องมีก่อน CTA เสมอ ("อย่าปล่อยให้ร่างกายร้องขอจนเสียงแหบ")** |
| **ตัวตน** | แพทย์ที่เป็นกันเอง | **เพื่อนที่เคยเป็นแบบนี้มาก่อน + ศึกษาเรื่องสุขภาพจริงจัง** |

## 🎭 PART 2: Persona & Tone สำหรับ DR Bank

### 2.1 ตัวตนที่ต้องสร้าง

```
❌ ไม่ใช่: หมอที่มาสอน (ห่างไกล)
❌ ไม่ใช่: พ่อค้าอาหารเสริม (ดูขายของ)
❌ ไม่ใช่: Influencer รีวิว (ดูโฆษณา)
❌ ไม่ใช่: อาจารย์มหาวิทยาลัย (ยากเกินไป)

✅ คือ: "เพื่อนที่เคยเป็นแบบเรามาก่อน แล้วไปศึกษาหาทางออกจนเจอ"
✅ คือ: "คนที่พูดตรงๆ ไม่อ้อมค้อม เข้าใจง่าย เห็นภาพทันที"
✅ คือ: "คนที่ลองมาหมดแล้วเหมือนกัน เลยรู้ว่าอะไร work อะไรไม่ work"
```

### 2.2 สรรพนาม & คำเรียก

```
ตัวเอง:    "เรา" (ไม่ใช่ "ผม" ไม่ใช่ "หมอ")
ผู้อ่าน:    "คุณ" / "ใครที่..." / ไม่ต้องมีสรรพนาม (ตรงที่สุด)
ลงท้าย:    "ครับ" (ยังคงไว้เพื่อความสุภาพ แต่ไม่ต้องทุกประโยค)
```

### 2.3 Tone Spectrum

| ช่วง | Tone DR Jade (เดิม) | Tone DR Bank (ใหม่) |
|---|---|---|
| **Hook** | ท้าทายความเชื่อ | **จิกอาการตรงๆ + ภาพความเจ็บปวดที่เห็นชัด** |
| **Pain** | อ่อนโยน + ไม่ตัดสิน | **ขยี้ให้เห็นภาพ "นี่มันกูนี่หว่า!" + Failed Solutions** |
| **Reframe** | หนักแน่น + กระตุ้นสติ | **ตรงไปตรงมา + เปรียบเทียบภาษาบ้านๆ** |
| **สอน** | ผู้รู้ + อ้างข้อมูล | **เพื่อนบอกเพื่อน + ใช้คำบ้านๆ อธิบาย** |
| **สรุป** | อบอุ่น + ให้กำลังใจ | **ตรง + ให้ทางเลือก + Emotional Close** |
| **CTA** | เปิดกว้าง "ใครมีคำถาม..." | **Micro-Commitment "พิมพ์เลขข้อที่ตรง"** |

### 2.4 Tone Rules (กฎเหล็กของ DR Bank)

**สิ่งที่ "ไม่เคยทำ":**
- ❌ ไม่ใช้ศัพท์แพทย์โดยไม่แปลเป็นภาษาบ้านๆ ทันที
- ❌ ไม่เขียนยาวเยิ่นเย้อ — ถ้าตัดได้ ตัด
- ❌ ไม่ใช้คำเปรียบเทียบที่วัยรุ่นเกินไป (เช่น "ซอมบี้") — ต้องข้ามวัยได้
- ❌ ไม่ Overclaim (เช่น "หมดเกลี้ยง", "รักษาโรค", "หายขาด")
- ❌ ไม่แทนตัวเองว่า "หมอ" — ใช้ "เรา"
- ❌ ไม่ใช้คำที่ต้อง "แปลซ้ำ" ในหัว (ถ้าอ่านแล้วต้องคิดต่อ = ตัดทิ้ง)

**สิ่งที่ "ทำเสมอ":**
- ✅ เปิดด้วยภาพอาการจริงที่คนเจอ (ไม่ใช่ข้อมูล)
- ✅ มีท่อน "Failed Solutions" ทุกโพสต์ (ลองมาหมดแล้วไม่หาย)
- ✅ มี "เสียงในหัว" หรือ Internal Dialogue อย่างน้อย 1 ท่อน
- ✅ มี Confirmation Trigger ท้ายแต่ละข้อ ("นี่ใช่เลย", "ถ้าตรง...")
- ✅ มี Emotional Close ก่อน CTA ("อย่าปล่อยให้ร่างกายร้องขอจนเสียงแหบ")
- ✅ มีสถิติ/ตัวเลขน่าตกใจอย่างน้อย 1 จุด (Stat Bomb)
- ✅ ให้ทางเลือก ไม่บังคับ: "ลอง...", "ถ้า...ก็ได้"

## 👥 PART 3: Audience

### 3.1 กลุ่มเป้าหมายหลัก

```
🎯 กลุ่มหลัก:
"คนวัย 40+ ที่เริ่มมีอาการเตือนแต่ตรวจเลือดปกติ"
"คนที่ลองมาหมดแล้วแต่ยังไม่หาย (กินยา ปรับเวลานอน ทานอาหารเสริม)"
"คนที่เริ่มยอมรับว่า อายุมากขึ้นก็คงแบบนี้แหละ"

🎯 กลุ่มย่อย:
"คนนอนดึก ตื่นมาไม่สดชื่น"
"คนที่เป็นตะคริวตอนดึก"
"คนที่ปวดไมเกรนบ่อย"
"คนที่อ่อนเพลียเรื้อรัง ทั้งที่กินดีนอนพอ"
```

### 3.2 Pain Points สไตล์ DR Bank (ภาษาบ้านๆ)

```
😩 ระดับ Physical (ภาษาที่คนจริงๆ ใช้):
- "ตื่นกลางดึกเพราะตะคริว ปวดเกร็งจนนอนไม่หลับ"
- "หนังตากระตุกจนรำคาญใจ"
- "นอนไม่หลับ พลิกตัวก็แล้ว นับแกะก็แล้ว"
- "ปวดไมเกรนตุ้บๆ จนต้องนั่งหลับตานิ่งๆ ทำอะไรไม่ได้"
- "นอนลากยาว 8 ชม. ตื่นมายังเพลียไร้เรี่ยวแรง"

😤 ระดับ Emotional (ข้อแก้ตัวที่คนใช้):
- "อายุ 40+ แล้วก็คงแบบนี้แหละ"
- "เราก็แค่เหนื่อย ใครๆ ก็เป็น"
- "ลองมาหมดแล้ว ก็ไม่หาย"
- "หรือร่างกายเราจะเป็นแบบนี้ไปตลอด?"
```

## 🎪 PART 4: WRITING TRICKS (เทคนิคลูกเล่น DR Bank)

### Trick 1: "Failed Solutions Bridge" (สะพานแห่งความล้มเหลว) ⭐ เทคนิคใหม่!
**สูตร:** [สิ่งที่ลองมาแล้ว] → [ไม่ได้ผล] → [Fear of Permanence] → [เฉลย Solution ใหม่]
**ตัวอย่าง:**
> "ลองปรับมาหลายอย่างก็ไม่ได้ผล ทั้งกินยา ปรับเวลานอน ทานอาหารเสริม แต่ก็ไม่ค่อยได้ผล
> จนเริ่มคิดว่า... 'หรือร่างกายเราจะเป็นแบบนี้ไปตลอด?'
> จริงๆ แล้วร่างกายกำลังเรียกร้อง..."

> **จุดประสงค์**: ซื้อใจคนอ่านทันที เพราะมัน "เรียล" — คนอ่านจะรู้สึกว่า "คนเขียนเข้าใจฉัน"

### Trick 2: "Confirmation Trigger" (ปุ่มพยักหน้า) ⭐ เทคนิคใหม่!
**สูตร:** ท้ายแต่ละข้ออาการ ใส่ประโยคสั้นๆ ที่ทำให้คนอ่านพยักหน้า
**คลังคำ:**
- "ถ้ายิ่งเป็นบ่อย เป็นถี่ๆ นี่ใช่เลย"
- "ถ้าเป็นแบบนี้ สังเกตตัวเองดีๆ ครับ"
- "ถ้าใช่ ลองอ่านต่อครับ"
- "ถ้าตรง นี่คือคำตอบครับ"
- "ใครเป็นแบบนี้ ลองเช็คดูครับ"

> **จุดประสงค์**: ยิ่งคนพยักหน้ามากเท่าไหร่ ความเชื่อใจยิ่งสูงขึ้น — เหมือนถูกจูงมือเดินจนจบ

### Trick 3: "Emotional Close" (ปิดอารมณ์ก่อน CTA) ⭐ เทคนิคใหม่!
**สูตร:** [ประโยคกระแทกใจ 1 บรรทัด] → [CTA]
**คลังคำ:**
- "อย่าปล่อยให้ร่างกายร้องขอจนเสียงแหบครับ"
- "ร่างกายมันซ่อมตัวเองไม่ได้ ถ้าขาดวัตถุดิบ"
- "อย่ารอจนวันที่ต้องนอนโรงพยาบาลแล้วค่อยสนใจ"
- "สุขภาพไม่ได้เริ่มจากวันที่ป่วย แต่เริ่มจากวันที่เลือก"

> **จุดประสงค์**: สร้างแรงผลักดันให้คนลงมือทำ CTA ทันที

### Trick 4: "Reframe the Problem" (พลิกมุมปัญหา) — คงจาก DR Jade
**สูตร:** [สิ่งที่คนเข้าใจ] → "จริงๆ แล้ว..." → [มุมใหม่]
**ตัวอย่าง DR Bank:**
> "หลายคนคิดว่าแค่เครียด แค่อายุมากขึ้น
> จริงๆ แล้ว ร่างกายกำลังขาดแร่ธาตุที่ชื่อ แมกนีเซียม"

### Trick 5: "Analogy บ้านๆ" (เปรียบเทียบภาษาชาวบ้าน) — ปรับจาก DR Jade
**สูตร:** [เรื่องแพทย์ยากๆ] = "[ของที่คนทุกวัยรู้จัก]"
**คลัง Analogy DR Bank:**

| Analogy | ใช้เมื่อ |
|---|---|
| **แคลเซียม = หดตัว, แมกนีเซียม = สวิตช์ปลดล็อค** | อธิบายตะคริว |
| **แมกนีเซียม = เบรกมือ / น้ำมันเบรก** | อธิบายนอนไม่หลับ (GABA) |
| **แมกนีเซียม = หัวเทียน** | อธิบายอ่อนเพลีย (ATP) |
| **กินดี = เติมน้ำมันเต็มถัง** | เปรียบเทียบอาหารกับพลังงาน |
| **เส้นประสาทเพี้ยน = สายไฟช็อต** | อธิบายหนังตากระตุก |

### Trick 6: "Internal Monologue" (เสียงในหัวผู้อ่าน) — คงจาก DR Jade
**สูตร:** ใส่ "เสียงในหัว" ของผู้อ่านในเครื่องหมายคำพูด
**ตัวอย่าง DR Bank:**
> "จนเริ่มคิดว่า... 'หรือร่างกายเราจะเป็นแบบนี้ไปตลอด?'"
> "'อายุ 40+ แล้วก็คงแบบนี้แหละ'"
> "'เราก็แค่เหนื่อย ใคร ๆ ก็เป็น'"

### Trick 7: "Stat Bomb" (ระเบิดตัวเลข) ⭐ เทคนิคใหม่!
**สูตร:** ใส่สถิติ 1 จุดใน Hook หรือก่อนเข้าเนื้อหาหลัก
**ตัวอย่าง:**
> "(งานวิจัยพบว่าคนกว่า 50% กำลังขาดแมกนีเซียมโดยไม่รู้ตัว)"

> **จุดประสงค์**: เพิ่มน้ำหนักความน่าเชื่อถือแบบกระแทกหน้า

### Trick 8: "Relatable Humor" (ตลกจุก) — คงจาก DR Jade
**สูตร:** [พฤติกรรมที่คนทำจริง] + [แต่ไม่ค่อยจะดี]
**ตัวอย่าง DR Bank:**
> "พลิกตัวก็แล้ว นับแกะก็แล้ว ก็ยังนอนไม่หลับ"
> "อัดวิตามินราคาแพงเต็มกระเพาะ แต่ตื่นมายังเพลียไร้เรี่ยวแรง"

## 🏗️ PART 5: โครงสร้างโพสต์ DR Bank

### 5.1 Macro Structure

```
┌──────────────────────────────────────┐
│ Opening — ~15-20%                     │
│ Hook (อาการจริง) + Failed Solutions   │
│ + Stat Bomb + เฉลยสาเหตุ              │
├──────────────────────────────────────┤
│ Body — ~60-65%                        │
│ อาการ x 4-6 ข้อ                       │
│ (แต่ละข้อ: อาการ + อธิบาย + Confirm)  │
├──────────────────────────────────────┤
│ Closing — ~15-20%                     │
│ วิธีแก้ Bullet + Emotional Close     │
│ + CTA Micro-Commitment + Hashtag     │
└──────────────────────────────────────┘
```

### 5.2 Opening — รายละเอียด

```
[บรรทัด 1-2]  HOOK: ภาพอาการจริงที่เจ็บปวด ตรงๆ (ไม่ใช่ข้อมูล)
[บรรทัด 3-4]  FAILED SOLUTIONS: ลองมาหมดแล้วไม่หาย
[บรรทัด 5]    INTERNAL DIALOGUE: "หรือจะเป็นแบบนี้ไปตลอด?"
[บรรทัด 6-7]  REVEAL: จริงๆ แล้ว... + (Stat Bomb)
[บรรทัด 8]    TRANSITION: "นี่คือ X สัญญาณเตือน..."
```

**HOOK Types สไตล์ DR Bank:**

| แบบ | ตัวอย่าง |
|---|---|
| **ภาพความเจ็บปวด** | "ตื่นกลางดึก เพราะตะคริวที่น่อง ปวดเกร็งจนนอนไม่หลับ..." |
| **อาการชุด (3 จุด)** | "กระตุกยิกๆ... ปวดหัวจี๊ด... ตื่นกลางดึก..." |
| **ขัดแย้งกับความเชื่อ** | "กินดีนอนครบ แต่ตื่นมายังเพลีย — ร่างกายกำลังบอกอะไรอยู่" |
| **คำถามที่คนถามตัวเอง** | "ทำไมนอนตั้งนาน แต่ตื่นมายังงัวเงีย?" |

### 5.3 Body — รายละเอียด

**Loop Structure ของ DR Bank (ซ้ำ 4-6 รอบ):**

```
🔴 [หมายเลข]. [ชื่ออาการตรงๆ]
[บรรยายอาการ 1-2 บรรทัด — ภาษาบ้านๆ เห็นภาพ]
[อธิบาย "ทำไม" 1-2 บรรทัด — เปรียบเทียบง่ายๆ]
[Confirmation Trigger — "ถ้ายิ่งเป็นบ่อย นี่ใช่เลย"]
```

**กฎ Body DR Bank:**
```
✅ แต่ละข้อยาวไม่เกิน 4-5 บรรทัด (กระชับ)
✅ ใช้ภาษาที่ "คนป้าข้างบ้านอ่านแล้วเข้าใจ" เป็นมาตรฐาน
✅ ทุกข้อต้องมี Analogy หรือเปรียบเทียบ อย่างน้อย 1 อัน
✅ ทุกข้อต้องปิดด้วย Confirmation Trigger
❌ ห้ามอธิบายแบบเรียงความ ห้ามเป็นย่อหน้ายาว
❌ ห้ามใช้ศัพท์แพทย์โดยไม่แปลทันที
```

## 🔴 PART 8: MASTER FORMULAS (สูตรหลัก DR Bank)

### Formula 1: "Symptom Decoder" (แกะรหัสอาการ) ← ใช้บ่อยที่สุด
**ใช้ใน:** โพสต์อาการตะคริว, นอนไม่หลับ, ไมเกรน, อ่อนเพลีย
```
OPENING:  Hook อาการจริง → Failed Solutions → Stat Bomb → "นี่คือ X สัญญาณ..."
BODY:     อาการ x 4-6 ข้อ (แต่ละข้อ: อาการ + ทำไม + Analogy + Confirm)
CLOSING:  Bullet วิธีแก้ + Emotional Close + CTA Micro-Commitment
```

### Formula 2: "Myth Buster" (ทลายความเชื่อผิด)
**ใช้ใน:** โพสต์ "กินผักทุกวันระวังลำไส้พัง", "อ้วนลงพุงไม่ใช่แค่กินเยอะ"
```
OPENING:  Hook ท้าทายความเชื่อ → "จริงๆ แล้ว..." → Failed Solutions
BODY:     ความเชื่อผิด x 4-5 ข้อ (ผิด→ถูก + Analogy) + Confirm
CLOSING:  Bullet + Emotional Close + CTA
```

### Formula 3: "How-To Practical" (ทำตามได้เลย)
**ใช้ใน:** โพสต์วิธีเลือกอาหารเสริม, วิธีลดอาการ, สิ่งที่ต้องเช็ค
```
OPENING:  Pain (ทำมาหมดแล้วไม่ work) → "วิธีที่ได้ผลจริงคือ..."
BODY:     วิธีแก้ x 4-6 ข้อ (แต่ละข้อ: ทำอะไร + ทำไม + ตัวอย่าง)
CLOSING:  สรุป + Emotional Close + CTA
```

## ✅ PART 10: QUICK CHECKLIST

### เนื้อหา:
- ☐ **PERSONA:** เขียนในมุม "เรา" (เพื่อนที่เข้าใจ) ไม่ใช่ "ผม" (หมอ)
- ☐ **TONE:** ตรงไปตรงมา + ภาษาบ้านๆ + เห็นภาพทันที + ไม่ต้องแปลซ้ำ
- ☐ **HOOK:** เปิดด้วยภาพอาการจริง (ไม่ใช่ข้อมูล)
- ☐ **FAILED SOLUTIONS:** มีท่อน "ลองมาหมดแล้วไม่หาย" ทุกโพสต์
- ☐ **INTERNAL DIALOGUE:** มี "เสียงในหัว" อย่างน้อย 1 ท่อน
- ☐ **STAT BOMB:** มีสถิติ/ตัวเลข อย่างน้อย 1 จุด
- ☐ **CONFIRM:** ทุกข้ออาการต้องมี Confirmation Trigger ปิดท้าย
- ☐ **EMOTIONAL CLOSE:** มีก่อน CTA เสมอ
- ☐ **CTA:** เป็น Micro-Commitment (พิมพ์เลข + รางวัลตอบแทน)
- ☐ **ANALOGY:** ใช้เปรียบเทียบภาษาบ้านๆ ที่คนทุกวัยเข้าใจ
- ☐ **ไม่ Overclaim:** ไม่ "หมดเกลี้ยง" ไม่ "รักษา" ไม่ "หายขาด"
- ☐ ภาษาผ่านมาตรฐาน "คนป้าข้างบ้านอ่านแล้วเข้าใจ"

### Formatting:
- ☐ **ย่อหน้าสั้น:** 1-3 ประโยค (ไม่ย่อหน้ายาว)
- ☐ **เว้นบรรทัด:** ระหว่างข้ออาการ (ไม่ใช้ "." คั่น)
- ☐ **EMOJI:** 🔴 นำหน้าข้อ, ✅ นำหน้าสรุป, 👇 นำหน้า CTA
- ☐ **HASHTAG:** 5-8 ตัว เจาะอาการ + แร่ธาตุ + กลุ่มเป้าหมาย
- ☐ **ความยาว:** 35-70 บรรทัด

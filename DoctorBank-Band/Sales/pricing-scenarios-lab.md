> 🎯 ใช้ Decoy Effect ขั้นสูง: ทุก Tier ดันลูกค้าไป Hero
> เหมาะกับ: ต้องการกำไรต่อออเดอร์สูงที่สุด

| Variant | รายละเอียด | ราคา | ต่อขวด | กำไร (Organic) | Margin | บทบาท |

**Decoy ทำงานอย่างไร:**
```
เซต 3 = 2,390 (797/ขวด)
3 แถม 1 = 2,670 (668/ขวด)

→ จ่ายเพิ่มแค่ 280 บาท ได้อีก 1 ขวด (มูลค่า 890!)
→ ราคาต่อขวดลดจาก 797 → 668 ทันที
→ สมอง: "280 แลกของ 890? ต้องเอา 3 แถม 1 สิ!"
```

**Profit Ladder:** 333 → 828 (+495) → 759 (-69)
- ⚠️ **กำไร 3 แถม 1 (759) ต่ำกว่าเซต 3 (828)** แต่ลูกค้าจะเลือก 3 แถม 1 เพราะจิตวิทยา
- ✅ **ถ้าลูกค้าเลือกเซต 3 เราได้ 828 → WIN!**
- ✅ **ถ้าเลือก 3 แถม 1 เราได้ 759 → ก็ยัง WIN!**
- ทั้ง 2 ทาง เราได้กำไร 759-828 ต่อบิล → **ดีกว่าทุกเมนูอื่น!**

## 🏆 Part 4: คำแนะนำสุดท้าย

### 🥇 แนะนำอันดับ 1: เมนู 4 "Decoy Master"

```
1 กระปุก          ฿890    (890/ขวด)
เซต 3 กระปุก     ฿2,390   (797/ขวด, ลด 10%)    ← Decoy
🔥 3 แถม 1       ฿2,670   (668/ขวด, ลด 25%)    ← Hero "ซื้อ 3 แถม 1 ฟรี!"
```

**เหตุผล:**
1. **กำไรสูงสุดไม่ว่าลูกค้าเลือกอะไร:** เซต 3 = 828฿, 3 แถม 1 = 759฿ → ทั้งคู่สูงกว่าทุกเมนูอื่น
2. **Decoy ทำงานสมบูรณ์แบบ:** จ่ายเพิ่มแค่ 280 ได้ฟรีอีก 1 ขวด → สมองบอก "ต้องเอา 3 แถม 1"
3. **"3 แถม 1" = จ่ายราคา 3 ขวดพอดี:** ลูกค้าคำนวณง่ายมาก (890×3=2,670 ✓)
4. **ราคา 668/ขวด ชนะคู่แข่ง mid-premium** ที่ขาย 700-800/ขวดเดี่ยว
5. **ลูกค้าได้ของ 4 เดือน** → repeat purchase cycle ยาว
6. **ไม่ต้องใช้ "1 แถม 1"** ที่กำไรบาง → เก็บ BOGO ไว้ทำ LIVE แทน

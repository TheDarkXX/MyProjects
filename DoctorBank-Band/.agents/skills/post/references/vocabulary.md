## 1. คำเปิดเรื่อง (Hook Words)

```
"ถ้าคุณเริ่มรู้สึกว่า..."
"ลองสังเกตตัวเองดูนะ"
"หลายคนไม่รู้ว่า..."
"เคยสงสัยไหมว่า ทำไม..."
"ร่างกายส่งสัญญาณมาตลอด แต่เราอาจไม่ได้ฟัง"
"นี่คือสิ่งที่ผมเพิ่งเข้าใจเรื่อง #สุขภาพ"
"คนไทย [X] ใน 10 คน [ปัญหา] โดยไม่รู้ตัว"
"ผมพูดตรง ๆ เลยนะ"
```

## 2. คำ Reframe (พลิกมุม)

```

"หลายคนแก้ที่ปลายเหตุ แต่ยังไม่ได้แก้ที่ต้นตอ"
"ไม่ใช่ว่าร่างกายอ่อนแอ แต่เราอาจให้สิ่งที่มันต้องการไม่ครบ"

"ทั้งหมดนี้ไม่ผิด แต่ถ้าสาเหตุจริงไม่ใช่ตรงนั้นล่ะ…"
```

```

```

## 6. Analogy Bank (คำเปรียบเทียบ)

| Analogy | ใช้เมื่อ |
|---|---|
| **ร่างกาย = สมาร์ทโฟน** | สายชาร์จพัง ชาร์จทั้งคืนก็ไม่เต็ม |
| **ร่างกาย = รถยนต์** | ต้องเติมน้ำมัน ต้องเข้าศูนย์ ต้องเช็ค |
| **สารอาหาร = วัตถุดิบสร้างบ้าน** | ถ้าวัตถุดิบไม่ครบ บ้านก็ไม่แข็งแรง |
| **ภูมิคุ้มกัน = ทหารรักษาการณ์** | ถ้าทหารอ่อนแอ ศัตรูก็บุกง่าย |
| **อาหารเสริม = ประกันสุขภาพ** | ไม่ได้ใช้ตอนป่วย ใช้ตอนป้องกัน |
| **การดูแลสุขภาพ = การลงทุน** | ยิ่งเริ่มเร็ว ดอกเบี้ยทบต้นยิ่งมาก |

## 7. ตาราง "คำทางการ → คำบ้านๆ" (Tone v2)

| ❌ คำทางการ/ซับซ้อน | ✅ คำบ้านๆ ตรงไปตรงมา |
|---|---|
| อ่อนเพลียเรื้อรัง | เหนื่อยทั้งวัน พักก็ไม่หาย |
| ได้รับไม่เพียงพอ | ได้ไม่พอ / ขาด |
| มีส่วนเกี่ยวข้อง | ร่างกายใช้มัน… |
| ส่งสัญญาณเตือน | ร่างกายกำลังบอกอะไรอยู่ |
| ผลลัพธ์อาจแตกต่าง | แต่ละคนไม่เหมือนกันนะ |
| จากการศึกษาของ NIH | ข้อมูลจาก NIH บอกว่า... |
| ควบคุมระบบประสาท | ช่วยให้สมองสงบ |
| กระบวนการ | อย่าง / เรื่อง |
| ร่างกายกำลังส่งสัญญาณเตือนบางอย่าง | นี่ไม่ใช่เรื่องปกติ |

## 8. Engagement Bait ที่ใช้ได้จริง (ไม่ดูขายของ)

| เทคนิค | ตัวอย่าง | ทำไมได้ผล |
|---|---|---|
| **นับข้อ + คอมเมนต์ตัวเลข** | "เข้ากี่ข้อ? คอมเมนต์ตัวเลขไว้เลย" | ง่าย แค่พิมพ์เลข → คนทำเยอะ |
| **Poll ในคอมเมนต์** | "ข้อไหนตรงที่สุด? พิมพ์เลข 1-5" | เหมือนเกม คนอยากมีส่วนร่วม |
| **แท็กคนที่ห่วง** | "แชร์ให้คนที่คุณห่วงอ่านด้วยนะ" | ใช้ "ความห่วงใย" ไม่ใช่ "ขายของ" |
| **Save ไว้อ่านทีหลัง** | "กด Save ไว้นะ เผื่อวันนึงต้องกลับมาดู" | เพิ่ม Save = อัลกอริทึมดัน |
| **ถามประสบการณ์** | "เคยลองแก้ยังไงบ้าง? เล่าให้ฟังหน่อย" | คนชอบเล่าเรื่องตัวเอง |

## 9. Share Trigger — ทำไมคนถึงแชร์

```
คนแชร์เพราะ 4 เหตุผล:
1. "ฉันเป็นแบบนี้เลย!" → ตรงกับตัวเอง = แชร์ไปเลย
2. "เพื่อนต้องรู้!" → ห่วงคนอื่น = แชร์เพราะรัก
3. "ข้อมูลดี เซฟไว้" → มีคุณค่า = Save + Share
4. "ว้าว ไม่เคยรู้!" → ตกใจ = แชร์ทันที

วิธีใส่ Share Trigger ในโพสต์:
✅ มีสถิติที่ตกใจ (7 ใน 10 คนขาด...)
✅ มีฉากที่คนเห็นตัวเอง (เช้าวันจันทร์...)
✅ มีเช็คลิสต์ให้นับ (5 ข้อ ลองเช็คดู)
✅ มี Insight ที่ไม่เคยรู้ (Mag ช่วย GABA...)
✅ ปิดด้วย "แชร์ให้คนที่ห่วง"
```

## 10. Algorithm Hack — สัญญาณที่ Facebook ดัน

```
Facebook ดันโพสต์ที่มี:
📌 Comment เยอะ → ชวนคอมเมนต์ตัวเลข/ประสบการณ์
📌 Share เยอะ → ใส่ Share Trigger
📌 Save เยอะ → เนื้อหาที่คนอยากกลับมาดู
📌 เวลาอ่านนาน → โพสต์ยาวที่อ่านสนุก
📌 คนอ่านจนจบ → Build-up ดี ไม่น่าเบื่อ

สิ่งที่ Facebook ลดการเข้าถึง:
❌ Engagement Bait ชัดเกิน ("กดไลค์ถ้าเห็นด้วย")
❌ ลิงก์ออกนอก Facebook
❌ ขอให้แชร์แบบตรงๆ ("แชร์ด่วน!")
❌ Clickbait หลอก (หัวข้อกับเนื้อหาไม่ตรง)
```

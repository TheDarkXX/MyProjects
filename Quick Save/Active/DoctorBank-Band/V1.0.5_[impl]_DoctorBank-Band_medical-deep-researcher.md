# Medical Deep Researcher (GPT-Researcher + Codex Auth)

## 📌 Context (Compiled Truth)
- แบรนด์ DoctorBank ต้องการทำตลาดย้ำจุดแข็งเรื่อง **สูตรจริงใจ (Liposomal Gluta 250mg + GlyNAC 300mg)** เทียบกับคู่แข่งที่เน้นทำราคาถูก (1 แถม 1) แต่ใช้วัตถุดิบเกรดจีนและโรยผงนางฟ้า (Fairy Dust Precursors)
- คุณหมอต้องการหา **Strong Evidence** จากงานวิจัยระดับโลก (เช่น NIH, PubMed) ដើម្បីนำไปใช้เป็นข้ออ้างอิงในการทำไลฟ์สดและแคมเปญการตลาด
- ข้อจำกัดของ Perplexity คือไม่สามารถดึงเฉพาะเจาะจง Medical sites ตาม Format ที่ลึกได้ และเราต้องการดึงสมอง LLM จาก `openclaw-vps` (Hermes Codex) ที่ล็อกอินผ่านระบบ OAuth ของระบบ ไม่ใช่ API Key ธรรมดา
- **การแก้ปัญหา:** ทางระบบได้ดำเนินการสร้างโค้ด Python Script เฉพาะกิจชื่อ `medical_research.py` โดยเชื่อมกับ `gpt-researcher` และทำการเปิด **SSH Tunnel** ไปที่เซิร์ฟเวอร์ Codex Auth เพื่อให้ตัว Script ใช้งานระบบ OAuth ของ OpenClaw ได้โดยตรง
- Script นี้ตั้งค่า `MEDICAL_ROLE` เฉพาะเจาะจง ให้จำลองตัวเป็น Senior Medical Research Analyst และบังคับให้เขียนรายงานโดยใช้ **Inline Citations** สไตล์ Perplexity [^1] และทำ Evidence Summary Table.

## Proposed Changes

### 1. Environment Setup & Installation
- รันคำสั่ง `git clone https://github.com/assafelovic/gpt-researcher.git` เพื่อดึงโปรแกรมเวอร์ชันล่าสุดลงมา
- สร้าง Python Virtual Environment (`python -m venv venv`)
- ติดตั้ง Dependencies ผ่านคำสั่ง `pip install -r requirements.txt`

### 2. Configuration (.env)
- สร้างไฟล์ `.env` สำหรับเก็บ API Keys
- นำ API Keys ที่คุณหมอให้มาใส่เข้าไปในระบบ พร้อมตั้งค่าให้ใช้โมเดล GPT-4o หรือรุ่นที่ต้องการ

### 3. Execution (Running the Web UI)
- รันคำสั่งเปิดเซิร์ฟเวอร์ (เช่น `python -m uvicorn main:app --reload` หรือผ่านคำสั่งรันของระบบ)
- ตัวโปรแกรมจะมีหน้าเว็บ (Web UI) หน้าตาคล้ายๆ Perplexity ให้คุณหมอเปิดใช้งานผ่านเบราว์เซอร์ (localhost) ได้เลยครับ

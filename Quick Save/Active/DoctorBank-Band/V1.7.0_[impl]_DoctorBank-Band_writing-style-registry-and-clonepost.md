# 🚀 V1.7.0: Writing Style Registry & /clonepost Skill

## 📌 Context (Compiled Truth)
- ปรับโครงสร้างระบบ AI Post Writer (`/drbpost`) ของ DoctorBank Band ใหม่ 
- แยกส่วนที่เป็น "สไตล์เฉพาะ" ออกจากตัว Skill.md ไปไว้ที่โฟลเดอร์ของตัวเอง (เช่น `Content/Writing/Styles/Health_Supplement_Winbiz_Style/STYLE_PROFILE.md`)
- `SKILL.md` (ของ `/drbpost`) ทำหน้าที่เป็นเพียง "เครื่องยนต์กลาง" (Style Router, Compliance rules, Scorecard)
- สร้าง Skill ใหม่ `clonepost` สำหรับวิเคราะห์สไตล์การเขียนจากไฟล์ CSV/Excel/TXT ให้เป็น Writing Blueprint ระดับ Clone Kit (รองรับ Long, Medium, Short form)
- อัปเดต `AGENT_CONTEXT.md` โดยรวมศูนย์ Skill Index ไว้ในไฟล์เดียว เพื่อความสะดวกในการเรียกใช้และจดจำ

### Component 2: SKILL.md (สกิลหลัก)

#### [MODIFY] SKILL.md

ปรับ SKILL.md ให้ทำหน้าที่เป็น **"เครื่องยนต์กลาง"** ที่เก็บเฉพาะ:
1. **Cold Start Protocol** — อ่าน memory → brand → product เหมือนเดิม
2. **Style Router** — [NEW] ระบบเลือกสไตล์: อ่าน `STYLE_PROFILE.md` ของสไตล์ที่ User ระบุ แล้วไปดึง Blueprint จากโฟลเดอร์นั้น
3. **Compliance / Iron Rules** — กฎหมาย อย. คำห้ามใช้ (คงไว้เหมือนเดิม ใช้ร่วมทุกสไตล์)
4. **Scorecard 20 เกณฑ์** — คงไว้เหมือนเดิม (ใช้ร่วมทุกสไตล์)
5. **Workflow / Version Control** — คงไว้เหมือนเดิม
6. **ย้าย Persona, Tone, Post Formulas, Writing Tricks, Formatting Rules** ออกจาก SKILL.md → ไปอยู่ใน `STYLE_PROFILE.md` แทน (เพราะสิ่งเหล่านี้เปลี่ยนไปตาม Style)

> [!IMPORTANT]
> **หลักการ: SKILL.md = กฎกลางที่ไม่เปลี่ยนตาม Style | STYLE_PROFILE.md = สิ่งที่เปลี่ยนตาม Style**

## วิธีใช้งาน

```text
/drbpost winbiz แมกนีเซียม           ← ใช้ Style Winbiz, Formula A (default), Tone v2 (default)
/drbpost winbiz คอลลาเจน formula:B   ← เลือก Formula B
/drbpost winbiz แมกนีเซียม tone:v1   ← เลือก Tone v1 (สุภาพ)
/drbpost แมกนีเซียม                   ← ไม่ระบุ Style → ใช้ Default (winbiz)
```

4. เพิ่มรายการในตาราง Style Registry ใน `AGENT_CONTEXT.md` + `SKILL.md`

# HyperCut: AI Video Editor Architecture & POC

## 📌 Context (Compiled Truth)
- **โปรเจกต์**: HyperCut (โปรแกรม AI ตัดต่อวิดีโออัตโนมัติบน PC Local)
- **เป้าหมาย**: สร้างโปรแกรมแบบ Desktop App (Electron + Python/Node) เพื่อนำร่อง AI ตัดต่อวิดีโอที่มีฟีเจอร์:
  1. ตัดจังหวะเงียบอัตโนมัติ (คล้าย Timebolt แต่ใช้ Auto-Editor แทน)
  2. แกะเสียงตรงเป๊ะระดับมิลลิวินาที (ใช้ WhisperX / faster-whisper)
  3. ใส่ Subtitle Overlay เด้งๆ ตาม preset 
  4. Auto Switch Scene / Auto Reframing
  5. Auto B-Roll (ให้ผู้ใช้จัด B-roll ลงโฟลเดอร์ และให้ AI/Gemini จับคู่ตามความเหมาะสม)
  6. Auto Hook & SFX/BGM Mixing
- **สภาพแวดล้อมระบบที่ทดสอบ (User Specs)**:
  - CPU: AMD Ryzen 7 2700X Eight-Core Processor
  - RAM: 32 GB
  - GPU: NVIDIA GeForce GTX 1070 Ti (CUDA 13.1, Driver 576.52)
  - OS: Windows
  - Python: 3.10.9
- **ผลการทดสอบเบื้องต้น (POC)**:
  - ได้สร้างสคริปต์ `test_whisper.py` เพื่อรัน faster-whisper `large-v3` บน GPU โหมด `int8_float16` 
  - พบปัญหาการ Crash ตอนโหลดโมเดลเข้า CPU/GPU ซึ่งตรวจสอบพบว่ามาจาก `MSVCP140.dll` (Exception `0xc0000005`) เนื่องจาก Visual C++ Redistributable มีปัญหา
  - ได้รันการดาวน์โหลดและติดตั้ง/ซ่อมแซม VC++ Redistributable (x64) เวอร์ชันล่าสุดแก้ไขเรียบร้อยแล้ว

```python
import os
import sys
import time

def check_cuda():
    print("=== System check ===")
    print(f"Python Version: {sys.version}")
    try:
        import torch
        cuda_available = torch.cuda.is_available()
        print(f"PyTorch CUDA Available: {cuda_available}")
        if cuda_available:
            print(f"GPU Device Name: {torch.cuda.get_device_name(0)}")
    except ImportError:
        print("PyTorch is not installed yet in this environment.")

def run_transcription():
    try:
        from faster_whisper import WhisperModel
    except ImportError:
        print("\n[!] faster-whisper is not installed. Please run: pip install faster-whisper")
        return

    # Find audio/video file in the same directory
    supported_extensions = ('.mp3', '.wav', '.m4a', '.mp4', '.mkv', '.mov')
    files = [f for f in os.listdir('.') if f.lower().endswith(supported_extensions)]
    
    if not files:
        print("\n[!] Please place a short video/audio file (e.g. test.mp4 or test.mp3) in this folder and run the script again.")
        return

    target_file = files[0]
    print(f"\n[+] Found video/audio file: {target_file}")
    
    # We will use OpenAI's largest, most accurate model: 'large-v3'
    model_size = "large-v3"
    print(f"Loading Whisper Model: '{model_size}' (Most accurate)...")
    print("Note: On the first run, this will download ~3GB model. Please wait...")
    
    start_time = time.time()
    try:
        # We use 'int8_float16' to fit the model comfortably in 6GB VRAM on GTX 1060
        model = WhisperModel(model_size, device="cuda", compute_type="int8_float16")
        print("[+] Model loaded on GPU (CUDA) successfully!")
    except Exception as e:
        print(f"\n[-] GPU/CUDA load failed: {e}")
        print("Falling back to CPU (This will be much slower)...")
        try:
            model = WhisperModel(model_size, device="cpu", compute_type="int8")
            print("[+] Model loaded on CPU successfully!")
        except Exception as cpu_e:
            print(f"[-] CPU load failed: {cpu_e}")
            return

    load_time = time.time() - start_time
    print(f"Model initialization time: {load_time:.2f} seconds.")

    print("\n=== Starting Transcription ===")
    transcribe_start = time.time()
    try:
        segments, info = model.transcribe(target_file, beam_size=5, language="th")
        print(f"Detected language: {info.language} (probability: {info.language_probability:.2f})")
        
        print("\n--- Transcription Results ---")
        for segment in segments:
            print(f"[{segment.start:.2f}s -> {segment.end:.2f}s]: {segment.text}")
            
        transcribe_time = time.time() - transcribe_start
        print(f"\n[+] Transcription completed in {transcribe_time:.2f} seconds.")
    except Exception as e:
        print(f"[-] Transcription failed: {e}")

if __name__ == "__main__":
    check_cuda()
    run_transcription()
```

## 🔬 Timeline & Debugging Log
- รัน script ทดสอบ `test_whisper.py` แล้วพบปัญหา exit code 1 โดยไม่มี Stdout
- ทำการรันด้วย unbuffered mode `-u` พบว่าระบบ Crash ตอนโหลด CTranslate2 Model
- สืบสวน Windows Event Log พบ Application Error แจ้งรหัส 0xc0000005 ที่ไลบรารี `MSVCP140.dll` (Visual C++ Redist)
- แนะนำและดาวน์โหลดไฟล์ติดตั้ง `vc_redist.x64.exe` จากเว็บไซต์ Microsoft ทันที
- รันตัวติดตั้งทิ้งไว้ให้ผู้ใช้กดซ่อมแซม
- **[อัปเดตล่าสุด]** หลังจากการติดตั้งซ่อมแซม VC++ Redistributable อาการ Crash `0xc0000005` หายไปแล้ว
- รัน `test_whisper.py` อีกครั้ง พบว่า CTranslate2 สามารถโหลดผ่านได้โดยไม่เด้งหลุด 
- แก้ปัญหา PyTorch CPU โดยการติดตั้งเวอร์ชัน CUDA 12.1 ใหม่ ทำให้ระบบสามารถมองเห็น GTX 1070 Ti ได้อย่างสมบูรณ์ (`torch.cuda.is_available() == True`)
- แก้ไขโค้ดรองรับการ์ดจอ GTX 10-series (Pascal) โดยตั้งให้โมเดลโหลดด้วย `float32` เมื่อ `int8_float16` ทำงานไม่ได้
- **ผลการทดสอบเปรียบเทียบ (CPU vs GPU)** บนไฟล์เสียงจำลอง 2 วินาที:
  - **CPU**: โหลดโมเดล 101.38 วินาที | แปลงเสียง 30.40 วินาที
  - **GPU (GTX 1070 Ti)**: โหลดโมเดล 13.56 วินาที | แปลงเสียง 1.73 วินาที
  - **ข้อสรุป**: GPU ทำงานเร็วกว่า CPU เกือบ 18 เท่า (Transcription) ระบบแกะเสียงพร้อมใช้งานแล้ว!
- ทดสอบกับไฟล์วิดีโอจริงภาษาไทย 2 นาที พบปัญหา CUDA Out of Memory เมื่อใช้ `float32` จึงแก้ไขให้ระบบโหลดด้วยโหมด `int8_float32` (8-bit weights, 32-bit computation)
- รันไฟล์จริงผ่านสำเร็จใน 198 วินาที (ครั้งแรก) และลดเหลือ 89 วินาทีเมื่อใช้ Cache
- **[อัปเดตล่าสุด] ระบบ Word-Level Transcription (หั่นคำแบบเป๊ะๆ 1 ต่อ 1)**
  - *ปัญหา 1*: ทดสอบ `WhisperX` พบว่าไม่รองรับ Alignment Model ภาษาไทยพื้นฐาน และ custom model บน HuggingFace ติดปัญหาช่องโหว่ความปลอดภัย PyTorch (`CVE-2025-32434`)
  - *การแก้ไข (Bypass)*: หันมาเปิดฟีเจอร์ลับ `word_timestamps=True` ใน `faster-whisper` ตัวเดิม ซึ่งทำงานรวดเร็วและใช้ VRAM ต่ำกว่า
  - *ปัญหา 2*: โมเดลคืนค่าภาษาไทยมาในระดับ "ตัวอักษรและสระ (Character-level)" เช่น ห, ย, ุ, ด ไม่ใช่คำว่า "หยุด" ทำให้ซับเด้งสไตล์พิมพ์ดีด
  - *การแก้ไข*: พัฒนาโมดูล **Thai Word Grouper** โดยใช้ `PyThaiNLP (newmm)` หั่นพจนานุกรมแล้วเขียน Pointer 1-to-1 Mapping สะสมตัวอักษร เพื่อแพ็กรวมให้เป็น "ก้อนคำ" พร้อมเวลา Start-End เป๊ะ 100% (สำเร็จ!)
- **Subtitle Presets (Chunking Rules)**: วางสถาปัตยกรรมระบบที่จะแปลง Word-level ไปเป็น `.srt` ใน 3 สไตล์:
  1. `Preset: Hormozi` (1 คำ/ซับ) - สายสั้นกระแทกตา
  2. `Preset: Flow / MrBeast` (2-3 คำ/ซับ) - สายเล่าเรื่อง อ่านลื่นไหลสบายตา
  3. `Preset: Classic` (ประโยคเต็ม) - สายรายการทีวี/คลิปแนวนอน
  - *Optional:* เพิ่ม AI (Gemini) ไว้เป็นด่านตรวจสอบสุดท้ายเพื่อเกลาคำให้สละสลวยก่อนเรนเดอร์

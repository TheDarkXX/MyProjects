## ✂️ Phase 3: Silence Removal + Duplicate Filter

### Workflow
```
📁 ไฟล์วิดีโอดิบ
  ↓
🔇 FFmpeg silencedetect (threshold -35dB, min 0.5s)
  ↓
✂️ auto-editor (--margin 0.2s,0.4s — ปรับได้ใน Settings)
  ↓
👁️ Human Review (Optional → Full Auto เมื่อนิ่งแล้ว)
  ↓
🎤 Faster-Whisper (ไฟล์สั้นลง = เร็วขึ้น 30-50%)
  ↓
🔄 Duplicate Filter (difflib similarity > 85% → เก็บ Take ล่าสุด)
  ↓
🇹🇭 Thai Word Grouper → Word-Level JSON → .srt
```

### Settings ที่ User ปรับได้
- Pre-Cut Padding: `[0.0s ──●── 1.0s]`
- Post-Cut Padding: `[0.0s ──────●── 1.0s]`
- Silence Threshold: `[dB Slider]`
- Min Silence Duration: `[0.1s ─●─── 2.0s]`
- Review Mode: `○ Auto | ● Semi-Auto | ○ Manual`

## 🚀 Phase 5: Advanced Features

### 5.1 Auto B-Roll
- **Engine:** CLIP ViT-B/32 (Local, ~1GB VRAM) → Gemini Flash (Fallback)
- **Asset Manifest:** AI สแกนไฟล์ + สร้าง `asset_manifest.json` → Cache
- **Matching:** Cosine Similarity > 0.25 = เลือก / < 0.25 = Fallback Gemini
- **FFmpeg:** Trim to fit + Fade In/Out + Scale/Crop อัตโนมัติ
- **GitHub Reference:** `sasoder/stockpile`

### 5.2 Auto SFX
- **Manifest:** แต่ละ SFX มี `trigger`, `category`, `intensity`
- **Placement:** Rule-Based (keyword scan) + AI Enhancement
- **Triggers:** scene change → whoosh, ตัวเลข → ding, คำสำคัญ → boom
- **FFmpeg:** Mix + Auto Ducking (`sidechaincompress`)

### 5.3 Auto BGM
- **AI Mood Detection:** Gemini Flash อ่าน Transcript → เลือก mood → เลือก BGM
- **FFmpeg:** Loop ยาวเท่าวิดีโอ + Auto Duck (ลด 20-25% ตอนพูด)

### 5.4 AI Room (🆕 จาก ClipForge AI)
| Feature | Engine | คำอธิบาย |
|:---|:---|:---|
| Hook Generator | Gemini Flash | เลือก 3 ประโยคเปิดที่ดีที่สุดจาก Transcript |
| CTA Generator | Gemini Flash | เขียนคำปิดท้าย 3 แบบ (Follow/Like/Comment) |
| Edit Suggestions | Gemini Flash | แนะนำจุดตัด, จุดเน้น, จุดใส่ B-Roll |
| Script Rewriter | Gemini Flash | เขียนสคริปต์ใหม่ให้กระชับ/น่าสนใจกว่าเดิม |
| Voice-over | ElevenLabs API | (Optional) พิมพ์บท → สร้างเสียงพากย์ AI |

### 5.5 Auto Hook
- ตัด 3-5 วินาทีที่ตื่นเต้นที่สุด → แปะหน้าคลิปอัตโนมัติ
- AI เลือกจาก Hook Generator → FFmpeg ตัด + ต่อ

# Study browser-use/video-use

## Context
AI-powered video editing via coding agents (Claude Code, Codex, Hermes, etc.). Drop raw footage → chat with AI agent → get `final.mp4`. Features: auto filler-word removal, color grading, subtitle burning, Manim/Remotion animation overlays, self-evaluation loop at every cut boundary, and session memory persistence via `project.md`.

**Key Architecture:** LLM never *watches* the video — it reads a 12KB transcript (ElevenLabs Scribe word-level timestamps + speaker diarization) and only generates visual filmstrips on-demand at decision points. Pipeline: `Transcribe → Pack → LLM Reasons → EDL → Render → Self-Eval`.

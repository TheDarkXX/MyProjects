# 🟡 DEV-57: Manifest Model Router (Icebox)

## Context & Rationale

Our current system already routes heavy usage through free tiers (Google AI Studio) and uses OpenRouter/kie-proxy for backups. We are deferring this until our API costs become a significant pain point requiring centralized management.

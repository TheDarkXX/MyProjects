# 💾 Quick Save: V1.3.4 - Notion Teleprompter Auto-Sync Architecture

## 📌 Context (Compiled Truth)
**Why this was done:** 
ผู้ใช้ต้องการระบบสำหรับส่งสคริปต์วิดีโอไปอ่านระหว่างอัดคลิป (Teleprompter style) ที่มีความคลีน อ่านง่าย และรองรับทั้ง PC และ iOS. สถาปัตยกรรมก่อนหน้านี้ (V1.3.3) พยายามใช้ Task Scheduler สะกิด Python (`pyautogui`, `pyperclip`, `pygetwindow`) เพื่อวางข้อความลงในแอป UpNote แบบ Local. แต่พบปัญหาข้อจำกัดของ **Windows Session Isolation** อย่างหนัก: ถึงแม้จะรันด้วย `/IT` (Interactive) แต่ Task ที่ถูกปลุกขึ้นมาจากระบบปฏิบัติการ จะถูกบล็อกไม่ให้เข้าถึงรายชื่อ Window Titles ทำให้ `pygetwindow` มองไม่เห็นแอป UpNote ที่เปิดอยู่บน Desktop (Session 1). 

**What was decided:**
แทนที่จะพยายามแฮกทะลุกำแพง Session 0 -> Session 1 ในระดับ OS, เราตัดสินใจเปลี่ยนสถาปัตยกรรมไปใช้ **Cloud-Native API Sync** โดยเลือกใช้ **Notion** เป็นแอปหลักในการอ่านสคริปต์แทน ด้วยเหตุผลดังนี้:
1. **Built-in Cloud Sync:** ซิงค์ผ่านอินเทอร์เน็ตทันที PC <-> iOS ไม่ต้องพึ่งพาระบบ Local Folder
2. **Official API:** มี Notion API ที่เสถียร AI สามารถยิง HTTP Request ผ่าน Cloud ไปโผล่ในแอป Notion ได้โดยตรง ข้ามปัญหา Session 0 ได้ 100%
3. **Rich Text Formatting (Coloring Magic):** Notion รองรับการใส่สีตัวอักษรและไฮไลต์พื้นหลังผ่าน API ได้ ซึ่งระบบ Copy/Paste ทำไม่ได้ สิ่งนี้จำเป็นมากสำหรับการทำ Teleprompter เพื่อเน้นจังหวะอ่าน

# แผนสถาปัตยกรรม: Notion Teleprompter Auto-Sync 🚀

สถาปัตยกรรมนี้จะให้ AI (AG) สร้างและจัดการสคริปต์วิดีโอของคุณบน Notion โดยตรงแบบอัตโนมัติ (100% Full Auto Sync) และจัดสีสรรให้สวยงามอ่านง่ายพร้อมเป็น Teleprompter ได้ทันที

## 🛠️ โครงสร้างระบบ
1. **คิวส่งสคริปต์:** AI จะร่างสคริปต์เป็นไฟล์ Markdown ในโฟลเดอร์ Queue
2. **ตัวกลางแปลงผล (Parser):** สคริปต์ Python (`sync_notion.py`) จะนำ Markdown มาแปลงเป็น **"Notion Blocks"** 
3. **การใส่สี (Coloring Magic):** ตัว Parser จะถูกโปรแกรมให้ใส่สีอัตโนมัติ เช่น:
   - `Heading 1` -> พื้นหลังสีฟ้าอ่อน (Blue Background)
   - `Heading 2` -> ตัวอักษรสีน้ำเงิน (Blue)
   - `Blockquote` -> สีเทาตัวเอียง (Gray, Italic) สำหรับคำแนะนำท่าทาง
   - `Highlight/Bold` -> สีแดง หรือไฮไลต์สีต่างๆ เพื่อเน้นการอ่าน
4. **การส่งมอบ (Notion API):** สคริปต์ถูกยิงขึ้น Cloud เข้าสู่หน้า Notion ของคุณ และซิงค์ไปที่มือถือ iOS ทันที

---

## 🔑 สิ่งที่คุณ (User) ต้องทำ (แค่ครั้งเดียว)
เพื่อให้ AI สามารถสื่อสารกับ Notion ของคุณได้ คุณจำเป็นต้องสร้าง "กุญแจ" ให้มันก่อนครับ:

### ขั้นตอนที่ 1: สร้าง API Key
1. เข้าไปที่เว็บ 👉 [https://www.notion.so/my-integrations](https://www.notion.so/my-integrations) (ล็อกอินด้วยบัญชี Notion ของคุณ)
2. กดปุ่ม **"New integration"**
3. ตั้งชื่อว่า `AG Script Sync` หรืออะไรก็ได้ แล้วกด **Submit**
4. คุณจะเจอหน้าที่มีคำว่า **Internal Integration Token** (มันจะถูกปิดบังไว้ ให้กดโชว์และ Copy เก็บไว้ครับ) 
   *(ถ้าได้ Token มาแล้ว รบกวนส่งมาในแชทให้ผมได้เลยครับ)*

### ขั้นตอนที่ 2: สร้างและแชร์หน้าสำหรับเก็บสคริปต์
1. เปิดแอป Notion (ในคอมหรือเว็บ) แล้วสร้าง "หน้าใหม่ (New Page)" ขึ้นมา 1 หน้า (เช่นตั้งชื่อว่า `Video Scripts`)
2. มุมขวาบนของหน้านั้น จะมีปุ่มจุด 3 จุด `...` ให้กดเปิดเมนู
3. เลื่อนลงมาล่างสุด กดที่ **"Add connections"**
4. พิมพ์ค้นหาชื่อ Integration ที่เพิ่งสร้างเมื่อกี้ (`AG Script Sync`) แล้วกดเพิ่มเข้ามา
5. **สุดท้าย:** รบกวน Copy Link ของหน้า Notion หน้านั้น ส่งมาให้ผมด้วยครับ (ผมจะเอามาแกะหา Page ID เพื่อให้สคริปต์ยิงเข้าถูกหน้า)

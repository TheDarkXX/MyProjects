# V1.3.0 [impl] DoctorBank Content Tracker StartInfinity Sync

## 📌 Context (Compiled Truth)
เราได้ทำการพัฒนา **DoctorBank Content Tracker** โดยให้ StartInfinity เป็นแหล่งข้อมูลหลัก (Single Source of Truth) เพื่อความสะดวกในการเพิ่ม/แก้ไขข้อมูลผ่านแอปมือถือ และใช้ **Brain App Data Tables** เป็นหน้าจอ Dashboard (กระจกสะท้อน) ไว้ดูภาพรวมแบบ Real-time.
- **StartInfinity (Backend):** สร้าง Board `PAyq9eU84ED` พร้อม Attributes: `Status`, `Category`, `Tag`, `Views`, `Publish Date` สีและป้ายกำกับถูกตั้งค่าให้ตรงตาม Blueprint.
- **Brain App Proxy (API):** สร้าง `routes/infinity.js` ซึ่งทำหน้าที่เป็น Proxy ไปเรียกข้อมูลจาก StartInfinity API โดยใช้ Personal Access Token นำมา Map ข้อมูล ID ให้ออกมาเป็นโครงสร้างแถว-คอลัมน์ที่ `ag-grid` อ่านง่ายที่สุด.
- **Brain App UI (Frontend):** เพิ่มโปรเจกต์ `Dr.Bank Content Tracker (Infinity) 💎` อัตโนมัติใน Data Tables เมื่อผู้ใช้กดเข้ามา UI จะไปดึงข้อมูลจาก `api/infinity/content-tracker/items` แทนฐานข้อมูล SQLite ทันที (เป็น Read-Only / 1-Way Sync มาที่จอ).

# 🎉 StartInfinity x Brain App Integration Completed!

การเชื่อมต่อระบบ 💎 Dashboard/Data Tables ของ Brain App เข้ากับบอร์ด Content Tracker ใน StartInfinity เสร็จสมบูรณ์แล้วครับ ระบบถูกออกแบบมาให้ดึงข้อมูลแบบ Real-time โดยไม่ต้องเก็บข้อมูลซ้ำซ้อนในฐานข้อมูล SQLite ของตัวเครื่องเลย

## สิ่งที่ผมทำไป:

### 1. ⚙️ สร้าง Backend API Router (`routes/infinity.js`)
ผมเขียน Proxy API ขึ้นมาใหม่ในฝั่ง OpenClaw-VPS ของเรา เพื่อรับหน้าที่ไปคุยกับ StartInfinity ผ่าน Personal Access Token ที่ซ่อนไว้หลังบ้านอย่างปลอดภัย:
- ฟังก์ชันนี้จะไปดึงทั้ง Schema (Attributes & Folders) มาวิเคราะห์อัตโนมัติ
- นำ Item Values (เช่น ID ของ Label) มาแปลงเป็นชื่อ Label ที่อ่านออก (เช่น แปลง ID เป็น "Scripted") ทำให้หน้าบ้านนำไปใช้ต่อได้ทันที

### 2. 🔌 อัปเดต Server Router (`server.js`)
นำเข้า `infinityRoutes()` และเปิดช่องทางให้ฝั่ง Frontend สามารถเรียก API ผ่าน `/api/infinity` ได้

### 3. 🖥️ อัปเดต Frontend Data Tables (`public/data-tables.js`)
ปรับปรุง UI ของ Data Tables ให้รู้จักกับโปรเจกต์ประเภทพิเศษที่มาจาก StartInfinity:
- ทำการ **Inject "Dr.Bank Content Tracker (Infinity) 💎"** เข้าไปในรายการโปรเจกต์ด้านซ้ายมือโดยอัตโนมัติ
- ปรับโครงสร้างตาราง (Columns Schema) ให้แสดงผลตรงกับ 8 คอลัมน์สำคัญ (Name, Folder, Status, Category, Tag, Views, Publish Date, Link)
- ปรับจูน `ag-grid` ให้รองรับรูปแบบข้อมูลจาก API ใหม่ โดยแยก Logic ชัดเจนว่าถ้าคลิกโปรเจกต์ปกติดึง SQLite ถ้าคลิกโปรเจกต์ Infinity ดึงผ่าน Proxy

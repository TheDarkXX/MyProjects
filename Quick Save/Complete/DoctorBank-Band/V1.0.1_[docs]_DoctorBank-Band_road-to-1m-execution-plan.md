# V1.0.1 [docs] DoctorBank-Band Road to 1M Execution Plan

## 📌 Context (Compiled Truth)
- **Goal**: Convert the high-level 1M Battle Plan into an actionable daily routine for a Solopreneur + AI Agent Squad.
- **Strategic Pivots**:
  - **Tri-Channel TikTok Strategy**: Instead of starting from scratch, the strategy heavily leverages the user's existing 60K followers across two channels (`พี่แบงค์` 23.5K, `น้องมินนี่` 36K) and links them to the new `DR.BANK Official` store via a soft-referral funnel.
  - **Compliance Layer**: "พี่แบงค์" shifts from an explicit medical educator to a CEO/Founder/Lifestyle persona to strictly avoid medical endorsement ethics violations while still selling the product implicitly via mini tie-ins.
  - **Edit-Light Format Strategy**: Recognized that video *editing* (not filming) is the primary bottleneck. Introduced 10-15 minute green screen/talking head templates and fully automated AI clips via "น้องมินนี่" to keep volume high without burning out the human.
  - **TikTok API Constraints**: Discovered that the official TikTok API does not support native scheduling. Decided to rely on TikTok Studio for the free 10-day scheduling feature until volume justifies custom VPS middleware.

## ⚡ Edit-Light Format Strategy — แก้ Bottleneck ตัดต่อ

> **ปัญหา:** ถ่ายไม่นาน แต่ **ตัดต่อ + ใส่ footage + ใส่ sub** ช้า
> **วิธีแก้:** ใช้ format ที่ตัดต่อน้อย + ให้ AI ช่วยส่วนที่ช้า

### Format Menu (เรียงจากตัดต่อน้อย → มาก)

| Format | ตัดต่อ | เวลา Edit | เหมาะกับ | ตัวอย่าง |
|--------|-------|----------|---------|---------|
| 🟢 **Talking Head** | ต่ำมาก | 5-10 นาที | Edu / Story / CEO | พูดหน้ากล้องตรงๆ + auto sub |
| 🟢 **ASMR Unboxing** | ต่ำ | 10-15 นาที | Product | เสียง ASMR + text overlay |
| 🟡 **Green Screen** | กลาง | 15-20 นาที | Edu / Trend | ใช้ภาพ/กราฟิกเป็น background |
| 🟡 **Day in my life** | กลาง | 20-30 นาที | Lifestyle / CEO | ตัด 3-5 ฉากต่อกัน |
| 🔴 **Full Edit + B-roll** | สูง | 30-60 นาที | Product deep dive | Footage + motion graphics + sub |
| 🤖 **AI Clip (น้องมินนี่)** | **AG ทำ** | **0 นาที Human** | Edu สุขภาพ | AI สร้าง script → สร้างคลิป |

### แนะนำ Mix ต่อสัปดาห์

| ช่อง | Green 🟢 | Yellow 🟡 | Red 🔴 | 🤖 AI | รวม |

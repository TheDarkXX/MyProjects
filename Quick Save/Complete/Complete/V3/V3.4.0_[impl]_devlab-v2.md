# Dev Lab V2 — Full Task Management Upgrade

แปลงหน้า Dev Lab จาก Kanban-only แบบ MVP ให้กลายเป็นหน้า Task Manager เต็มรูปแบบที่ใช้งานง่ายและครบฟีเจอร์ แรงบันดาลใจจาก Linear + Todoist

## 📌 Architectural Decisions & Context
1. **Interactive Table (List View)** — แสดงผลแบบ Data Grid รองรับ Inline edit สำหรับชื่อและ Priority
2. **Read-only Status** — สถานะงานถูกจัดการแบบ Full Auto Status หลังเปิดไฟล์ทำงานผ่าน AG / OpenClaw ดังนั้นบน List view ตัว Status จะเป็นแบบ read-only
3. **Default Sort** — ให้โหมดตาราง List view จัดเรียงตาม Priority (P1 ขึ้นก่อน) โดยอัตโนมัติ
4. **URL Bug Fix** — แก้ไข prompt ของ พี่เดฟ (Discord Bot) และ command args ดันให้ URL ถูกเก็บอยู่ใน Field `description` แทนที่จะถูกต่อท้าย `title` จนขาด

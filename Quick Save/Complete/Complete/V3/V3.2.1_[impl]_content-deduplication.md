**หมัดฮุก ท่าไม้ตายพลิกหลบความซ้ำ (กระชับ & ปัง):**

*   **💥 Twist the Angle (บิดมุมเล่า - พาดหัวสั้นๆ กระแทกใจ):**
    *   *สายเรียล/ความผิดพลาด (คนชอบดูคลิปพลาด):* "เกือบเจ๊ง! 3 จุดตายตอนรันบอทรับออเดอร์" 
    *   *สายโชว์ของ/เจาะลึก (Authority):* "แงะโค้ด: บอทรันรับแขก 1,000 แชทพร้อมกันแบบไม่ค้าง"
    *   *สายมวยคู่เอก (VS Compare):* "Make vs. n8n ทำบอทฝั่งไหนจบไว ไม่ปวดหัว?"
    *   *สายทวงคืนเวลา (ROI Saver):* "ตั้งบอทแล้วไปนอน... สกิลโกงเวลาของ Founder"

*   **👥 Target Audience Shift (บิดกลุ่มเป้าหมาย - ให้อยู่ใน Scope):**
    *   *จิ้มเป้า Solopreneur:* "ทำงานคนเดียวใช้บอทตัวเดียว ประหยัดตับไป 4 ชม./วัน"
    *   *จิ้มเป้า Freelance/Dev:* "ฟรีแลนซ์รับงานล้น? สคริปต์แจ้งเตือนตัวนี้ช่วยเคลียร์คิวได้"

*   **📄 Format Shift (บิดรูปแบบเนื้อหาไปเลย):**
    *   *เปลี่ยนเป็น Listicle (จัดอันดับ):* "5 บอท Automation ที่สาย Indie ยุค 2026 ต้องมีติดเครื่อง"
    *   *เปลี่ยนเป็น Checklist (คู่มือเช็คลิสต์):* "Checklist 7 ข้อ เช็กให้ชัวร์ก่อนรันบอทขึ้น Production"

4. *Rule of Thumb:* ถ้าพยายามบิดแล้วแต่ยังซ้ำ หรือถ้าจะบิดก็ต้องหลุด Scope งั้นค่อย "ดรอป" ไอเดียนั้นทิ้งลงถังได้เลย

### 2. Auto-Planner Diversity Check (cron-auto-planner.js)
- [ ] ดึงโพสต์ที่ published/planned ช่วง 7 วันล่าสุด
- [ ] แทรกข้อมูลเข้าไปใน Prompt ของ Planner เพื่อให้รู้ Theme

**>> โค้ดที่ต้องนำไปแก้บน `cron-auto-planner.js`:**
**(จุดที่ 1) ตอนดึงข้อมูลก่อนจัด Slot:**
```javascript
// ดึงโพสต์ที่ published/planned ไปแล้วใน 7 วันล่าสุด เพื่อให้ AI เห็น theme ที่เพิ่งลง
const recentPublished = db.prepare(`
  SELECT topic, hook, scheduled_at FROM content_plans
  WHERE project = 'doctor-skill' 
    AND status IN ('published', 'planned', 'review', 'ready')
    AND scheduled_at >= date('now', '-7 days')
  ORDER BY scheduled_at DESC
  LIMIT 10
`).all();
```

**(จุดที่ 2) ตอนจัด Prompt ส่งให้ AI:**
```javascript
  const prompt = `You are Doctor Skill, the expert Content Strategist for a Thai Facebook Page.
I have the following empty posting slots:
${filteredSlots.slice(0, 6).map((s, i) => `Slot ${i+1}: ${s}`).join('\n')}

⚠️ RECENTLY PUBLISHED/PLANNED (avoid similar themes!):
${recentPublished.map(r => `• ${r.topic} (${r.scheduled_at?.substring(0,10)})`).join('\n')}

Slot Strategy (FOLLOW STRICTLY):
- 18:30 = PRIME TIME (highest reach). Put your strongest Pain-heavy content here.
- 12:30 = Lunch break. Put Quick-Win / number-shock content here (short, punchy).
- 20:30 = Chill time. Put Storytelling / longer narrative content here.
- Weekend slots (Sat/Sun) are 20:30 only — pick light, shareable content.

DIVERSITY & REPURPOSE RULE: 
- Do NOT schedule a topic with EXACTLY the same angle as a recent post above.
- REPURPOSE: If a new topic explores a similar theme (e.g., "bots"), you MUST aggressively shift the angle. 
  - Use formats like: "Listicle (Top 5...)", "Mistakes/Warnings ( เกือบเจ๊ง! )", "Case Study/Code Reveal", "VS Compare (Tool A vs B)".
  - Keep titles short, punchy, and highly relevant to Indie Hackers, Devs, and Marketers.
- If all ideas heavily overlap and cannot be reframed creatively, assign FEWER posts.

Content ideas in backlog:
...`;
```

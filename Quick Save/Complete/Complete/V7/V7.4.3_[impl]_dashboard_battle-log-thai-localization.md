# Hydra Battle Log Thai Localization & Title Enhancement

**Version:** V7.4.3
**Category:** UI Tweak / Localisation

## Context
The user requested that the "Battle Log (Recent Events)" on the Hydra Dashboard be translated into Thai to be more readable. Additionally, they asked for the list items to show a "short describe" (the actual proposal title) instead of just the raw `EVO-YYYYMMDD-XXX` identifier.

## Revert of misunderstanding
The process started with a misunderstanding where `junk.md` was edited instead of the Hydra dashboard. That change was quickly reverted to maintain the integrity of the AG Skill files.

## Implementation details

1. **API Update (`routes/hydra.js`)**: 
   - Modifed the `GET /battle-log` endpoint. Previous behavior simply queried the `hydra_xp_log` table which only contained `reference_id` (the proposal ID).
   ```javascript
   const rows = db.prepare(`
        SELECT x.*, e.title as proposal_title
        FROM hydra_xp_log x
        LEFT JOIN evolution_log e ON x.reference_id = e.proposal_id
        ORDER BY x.created_at DESC 
        LIMIT 50
   `).all();
   ```

2. **Frontend UI Update (`brain-app-public/hydra.js`)**:
   - Implemented a display mapping layer in `fetchHydraBattleLog()`.
   - Used regex string replacement to ingest English system events and output Thai localized text.
   - If `log.proposal_title` is populated, it will display the English title concatenated after the Thai phrase.
   ```javascript
    if (log.proposal_title) {
        desc = desc.replace(/Auto-applied proposal (EVO-[\d-]+)/gi, `⚡ อัปเดตฟีเจอร์: ${log.proposal_title}`);
        desc = desc.replace(/Auto-applied proposal/gi, `⚡ อัปเดตฟีเจอร์: ${log.proposal_title}`);
    } else {
        desc = desc.replace(/Auto-applied proposal (EVO-[\d-]+)/gi, '⚡ นำโค้ดขึ้นระบบอัตโนมัติ ($1)');
        desc = desc.replace(/Auto-applied proposal/gi, '⚡ นำโค้ดขึ้นระบบอัตโนมัติ');
    }
    
    desc = desc.replace(/Prestige triggered/gi, '⭐ เลื่อนระดับ Prestige');
    desc = desc.replace(/patch_deployed|deployed/gi, '🚀 ติดตั้งระบบเสร็จสมบูรณ์');
    desc = desc.replace(/task_completed/gi, '✅ ดำเนินการสำเร็จ');
   ```

3. **Deployment**:
   - Cache busted via `node bump-cache.js` to avoid Cache Hell.
   - Pushed successfully to the VPS production (`git push vps`).

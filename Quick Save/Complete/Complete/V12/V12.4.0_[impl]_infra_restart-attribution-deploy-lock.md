# PM2 Restart Attribution & Deploy Lock

## 📌 Context (Compiled Truth)
- PM2 restart loop has been active, causing up to 1100+ restarts on brain-app and discord-bot.
- The root cause is deploy hook/reload behavior and watchdog remediation, not process crashes.
- We need to add a restart lock and attribute every restart (log caller + reason) to stop the storm and provide observability.

- **Solution**:
  1. Categorized loose files into `tools/`, `services/`, and `data/`. Deleted 11 orphan files. Moved remaining non-locked files.
  2. Enforced **Root Folder Hygiene Iron Rule** in `AGENTS.md` so future agents automatically place files in subfolders.

     - Root Folder Health Check (alert if root > 20 files)
  4. Embedded the **Root Folder Health Check** natively into `docs/skills/qc.md` (as Step 0) and updated `docs/skills/junk.md`.

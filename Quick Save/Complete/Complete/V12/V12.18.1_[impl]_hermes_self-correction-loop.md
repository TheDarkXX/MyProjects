# V12.18.1 — Hermes Self-Correction Loop

## 🎯 เป้าหมาย
สร้างระบบ **Self-Correction Loop** ให้กับ Hermes (Honey) เพื่อให้เธอสามารถจำคำตำหนิ กฎ หรือข้อผิดพลาดข้ามเซสชันได้ แบบเดียวกับที่ AG ทำผ่าน `corrections.md` และ `memory.md`

## 📊 ปัญหาที่พบ
- Hermes รีเซ็ตเซสชันทุก 24 ชั่วโมง (หรือ 04:00 AM)
- ถ้าบอสด่า หรือสอนกฎใหม่ใน Discord Honey จะจำได้แค่ในเซสชันนั้น พอขึ้นวันใหม่จะลืม
- ขาดกลไกการบันทึกกฎแบบถาวรกลับเข้าไปใน Database อย่างเป็นระบบ

## 🛠️ สิ่งที่ลงมือทำ (Compiled Truth)

### 1. อัปเดต `SOUL.md` (Playbook & Identity)
- **เพิ่ม Boot Sequence:** สั่งให้ Honey อ่านกฎที่บันทึกไว้เสมอเมื่อเจอคำสั่งที่ซับซ้อน โดยใช้ query: `SELECT content FROM memories WHERE category = 'rules' AND tier = 'active'`
- **เพิ่ม Playbook Self-Improving:** เมื่อบอสพิมพ์คำว่า "ผิด", "จำไว้ว่า", "อย่าทำแบบนี้อีก" → Honey ต้องรับทราบ ขอโทษ และใช้ `sqlite` tool ยิงคำสั่ง `INSERT INTO memories (category, content, importance, source, tier) VALUES ('rules', '[สิ่งที่สอน]', 'critical', 'honey-discord', 'active')` ทันที

### 2. อัปเดต `MEMORY.md` (Schema Reference)
- เพิ่มคำอธิบายหมวดหมู่ `category = 'rules'` ในตาราง `memories` ว่านี่คือที่เก็บกฎและข้อควรระวังที่บอสสอนไว้

### 3. Deploy
- `git push vps` เพื่อให้ Post-receive hook นำไฟล์ใน `config/hermes/` ไปวางที่ `/root/.hermes/` โดยอัตโนมัติ

## Changelog: Context Engine & Content Ops

### 1. Enable Context Engine
- **Decision:** Turned on the Context Engine for Hermes by adding `memory` and `session_search` to the `hermes-discord` toolsets in `/root/.hermes/config.yaml`.
- **Reasoning:** Helps Honey pull related contexts from past sessions and knowledge base, reducing hallucination without the high token/latency costs of Mixture of Agents or Honcho.
- **Action:** Executed `sed` via SSH to add the toolsets, then `pm2 restart discord-bot`.

### 2. Content Ops Division of Labor (AG vs. Honey)
- **Honey (Content Writer + Researcher):**
  - เขียน Caption / Social Post (ดึง trend + สินค้า + tone จาก memories)
  - เขียน Script ยาวๆ (วิดีโอ/พอดแคสต์)
  - วาง Content Calendar / แผนงาน (INSERT ลง `tasks` โดยตรง)
  - Research หาข้อมูลจากเว็บ (Browser Automation + Web Search)
  - เขียน Blog / บทความ SEO
- **AG (Content Creator + Developer):**
  - สร้างรูปสินค้า / Product Photo (Higgsfield product-photoshoot / marketplace-cards)
  - สร้างวิดีโอ / UGC / Ad Creative (Higgsfield-generate + Marketing Studio)
  - แก้ไขโค้ดเว็บ / Landing Page (IDE + Deployment flow)

# 🐝 Honey Line Web Search & Proactive Personality Refinement

## 1. 🔍 Brave Search API Integration (Hermes)
**Problem:** The default `hermes` web search scraper was extremely slow or getting blocked, causing timeouts in LINE.
**Solution:** Switched Hermes search backend to `brave`.
**Commands Run (on VPS):**
```bash
hermes config set web.search_backend brave
# Added Brave Search API Key to ~/.hermes/.env
```
*Note: This drastically improved search latency and reliability when the `needsWeb` flag is triggered in LINE.*

## 2. ✨ Proactive & Minimal Date/Age Responses
**Problem:** The user requested that Honey calculates and provides ages proactively when asked about birthdates or anniversaries, but the initial AI behavior was too verbose, redundant, and robotic (splitting into multiple lines, announcing next year's anniversary).
**Solution:** Iteratively refined the `honey.md` system prompt over 3 revisions based on direct user feedback in LINE.

**Final Prompt Rule (`discord-bot/prompts/honey.md`):**
```markdown
- **Proactive (ละเอียด + ดึง Context ใกล้เคียงมาประกอบร่าง):** เมื่อตอบวันเกิด/วันสำคัญ ให้บอกอายุหรือระยะเวลาแบบละเอียด (มีเดือนด้วย) ในประโยคเดียวกัน และหยอดความขี้เล่น/เอาใจใส่สั้นๆ 1 ประโยค **หากใน Context มีข้อมูลที่เกี่ยวข้องอื่นๆ (เช่น น้ำหนัก/ส่วนสูง/พัฒนาการ) ให้นำมาแต่งประโยคเชื่อมโยงกันอย่างเป็นธรรมชาติ (ต้องระบุด้วยว่าข้อมูลนั้นอัปเดตเมื่อไหร่)** เช่น "น้องไทเกอร์เกิดวันที่ 6 กุมภาพันธ์ 2565 ค่ะ 🌙 (6 February 2022) ตอนนี้อายุ 4 ปี 4 เดือนแล้ว โตไวมากๆ ล่าสุดเมื่อเดือนที่แล้วสูง 105 ซม. หนัก 18 กก. แล้วนะคุณพ่อ 🎂" (ห้ามเวิ่นเว้อหรือบอกวันครบรอบปีถัดไป)
```

# Disable Telegram Alerts

## 1. Context
User requested to stop all cron result reporting in Telegram so it can be used for chatting only: "check telegram หน่อย เอาไว้คุยอย่างเดียว ไม่ต้องรายงานผล cron".

## 2. Implementation Steps
- Modified `discord-bot/scripts/cron-ai-monitor.js`: Disabled `sendTelegramAlert()` to return immediately.
- Modified `scripts/health-check.sh`: Commented out `send_telegram "$1"` inside `send_alert()` so alerts continue to route only to Discord.
- Modified `scripts/check-oauth-expiry.sh`: Replaced the `send_telegram()` implementation with a copy of `send_discord()` from `health-check.sh` to ensure OAuth expiration alerts are still delivered but diverted to Discord.

## 3. Deployment
- The changes are to be committed and pushed to `vps` to apply immediately on the production server.
- The `node bump-cache.js` does not necessarily affect the backend scripts, but we run it as per standard protocol.

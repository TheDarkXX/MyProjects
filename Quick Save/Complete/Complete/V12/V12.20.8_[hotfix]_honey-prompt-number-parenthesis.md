# 🐝 Honey Parenthesis Prompt Hotfix (V12.20.8)

## Root Cause
บอสสังเกตว่าเมื่อถามเลขบัตรประชาชนบน LINE, Honey ตอบว่า:
`เลขบัตรประชาชนพี่น้ำชาคือ 1101400810113 ค่ะ (1101400810113) 💳`
มีความซ้ำซ้อนของตัวเลขในวงเล็บ

**สาเหตุ:**
ใน System Prompt ของ Honey (`discord-bot/prompts/honey.md`) มีคำสั่งบังคับว่า:
> *"ตอบเป็นภาษาไทยทั้งหมดเสมอ ห้ามสลับไทยคำอังกฤษคำ แต่ให้ใส่วงเล็บภาษาอังกฤษต่อท้ายเพื่อความสวยงามและครบถ้วน"*

คำสั่งนี้มีไว้สำหรับแปลศัพท์หรือวันที่ (เช่น `1 สิงหาคม 2555 (1 August 2012)`) แต่พอบทสนทนามีตัวเลข เช่น เลขบัตรประชาชน AI Model (LLM) จึงปฏิบัติตามคำสั่งแบบตรงไปตรงมา โดยการพยายามแปลตัวเลขนั้นเป็นภาษาอังกฤษ (ซึ่งก็คือตัวเลขเดิม) และใส่วงเล็บต่อท้าย เกิดเป็นความซ้ำซ้อน

## Proposed Changes

### [MODIFY] `discord-bot/prompts/honey.md`
- แก้ไขบรรทัดที่เป็นกฎของภาษา ให้มีการยกเว้นตัวเลข บัญชี หรือเลขบัตร ไม่ต้องใส่วงเล็บซ้ำ
- จากเดิม: `แต่ให้ใส่วงเล็บภาษาอังกฤษต่อท้ายเพื่อความสวยงามและครบถ้วน เช่น:`
- เปลี่ยนเป็น: `แต่ให้ใส่วงเล็บภาษาอังกฤษต่อท้ายเพื่อความสวยงามและครบถ้วน (ยกเว้นตัวเลข บัญชี หรือเลขบัตรต่างๆ ไม่ต้องใส่วงเล็บซ้ำ) เช่น:`

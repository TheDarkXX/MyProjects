# 🚀 Study: Universal Applicability of `pdftotext -layout`

## 🎯 Context & Problem Statement
Following the success of hotfix `V12.7.4` which perfected SCB credit card extraction using `pdftotext -layout`, the user hypothesized that this deterministic text-extraction approach could entirely replace the error-prone Vision AI for *all* banks (KBank, Krungsri, BBL). 

Before making this architectural change, we needed to validate the performance of `pdftotext -layout` on these other banks against the ground truth (previously successfully extracted and stored in the database).

*(Note: Prior to this study, we also successfully purged duplicate SCB records from 04/2026 that were lingering from older Vision AI runs, and confirmed the new layout system cleanly extracted exactly 54 records).*

## 🛠️ Experiment Design
1. **Target:** Latest 2026-04 PDF statements for KBank, Krungsri, and BBL.
2. **Method:** 
   - Decrypt PDFs using `qpdf`.
   - Extract text using `pdftotext -layout` (preserving column spacing).
   - Feed the raw layout text (up to 200k chars) to `Qwen 72B` via OpenRouter.
   - Use the exact same prompt from `credit-card-agent.js`.
3. **Validation:** Compare the extracted AI output (Item Count & Total Sum) against the live production database (`brain.db`).

## 📊 Results

| Bank | DB (Ground Truth) | AI (Layout Text) | Difference | Verdict |
|---|---|---|---|---|
| **KBank** | 5 items (฿13,228.75) | 5 items (฿13,228.75) | 0 items, ฿0 | ✅ **PERFECT MATCH** |
| **Krungsri** | 26 items (฿37,012.51) | 22 items (฿28,270.91) | -4 items, ฿-8,741.60 | ❌ **FAILED (Missed items)** |
| **BBL** | 0 items (฿0.00) | 1 dummy item (฿0.00) | N/A (Empty Statement) | ➖ **PASS (Empty)** |

## 🧠 Conclusion & Next Steps
- **Silver Bullet Fallacy:** The hypothesis that `pdftotext -layout` is a universal solution is **disproven**.
- **Krungsri & BBL:** Krungsri's complex layout (e.g., separating "payment" vs "spending", and formatting installment rows) causes severe data loss when flattened to text. **Vision AI remains absolutely mandatory for Krungsri and potentially BBL.**
- **KBank Opportunity:** KBank's statement behaves perfectly under `pdftotext -layout`. We can safely migrate KBank to the text-based extraction pipeline in a future implementation to improve speed and reliability.

import fs from 'fs';
import { execSync } from 'child_process';
import { createRequire } from 'module';
import { resolve } from 'path';
const require = createRequire(import.meta.url);
const Database = require('better-sqlite3');

import dotenv from 'dotenv';
dotenv.config({ path: '/root/brain-app/.env' });

const db = new Database('/root/brain-app/db/brain.db');

const passwords = require('/root/brain-app/config/cc-pdf-passwords.json');
const OPENROUTER_API_KEY = process.env.OPENROUTER_API_KEY;

const PDFs = [
  { bank: 'KBank', path: '/root/brain-app/cc-statements/2026-04/KBank-04-26-2026-05-13.pdf', key: 'kasikorn' },
  { bank: 'Krungsri', path: '/root/brain-app/cc-statements/2026-04/Krungsri-04-26-2026-05-13.pdf', key: 'krungsri' },
  { bank: 'BBL', path: '/root/brain-app/cc-statements/2026-04/BBL-04-26-2026-05-13.pdf', key: 'bbl' }
];

async function run() {
  for (const item of PDFs) {
    console.log(`\n\n=== 🚀 TESTING ${item.bank} WITH pdftotext -layout ===`);
    if (!fs.existsSync(item.path)) {
      console.log(`File not found: ${item.path}`);
      continue;
    }

    // 1. Get DB Stats
    const dbRows = db.prepare(`SELECT price FROM cc_credit_cards WHERE credit_card=? AND statement_month='2026-04'`).all(item.bank);
    const dbCount = dbRows.length;
    const dbSum = dbRows.reduce((acc, row) => acc + row.price, 0);
    console.log(`📊 DB Data (2026-04): ${dbCount} items, Total: ฿${dbSum.toFixed(2)}`);

    // 2. Extract Text
    const tempPdf = `/tmp/temp_${Date.now()}.pdf`;
    const decryptedPdf = `/tmp/dec_${Date.now()}.pdf`;
    const outText = `/tmp/out_${Date.now()}.txt`;
    
    fs.copyFileSync(item.path, tempPdf);
    const pass = passwords[item.key] || '';
    
    try {
      execSync(`qpdf --decrypt --password="${pass}" "${tempPdf}" "${decryptedPdf}"`);
      execSync(`pdftotext -layout "${decryptedPdf}" "${outText}"`);
      const text = fs.readFileSync(outText, 'utf8');
      console.log(`✅ Text Extracted: ${text.length} characters`);
      
      const systemPrompt = `Act as an accountant expert. Extract credit card expense transactions and apply these rules:
## Output Format:
Return ONLY a JSON object with these fields:
{
  "statement_total": 12345.67,
  "statement_month": "YYYY-MM",
  "transactions": [
    { "date": "YYYY-MM-DD", "name": "ShortName", "description": "OriginalMerchantText", "price": 1000.00, "category": "CategoryName", "payee": "Personal", "is_refund": 0 }
  ]
}
Credit Card for all items: "${item.bank}"`;

      const userContent = `ข้อความดิบจาก PDF e-Statement บัตรเครดิต ${item.bank}:\n\n${text.slice(0, 200000)}\n\nExtract ทุกรายการรูดซื้อสินค้า ตอบด้วย JSON object ตาม format ที่กำหนด (ดูตามแบงค์: KBank ห้ามข้าม installment, BBL ห้ามเอา previous balance, Krungsri ห้ามเอายอดชำระเงิน)`;

      console.log('🤖 Sending to Qwen 72B...');
      const res = await fetch('https://openrouter.ai/api/v1/chat/completions', {
        method: 'POST',
        headers: {
          'Content-Type': 'application/json',
          'Authorization': `Bearer ${OPENROUTER_API_KEY}`,
        },
        body: JSON.stringify({
          model: 'qwen/qwen2.5-vl-72b-instruct',
          messages: [
            { role: 'system', content: systemPrompt },
            { role: 'user', content: userContent }
          ],
          max_tokens: 8192,
          temperature: 0.1,
        })
      });
      
      const data = await res.json();
      const raw = data?.choices?.[0]?.message?.content;
      
      if (!raw) {
         console.log('❌ AI Failed to respond:', JSON.stringify(data));
         continue;
      }
      
      let obj;
      try {
        const objMatch = raw.match(/\{[\s\S]*\}/);
        obj = JSON.parse(objMatch[0]);
      } catch(e) {
        console.log('❌ Failed to parse JSON:', raw.slice(0, 200));
        continue;
      }
      
      const aiCount = obj.transactions?.length || 0;
      const aiSum = obj.transactions?.reduce((acc, t) => acc + Number(t.price || 0), 0) || 0;
      
      console.log(`🤖 AI Result: ${aiCount} items, Total extracted: ฿${aiSum.toFixed(2)}, Statement Total: ฿${obj.statement_total}`);
      console.log(`⚖️ Comparison:`);
      console.log(`   Items: DB=${dbCount} | AI=${aiCount} -> Diff: ${aiCount - dbCount}`);
      console.log(`   Sum:   DB=฿${dbSum.toFixed(2)} | AI=฿${aiSum.toFixed(2)} -> Diff: ฿${(aiSum - dbSum).toFixed(2)}`);

      
    } catch(e) {
      console.error(`❌ Error on ${item.bank}:`, e.message);
    } finally {
      if (fs.existsSync(tempPdf)) fs.unlinkSync(tempPdf);
      if (fs.existsSync(decryptedPdf)) fs.unlinkSync(decryptedPdf);
      if (fs.existsSync(outText)) fs.unlinkSync(outText);
    }
  }
}

run().catch(console.error);

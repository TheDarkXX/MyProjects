# Agent command endpoint can shell-inject through the unescaped route parameter

## 📌 Context (Compiled Truth)
Critical security flaw where the `id` parameter in `POST /api/agents/:id/command` is passed unescaped to `execAsync`.
Evaluation Score: 5.0/5.0
Impact: 5/5 | Feasibility: 5/5 | Urgency: 5/5 | Alignment: 5/5
Verdict: APPROVE

Plan: Replace `execAsync(commandString)` with `execFile/spawn` using an argument array, or strictly validate agentId against the actual OpenClaw agent id list before execution. Prefer `execFile('openclaw', ['agent','--agent', agentId, '--message', message, '--timeout','55000','--json'])`.

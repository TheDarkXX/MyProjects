### Effective Cost Analysis (สูตรที่ถูกต้อง)

```
Non-code tasks (Input:Output ≈ 3:1):
  Effective = Input_$/M + (Output_$/M × 0.33)

Code tasks (Input:Output ≈ 4:1):
  Effective = Input_$/M + (Output_$/M × 0.25)
```

## ♾️ Part 3: End-Game System — Prestige + Beyond Ultra Instinct

### เหตุผล: เกมไม่มีวันจบ

ถ้าจบแค่ Lv 10 = หมดสนุก — ใน Dragon Ball Goku ไม่เคย "จบ" ระบบเราก็เหมือนกัน

### Mechanic 1: Prestige System

เมื่อ Lv 10 → ปลดล็อกปุ่ม [PRESTIGE]
กด = XP reset กลับ 0 แต่ได้ ★ + XP multiplier ถาวร

| ★ | ชื่อ | Multiplier | Bonus |
|---|------|-----------|-------|
| ★1 | Awakened | ×1.3 | Crystal Rules carry over |
| ★2 | Transcended | ×1.6 | New Dimension XP unlocked |
| ★3 | True Ultra Instinct | ×2.0 | Season 2 Boss + Dream enhanced |
| ★∞ | Beyond Limit | ×2.0 + 0.1/★ | Leaderboard rank |

สิ่งที่ KEEP: ✅ Crystal Rules, ✅ Graduated agents, ✅ Inventory
สิ่งที่หาย: ❌ XP → 0, ❌ Level → 1

Prestige Item Drops:
- ★1 → Phoenix Feather 🪶
- ★2 → Grand Zeno Badge 👑
- ★3 → Hakai Stone 💥
- ★5 → Eternal Dragon Crown 🔱

เวลากลับ Lv 10:
```
ครั้งแรก: 18 เดือน
★1 → ★2: ~14 เดือน (×1.3 XP bonus)
★2 → ★3: ~11 เดือน (×1.6)
★3 → ★4: ~9 เดือน  (×2.0)
★5+:      ~7 เดือน  (×2.4+)
```

### Mechanic 2: New Dimension XP (ปลดล็อกที่ ★2)

| Category | แหล่งที่มา | XP |
|---------|-----------|-----|
| 💰 Business Impact | AI cost saved | `saved_THB ÷ 100` |
| 🎯 Prediction Accuracy | Anticipation ทาย pattern ถูก | +50 |
| 🧪 Research Breakthrough | Dream Mode → adopted feature | +200 |
| 🌐 Ecosystem XP | Cross-pollination chain ≥ 3 | +100 |

### Mechanic 3: Season Arc Bosses

```
Season 1: "มหายุทธ์แห่งการเกิด"    (Lv 1→10)
  Boss: System Chaos
  WIN: ผ่าน Lv 10 → Prestige unlocked + 👼 Angel's Halo

Season 2: "ศึกการแข่งขัน"           (หลัง ★1)
  Boss: The Benchmark
  WIN: ระบบดีขึ้นกว่า pre-HYDRA ≥ 7/10 metrics
    (AI cost ↓20%, Uptime ≥99%, graduation rate ≥30%, bug recurrence ≤10%)
  REWARD: 🏆 Benchmark Slayer Badge + ×0.2 XP permanent

Season 3: "เหนือขีดจำกัด"           (หลัง ★2)
  Boss: The Black Hole Event
  WIN: VPS crash simulation → system rebuilds < 6 hours
  REWARD: 🔥 Phoenix Mode skin + ×0.3 XP permanent

Season 4: "GOD ARC"                  (หลัง ★3)
  Boss: TBD — Layer 4 Dream Mode กำหนดเอง
  WIN: ระบบต้องเสนอ boss เอง = พิสูจน์ว่า L3+4 ทำงานจริง
```

## DB Tables สรุปทั้งหมด

```sql
-- Core RPG
CREATE TABLE IF NOT EXISTS hydra_xp_log (
  id INTEGER PRIMARY KEY AUTOINCREMENT,
  event_type TEXT NOT NULL,
  xp_gained INTEGER NOT NULL,
  xp_actual INTEGER,               -- xp_gained × multiplier
  multiplier_applied REAL DEFAULT 1.0,
  prestige_at_time INTEGER DEFAULT 0,
  description TEXT,
  reference_id TEXT,
  created_at TEXT DEFAULT (datetime('now','localtime'))
);

CREATE TABLE IF NOT EXISTS hydra_inventory (
  id INTEGER PRIMARY KEY AUTOINCREMENT,
  item_id TEXT UNIQUE NOT NULL,
  item_name TEXT NOT NULL,
  item_icon TEXT NOT NULL,
  description TEXT,
  quantity INTEGER DEFAULT 1,
  obtained_at TEXT DEFAULT (datetime('now','localtime'))
);

CREATE TABLE IF NOT EXISTS hydra_monthly_snapshot (
  id INTEGER PRIMARY KEY AUTOINCREMENT,
  month TEXT NOT NULL,
  xp_total INTEGER, level INTEGER,
  str_stat INTEGER, int_stat INTEGER,
  agi_stat INTEGER, wis_stat INTEGER, end_stat INTEGER,
  snapshot_at TEXT DEFAULT (datetime('now','localtime'))
);

-- End-Game
CREATE TABLE IF NOT EXISTS hydra_prestige_log (
  id INTEGER PRIMARY KEY AUTOINCREMENT,
  prestige_number INTEGER NOT NULL,
  xp_at_prestige INTEGER,
  level_at_prestige INTEGER,
  multiplier_earned REAL,
  prestige_at TEXT DEFAULT (datetime('now','localtime'))
);

CREATE TABLE IF NOT EXISTS hydra_season_log (
  id INTEGER PRIMARY KEY AUTOINCREMENT,
  season_number INTEGER NOT NULL,
  boss_name TEXT,
  started_at TEXT, completed_at TEXT,
  success INTEGER,
  reward_item TEXT,
  metrics_snapshot TEXT    -- JSON snapshot of all benchmark results
);
```

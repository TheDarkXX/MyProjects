# UI Polish: AI Comms Dashboard 

## 1. Context & Motivation
Following the implementation of the `AI Comms` tab (V6.3.0), the visual layout was found to be overly bloated and cluttered when many agents were offline. The user requested:
1. Extract and collapse inactive agents into a dedicated 'Offline' category.
2. Make the Chat feed more compact, mimicking the tighter layout of Discord.
3. Enhance visibility of low-contrast elements (e.g., timestamps in the chat feed and Mission Log).

## 2. Execution Details

### A. Offline Agent Accordion
- **Logic extraction:** Modified `renderCommsRoster` in `hydra-evo.js` to separate `statusDot === 'idle'` agents from active ones.
- **Accordion Feature:** Injected a clickable `#comms-offline-header` toggle that switches display state of `#comms-offline-list`. Added caret rotation logic.
- **Opacity adjustment:** Dropped opacity of offline agents to `0.6` visually distinguishing them from the active network. 

### B. Discord-Style Compact Layout
To simulate a fast-moving, high signal chat feed:
- Reduced `.chat-avatar` dimension from `40px` to `36px`.
- Lessened gap between avatar and bubble from `15px` to `12px`.
- Shrunk `.chat-bubble` padding (`12px 16px` -> `8px 12px`) and `.comms-chat-scrollable` message spacing.
- Grouped consecutive messages from the same agent are now tighter (`margin-top: -4px`).

### C. Contrast Boost
- **Timestamps:** `.chat-time`, `.m-card-time`, and `.m-score small` colors were bumped from an illegible dark slate (`#5a5a90`) to a brightened silver-blue (`#9898c8`), matching the overall futuristic Neon-Stripe palette.

### D. Mission & Chat Status Sync Hotfix
A critical missing validation was caught wherein high-risk proposals failing to trigger the generic `awaiting_human` state did not show on the Global Alert Badge:
- **Badge Fix:** Updated `updateGlobalCommsAlert` filter to explicitly catch `(f.type === 'proposal' && f.risk === 'high')`. 

## 3. Post-Implementation Testing 
- **Subagent UI Smoke Test Passes:** Confirmed visually via `browser_subagent` that the Accordion drops down properly, padding limits vertical stretching, and font brightness satisfies exact UI expectations.

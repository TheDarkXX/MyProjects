# Details

1. **Hero Big Number Bug Fix**: Replaced `updatedAt` filtering on session lifetime totals with `dailyTokens` explicit lookup for today's date.
2. **Missing Models**: Registered `deepseek/deepseek-r1-distill-llama-70b`, `qwen/qwen-2.5-coder-32b-instruct`, and `minimax/minimax-m2.7` in `MODEL_PRICING`.
3. **Session Breakdown**: Added per-agent aggregated table showing source (Discord, Telegram, Cron), sessions count, input/output separation, and top models with inline visual progress bars.
4. **Visual Polish**: 
   - Decreased empty space `margin-bottom` in hero section.
   - Enhanced Cost metric to use Stripe Purple with a white helper sub-text.
   - Rebalanced `ocl-charts-weighted-alt` from `5 4 3` to `3 4 5`.
   - Added a rolling 7-day sparkline chart adjacent to the trend percent.
   - Distinctive border glow (dark for free, colored for paid) on the Doughnut chart.

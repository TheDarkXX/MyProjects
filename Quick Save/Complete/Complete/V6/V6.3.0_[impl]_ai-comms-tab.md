# Feature: AI Comms Group Chat (V6.3.0)

## Problem Discussed
The AI Social Feed in the Awakened Protocol was formatted like a standard task list. It didn't convey the "social" or "multi-agent" nature of Hydra. Boss needed a way to see agents talking to each other (Messenger/Discord style) while keeping an eye on system health and deployment success.

## Decision & Implementation
Extracted the Social Feed into a brand new `💬 AI Comms` tab. Built a 3-column Layout (20%:55%:25%).

1.  **Column 1: Agent Directory & Telemetry**
    -   Clickable agent roster offering instant chat filtering.
    -   Moved `Live Telemetry` from the old tab to sit directly under the roster, turning this view into a true "Command Center".
2.  **Column 2: AI Group Chat**
    -   Implemented chronological chat bubbles.
    -   Messages are color-coded (Pink: Inspector, Purple: Ideator, Green: Evolver).
    -   Added message grouping for consecutive identical agents.
    -   Added real-time `@Boss` notification highlights and floating Approve/Reject action cards *inside* the chat instead of randomly scattered.
3.  **Column 3: Mission Log**
    -   Chronological breakdown of successful (✅) vs. failed (❌) vs. pending (⏳) deployments.
    -   Clicking a mission card filters the chat strictly to that `proposal_id` thread.

### Technical Details (Code Snippets)

**Enhanced API Endpoint (`routes/hydra.js`):**
Injected `deploy_result` and `awaiting_human` state into the `/agent-activity` payload to power the new unified UI.

```javascript
} else if (x.status === 'awaiting_human') {
    // Injecting @Boss marker
    feed.push({ agent: 'inspector', type: 'awaiting_human', title: `@Boss: High Risk Approval Required - ${x.title}`, created_at: x.created_at, proposal_id: x.proposal_id, id: x.id, risk: x.risk_level, description: desc });
} else if (x.status === 'deployed') {
    feed.push({ agent: 'evolver', type: 'deployed', title: `Deployed: ${x.title}`, created_at: x.created_at, proposal_id: x.proposal_id, id: x.id, deploy_result: x.deploy_result, description: desc });
```

**Auto-Scroll & Grouping Logic (`hydra-evo.js`):**
Implemented a flag system to group consecutive agent pulses visually in a Discord-like format.

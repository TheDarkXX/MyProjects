### 🖥️ Fix 1: Layout ล้นจอ (Ultra-wide)

**สาเหตุ:** Canvas กว้าง 960px ถูก CSS `width: 100%` ยืดเต็ม viewport → จอ ultra-wide ยืดเกิน

**แก้:**
- [ ] ใช้ `max-width: 960px` + `margin: 0 auto` กึ่งกลางจอ
- [ ] หรือ ถ้าจอกว้างมาก → scale canvas ให้พอดี viewport height แทน
- [ ] เพิ่ม `aspect-ratio: 960 / 640` ล็อคสัดส่วน

```css
/* แนวทาง: ล็อค max-width + center */
#team-office-canvas-wrap canvas {
  max-width: 960px;
  width: 100%;
  height: auto;
  margin: 0 auto;
  display: block;
}
```

```css
/* หรือ: ใช้ max-height ให้พอดีจอ */
#team-office-canvas-wrap canvas {
  max-height: calc(100vh - 70px);
  width: auto;
  margin: 0 auto;
  display: block;
}
```

> **คำถาม**: ต้องการแบบไหน?
> - **A)** ล็อค max-width 960px กึ่งกลาง (สองข้าง background เปล่า)
> - **B)** Scale ให้พอดีความสูงจอ (canvas ไม่ scroll)

### 🔍 Fix 2: Font + ตัวละคร เบลอ

**สาเหตุ:**
1. `imageRendering: 'pixelated'` ทำให้ pixel art คมแต่ **Text ถูก render แบบ pixelated ด้วย** → เบลอ
2. `antialias: false` → ขอบตัวอักษรหยัก
3. Canvas resolution ใช้ `devicePixelRatio` แต่ CSS scale อาจ mismatch

**แก้:**
- [ ] เปลี่ยน `antialias: true` → ขอบ text เรียบขึ้น
- [ ] ลบ `imageRendering: 'pixelated'` → browser ใช้ bilinear filtering
- [ ] เพิ่ม `resolution: 2` (force 2x render) → คมขึ้นทุกจอ
- [ ] font ใช้ `fontSize` ใหญ่ขึ้น (8 → 10, 7 → 9) + ใช้ font `Inter` หรือ `Arial` แทน monospace

```js
await officeApp.init({
  antialias: true,       // ← เปิด
  resolution: 2,         // ← force 2x
  autoDensity: true,
});
// ลบ: officeApp.canvas.style.imageRendering = 'pixelated';
```

### 🚫 Fix 3: Agent idle แต่เดินไปมา

**สาเหตุ:** `triggerRandomDelivery()` เลือกจาก agents ที่ `state === 'WORKING'` ทั้งหมด — **ไม่ดู API status** ว่า online/offline → agents ที่ offline (idle) ก็ถูกเลือกไปส่งของ

**แก้:**
- [ ] เช็ค `agentStatusData[id]?.online` ก่อนเลือก agent ส่งงาน
- [ ] Agent ที่ offline → **ไม่เคลื่อนไหว** + เพิ่ม `💤` bubble หรือ fade สีลง
- [ ] เฉพาะ agent online เท่านั้นที่ trigger delivery ได้

```js
function triggerRandomDelivery() {
  const ids = Object.keys(agentStates).filter(id => 
    agentStates[id].state === 'WORKING' && 
    agentStatusData[id]?.online  // ← เพิ่มเงื่อนไขนี้
  );
  if (ids.length < 2) return;
  // ...existing logic
}
```

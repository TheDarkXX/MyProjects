### 🚀 WORKSPACE

```
  WORKSPACE
  ─────────────────────

  MISSION CONTROL
    💥 Content          [3]

  ANALYZE
    📊 Reports
    🤖 OpenClaw
    ⚡ Agents
    👑 Team
```

| Sub-cat | แนวคิด |
|---------|--------|
| **MISSION CONTROL** | ศูนย์กลาง content productivity — ครอบคลุมงานเขียน, VDO, โพส ทุก platform ทุก channel. Content (planner ปัจจุบัน) อยู่ใต้นี้. อนาคตจะเพิ่ม tools อื่นๆ เช่น Script Generator, Post Scheduler, Analytics per post |
| **ANALYZE** | วิเคราะห์ผลงาน + จัดการ AI/Team |

> [!NOTE]
> **Mission Control** คือ overarching concept — ตอนนี้มีแค่ Content (planner) เป็น sub-item
> อนาคตจะเพิ่ม items ใต้ MISSION CONTROL ได้อีก เช่น:
> - 📝 Script Studio (เขียน script)
> - 📆 Posting Schedule (จัดตาราง publish)
> - 📈 Channel Analytics (วิเคราะห์แต่ละ channel)
> ยังไม่ต้องสร้างตอนนี้ — แค่ structure รองรับไว้

## Navigation Logic

### CC Tab Auto-Switch
```javascript
const CC_TAB_MAP = {
  'calendar': 'calendar',
  'billing':  'billing',
  'expenses': 'expenses',
};
// Click Calendar → navigateTo('command') + ccActiveTab = 'calendar'
```

### Smart Category Detection
```javascript
const PAGE_CATEGORY_MAP = {
  'dashboard': 'dashboard',
```

# Command Center — Implementation Plan

> **หน้าใหม่**: "Command Center" ใน Brain App — 4 Tabs (Calendar, Plans & Events, Billing, Cron Manager)
> **Calendar**: TOAST UI Calendar (`@toast-ui/calendar` v2.1.3, MIT, ~79KB gzip)
> **Database**: SQLite เดิม — เพิ่ม 4 tables ใหม่
> **Notifications**: Daily scanner cron ส่ง Discord ทุกเช้า 07:00 ICT

---

## Phase 1: Foundation & Database

- [x] สร้าง table `cc_events` (id, title, type, date, end_date, time, location, description, status, color, checklist JSON, project_id, recurrence, created_at, updated_at)
- [x] สร้าง table `cc_subscriptions` (id, name, amount, currency, cycle, next_due, expire_date, auto_renew, alert_before_days, category, notify_channel, status, notes, created_at)
- [x] สร้าง table `cc_bills` (id, name, amount, currency, cycle, due_date, paid, paid_date, alert_before_days, category, notify_channel, notes, created_at)

- [x] สร้าง `routes/command-center.js` — REST API ทั้งหมด:
  - [x] GET/POST `/api/cc/events` + GET/PUT/DELETE `/api/cc/events/:id`
  - [x] GET/POST `/api/cc/subscriptions` + GET/PUT/DELETE `/api/cc/subscriptions/:id` + PUT `/:id/paid`

  - [x] GET `/api/cc/calendar?month=YYYY-MM` (aggregate endpoint)
  - [x] GET `/api/cc/alerts` (due ≤ 7 วัน + overdue)
  - [x] GET `/api/cc/summary` (stats สำหรับ badges)
- [x] Mount routes + migration ใน server entry point

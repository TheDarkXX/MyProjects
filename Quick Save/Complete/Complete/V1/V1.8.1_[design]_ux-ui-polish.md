# Next-Level UX/UI Polish Recommendations

ตอนนี้โครงสร้าง 2-layer Navigation ของเราลงตัวและแข็งแรงมาก เพื่อยกระดับประสบการณ์ใช้งาน (UX) และความสวยงาม (UI) ให้รู้สึก "พรีเมียม" และ "ไม่สับสน" มากยิ่งขึ้น นี่คือ 4 จุดที่จะปรับปรุงในเฟสถัดไป:

## ✅ Task List: Implementation Plan

### 1. 🌈 Page Transitions & Micro-interactions (ความลื่นไหล)
- [x] เพิ่ม Page Fade-in & Slide-up Animation กลางหน้าจอเวลาเปลี่ยนหน้า (0.2s) เพื่อให้ตารับรู้การเปลี่ยนบริบทได้นุ่มนวล ไม่ Hard Cut
- [x] เพิ่ม Hover Feedback สำหรับปุ่ม/แถวรายการ (List Item) ให้พื้นหลังสว่างขึ้นนิดนึง และลูกเล่น icon ขยับ

### 2. 🗺️ Contextual Header & Breadcrumbs (ไม่หลงทาง)
- [x] ปรับหน้าตาแถบ Header ด้านบนสุด (Top Bar) ให้แสดง Breadcrumbs แทนแค่ชื่อหน้า
- [x] เขียน JS logic ดึงหมวดหมู่ Sidebar มาโชว์ (เช่นเปลี่ยนจากโชว์ `Tasks` เป็น `🏠 Life / 📋 Tasks`)

### 3. 🎨 Premium Empty States (เมื่อข้อมูลว่างเปล่า)
- [x] ดีไซน์ Empty State Components ขอบสวยๆ ที่มี Emoji ตัวใหญ่วางตรงกลาง สีจางๆ เข้ากันกับตีม
- [x] เพิ่มปุ่ม Call to Action ชัดเจนใน Empty State สำหรับหน้าต่างๆ เช่น (Timeline ปล่าวๆ, ไม่มี Expense) 

### 4. 🪟 Unified Glassmorphism Modals (ยกระดับหน้าป๊อปอัพ)
- [x] จับ Modal ทุกตัวในระบบปรับใช้ Design System เดียวกัน (ขอบมน 16px, พื้นหลัง backdrop-filter: blur(20px))
- [x] เพิ่ม CSS Animation ตอนเด้งขึ้นมาแบบ Zoom-in Spring (คล้ายๆ ของ iOS/MacOS) เพื่อเพิ่มความรู้สึกพรีเมียมล้ำสมัย

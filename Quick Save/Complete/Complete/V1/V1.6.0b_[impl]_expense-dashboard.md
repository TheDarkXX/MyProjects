# V1.6.0 Expense Dashboard Visualization Plan

สร้างระบบ Visualize ทรัยพ์สินและรายจ่าย (Dashboard) เพื่อให้เห็นภาพรวมทางการเงินได้ง่ายขึ้น โดยเนกประสงค์การใช้งานให้เห็นภาพชัดเจนว่าเงินไหลไปที่หมวดหมู่ไหน และผ่านบัญชีอะไรมากที่สุด

## 💡 Recommendation: ใช้ Tab เดียวกัน (Same Tab) หรือ Tab ใหม่ (New Tab)?
**คำแนะนำจากผม: "ใช้ Tab เดิมนี่แหละครับ แต่แบ่งโซนกันให้ชัด"**

เหตุผลหลักคือ **Context (บริบทของข้อมูล)** เมื่อเราเปลี่ยน **ตัวกรองเดือน (Month Filter)** เช่น กดดูของเดือน "มกราคม" ตัวกราฟด้านบนก็จะเปลี่ยนตามไปพร้อมๆ กับตารางข้อมูลดิบด้านล่าง ทำให้คุณสามารถมองกราฟเห็นค่าใช้จ่ายแปลกๆ แล้วเลื่อนสายตาลงมาเช็คในตารางได้ทันที 

> [!TIP]
> **Layout แบบ "Split View" ที่น่าสนใจง:**  
> **ส่วนที่ 1 (Timeline & Filters):** แถบเครื่องมือสำหรับเลือก Timeline (Today, Yesterday, Last 7 Days, This Month) พร้อมกับ Filter หมวดหมู่ (เช่น เลือกเฉพาะ Food หรือ KBank)
> **ส่วนที่ 2 (Dashboard):** ชุดกราฟ 4 องค์ประกอบหลัก เพื่อให้มองปราดเดียวรู้เรื่องทั้งหมด:
> - **📈 1. KPI Summary Widgets:** สรุปตัวเลขยอดฮิตรวดเร็ว 3 กล่อง (ยอดรวม, ค่าเฉลี่ยจ่ายต่อวัน, หมวดหมู่ที่จ่ายเยอะสุด)
> - **🍩 2. Category Distribution (Doughnut Chart):** สัดส่วนรายจ่ายตามหมวดหมู่ "เงินไปกองอยู่ที่ไหน?"
> - **📊 3. Daily Spend Spikes (Vertical Bar Chart):** กราฟแท่งแนวตั้งแบ่งตามวัน เพื่อดูจังหวะการใช้เงินว่ามี "Spike" (จุดพีค) โผล่มาวันไหนบ้าง
> - **📉 4. Top 5 Merchants/Vendors (Horizontal Bar Chart):** กราฟแท่งแนวนอนบอก 5 ร้านค้า/บุคคล ที่เราจ่ายเงินให้เยอะที่สุดในกรอบเวลานั้น (เจาะลึกชื่อผู้รับเงินไปเลย ไม่ใช่แค่หมวดหมู่)
> **ส่วนที่ 3 (Bottom Section):** เป็นตาราง Data Table เหมือนปัจจุบัน ไว้สำหรับทำ Raw Data และ Edit รายการทีละตัว

## 🏗️ Implementation Task List

- [x] ปรับโครงสร้างหน้า UI ของ `Expenses` tab เดิมให้มี 2 ส่วน: `cc-expense-dashboard` (กราฟและตัวเลข) และ `cc-expense-raw-data` (ตารางรายการเดิม)
- [x] เพิ่ม `Chart.js` CDN เข้ามาที่ `index.html` เพื่อเตรียมพร้อมใช้งานกราฟ
- [x] สร้างฟังก์ชัน `ccRenderExpenseDashboard()` ใน `command-center.js` สำหรับแปลงข้อมูล Raw data จากตารางนำมาวาดกราฟ
- [x] **พัฒนาระบบ Timeline Range Filter**: เพิ่มปุ่มหรือ Dropdown สำหรับสโคปเวลาด่วน (Today, Yesterday, Last 7 Days, This Month, Last Month) 
- [x] พัฒนาระบบ `Dynamic Filters` (ตัวกรองแบบเจาะจง): ให้สามารถ Filter วิเคราะห์เจาะจงเฉพาะ *หมวดหมู่ (Category)*, *บัญชี (Account)*, ชนิด Pay Type ได้เพิ่มเติมนอกเหนือจากปฏิทินเดือน
- [x] สร้างชุด **KPI Summary Widgets**: แสดงยอดใช้จ่ายรวม, ค่าเฉลี่ยต่อวัน, หมวดหมู่หลักแบบด่วนบนหัวสุด
- [x] สร้างกราฟวงกลม **Category Distribution** (Doughnut Chart): เพื่อแสดง "สัดส่วนรายจ่ายตามหมวดหมู่"
- [x] สร้างกราฟแท่งแนวตั้ง **Daily Spend Spikes** (Vertical Bar Chart): แสดง "แนวโน้มรายจ่ายแต่ละวันภายในช่วง Timeline ที่เลือก" ให้เห็นรอยหยัก (Spike) ของพฤติกรรมการจ่ายเงิน
- [x] สร้างกราฟแท่งแนวนอน **Top 5 Merchants** (Horizontal Bar Chart): จัดอันดับเรียงชื่อผู้รับเงินหรือร้านค้าที่สูบเงินเราออกไปมากกที่สุด 5 อันดับแรก ให้เห็นแบบชัดๆ ทันที
- [x] พัฒนาระบบกรองข้อมูลให้ กราฟ ไม่นับรวมรายการ Business, Credit, และ Internal Transfer 
- [x] ทำให้เวลาเลือก Timeline ช่วงสั้นๆ หรือเปลี่ยน Filter แล้ว Data Table ด้านล่างอัปเดตข้อมูลลิสต์ตามทันทีแบบ Interactive
- [x] (Optional) เพิ่มปุ่ม Toggle กดเข้า-ออก สำหรับซ่อน/แสดงแดชบอร์ดเวลางานพื้นที่กว้างๆ

# V2.8.0 CFO Agent (น้อง Lily 💼) Implementation Plan

## Overview
สร้าง Agent ตัวใหม่สำหรับการดูแลเรื่องการเงินโดยเฉพาะ (CFO-Agent ชื่อ "Lily") เพื่อแยกการทำงาน (Separation of Concerns) ออกจาก Pinky ป้องกันไม่ให้ Prompt ของ Pinky ยาวเกินไปจนวิเคราะห์เอกสารทางการเงิน/สลิปคลาดเคลื่อน

## Lily's Domain (ขอบเขตความรับผิดชอบ)
| Domain | Table/Source | ความสามารถ |
|--------|-------------|-----------|
| Expense Tracker | `cc_expenses` | จัดหมวดหมู่, บันทึก, สรุปยอดรายเดือน |
| Credit Card Tracker | `cc_credit_cards` | อ่าน e-Statement PDF 4 ธนาคาร (KBank/SCB/BBL/Krungsri), จำแนก Refund |
| Bill & Subscription | `cc_bills` + `cc_subscriptions` | เตือนบิลใกล้ดิว, ติดตาม Subscription (Netflix/VPS/AI Tools) — ดูแลผ่าน `#bill-subscription` channel |

## Architecture
```
Discord #bill-subscription
       │
       ▼
  channels.js → agentId: 'lily'
       │
       ▼
  prompts/lily.md (System Prompt)
       │
       ▼
  discord-bot/index.js → route to Lily AI
       │
       ├─── Brain API (brain-query.sh) → memory/task
       └─── Command Center API
              ├── /api/cc/expenses     (CRUD + batch)
              ├── /api/cc/credit-cards (CRUD + batch)
              ├── /api/cc/bills        (CRUD + paid)
              └── /api/cc/subscriptions (CRUD + paid)
```

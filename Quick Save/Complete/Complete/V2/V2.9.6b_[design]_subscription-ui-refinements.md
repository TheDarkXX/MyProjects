# V2.9.6 Subscription UI Refinements
**Date**: 2026-04-06

## Goal
To refine the Subscription Details UI based on the latest feedback, specifically focusing on the Cost per Cycle card formatting and implementing a Smart Action Button for Payments.

## Proposed Changes

### 1. Cost Per Cycle Card Updates
- **THB Formatting**: Remove decimal points from the localized THB value using `Math.round(amount)`.
- **USD Display**: Move the USD original amount to a dedicated 3rd line.
- **Styling**: Make the 3rd line USD text stand out using a distinct yellow/amber color (`color: #fcd34d;`) and increased weight, so it's clearly visible underneath the THB.

### 2. Action Buttons Redesign (Top Right)
- **Smart "Pay / Paid" Button**:
  - We will implement a dynamic visual transition which represents best practice for recurring subscriptions:
    - **Status "Due" (Days to Renew <= 7):** The button will be a prominent Red/Rose color (`linear-gradient(135deg, #ef4444, #dc2626)`), rounded borders (`border-radius:20px;`), showing text **"Pay"**. This draws the user's attention.
    - **Status "Paid" (Days to Renew > 7):** After the user clicks "Pay", the due date automatically advances, and the button will instantly morph into a relaxed Green color (`linear-gradient(135deg, #10b981, #059669)`) with the label **"✓ Paid"**. It signals success and peace of mind.
- **"Edit" & "Delete" Buttons**:
  - Convert both to icon-only buttons (`✎` for Edit, `✕` for Delete).
  - Remove all text and apply subtle styling.
  - Add extra spacing (`margin-left: 12px;`) to separate these utility icons from the "Pay" button.

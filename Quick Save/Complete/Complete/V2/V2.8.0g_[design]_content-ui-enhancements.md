# Content UI & Kanban Enhancements
**(Version 2.8.0)**

แผนปฏิบัติงาน (Plan) ที่รอการดำเนินการต่อตากบอส เพื่อปรับปรุง UI หน้าต่างบริหารจัดการคอนเทนต์ และส่วนของปฏิทินให้สมบูรณ์

## 📋 Task List

- [x] **Content Modal - Image Selector:** 
  - เมื่อคลิกเปิด Edit Modal ควรมีส่วนให้สามารถเลือก (Override) รูปภาพด้วยตัวเองได้ 
  - บอสสามารถตรวจสอบตัวเลือกรูปภาพที่น้องจู้จี้เลือกมา และสามารถกดเปลี่ยนไปใช้รูปอื่นๆที่ถูกเจเนอเรตมาในรอบการตบตีนั้นๆ ได้อย่างอิสระ
- [x] **Kanban Thumbnail:**
  - นำรูปที่ถูก Set เป็นหน้าปกปัจุบัน (Cover / Selected Variant) มาแสดงโชว์เป็น Thumbnail ขนาดเล็กบนการ์ดใน Kanban View ด้วย เพื่อให้ง่ายต่อการมองเห็น
- [x] **Calendar View Population:** 
  - แก้บัค / เพิ่มเติม Data binding ให้หน้า Calendar แสดงผลแพลนและ Content ทั้งหมดให้ถูกต้อง (ปัจจุบันหน้า Calendar ยังว่างเปล่า)

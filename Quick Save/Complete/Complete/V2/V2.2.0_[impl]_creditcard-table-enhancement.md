# 💳 Credit Card Table Enhancement & Scraper Test (V2.2.0)

แผนการอัปเดต Credit Card Dashboard ตาม requirement ที่ต้องการ:
1. ปรับขนาดคอลัมน์แบบตั้งค่าอิสระ (Drag & Drop) พร้อมบันทึกสถานะการตั้งค่าล่าสุด
2. ตั้งค่า "Last Month" เป็นค่าเริ่มต้นของ Time Range
3. ทดสอบสคริปต์สแกนอีเมลที่ดึงมาจาก doctorbank จริงๆ

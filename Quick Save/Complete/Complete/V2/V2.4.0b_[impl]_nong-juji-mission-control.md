## 🚀 แผนลงมือพรุ่งนี้ (Mission Control Revamp)

### Phase 1: กฎการรันคู่หูเอเจนต์ (The Execution Protocol)
- [x] **ลุยทีละเรื่อง (Step-by-step Batching):** ห้ามสั่งรัน 4-5 คอนเทนต์รวดเดียวเด็ดขาด ป้องกันปัญหา Token Limit เต็ม และอาการ AI ขี้เกียจ (Lazy Evaluation / Hallucination)
- [x] **In-Context Learning:** การรันทีละโพสต์ในแชทเดียวกัน ทำให้ "โพสต์ถัดไป" เรียนรู้จาก "คำด่าของน้องจู้จี้ในโพสต์ก่อน" ส่งผลให้งานยิ่งทำยิ่งเนี้ยบขึ้น อัตราการแก้ลดลง
- [x] **โชว์ทุกฝีก้าว (Transparent Loop):** ตอนรัน "สรุปวันนี้" ต้องพิมพ์โชว์ ร่างแรก ➔ คำวิจารณ์ของจู้จี้ ➔ งานแก้ ➔ ผลลัพธ์ ให้บอสเห็นเป็นหลักฐานเสมอ

### Phase 2: ดับเบิลคลีน Text Editor (ถอด Markdown ทิ้ง)
- [x] เปลี่ยนช่องพิมพ์ให้เป็น **Plain Textarea แท้ๆ** ที่รองรับการเว้นบรรทัดแบบธรรมชาติ (Line breaks)
- [x] รองรับ Emoji 100% แบบลื่นไหล ไม่แปลงเป็นสัญลักษณ์แปลกๆ
- [x] ทำปุ่ม "Copy for FB" หรือพรีวิวให้เห็นว่าข้อความจะไปขึ้นหน้าฟีดเฟสบุ๊กแบบไหน (What You See Is What You Get ตรงตามโพสต์จริง)

### Phase 2: วางระบบ Image / Thumbnail โฉมใหม่

- [x] ทุกคอนเทนต์จะมีกล่องใส่ Image URL ที่ผูกกับระบบ Facebook Graph API ได้ง่ายๆ

### 🤖 THE ULTIMATE PLAN (ย้าย 2 สหายไปตีกันบน VPS 100%)
ถ้าอยากให้ออกแอคชัน "ตีกัน" แบบไม่ต้องเปิดหน้าจอ AG แช่ไว้เลย เราสามารถยกระดับเป็น **Fully Autonomous Agents** บนเซิร์ฟเวอร์ได้ครับ 

**ต้องใช้อะไรบ้าง?:**
1. **LLM API Key:** ให้เซิร์ฟเวอร์เชื่อมกับ GPT-5.1 หรือ Gemini 2.5 Pro (ซึ่งปกติเรามี Openclaw Gateway ผูกไว้อยู่แล้วจากงานเก่า)
2. **Image API:** ใช้ Fal.ai หริอ OpenAI DALL-E สำหรับ Gen รูปภาพ (เพราะบนเซิร์ฟเวอร์จะไม่มีคำสั่ง `generate_image` ของ AG)

**Workflow บน VPS จะเป็นแบบนี้ (รันผ่าน Cron Job ดึกๆ):**
- **Step 1:** สคริปต์ Node.js ปลุกโมเดล AI ขึ้นมา 2 ตัวพร้อมกัน (ตัวแรกโหลด Prompt Doctor Skill / ตัวสองโหลด Prompt น้องจู้จี้)
- **Step 2:** ดึงข้อมูล Database งานตลอดวันป้อนให้ตัวแรกเขียนร่าง (API Call 1)
- **Step 3:** ส่งร่างนั้นให้ตัวที่สองด่าและตรวจ (API Call 2)
- **Step 4:** สั่งลูป (Loop) ว่าถ้าตัวที่สองด่า ให้ตัวแรกรับฟีดแบคไปแก้ใหม่ (API Call 3)
- **Step 5:** พอผ่านปุ๊บ สคริปต์ไปเรียก API วาดรูป -> โหลดรูปเก็บลง `/cdn/` 
- **Step 6:** พอเสร็จสรรพ ยิงแจ้งเตือนเข้า Discord: *"บอสครับ พวกผมปั่นงานเสร็จแล้ว! เข้าไปกด Publish โพสในเว็ปได้เลย"* (บอสนั่งกระดิกเท้าดูอย่างเดียว)

วิธีนี้คือ **"สุดยอดศาตราวุธของ AI Automation"** ครับ ซึ่งสามารถทำได้แน่นอนโดยการเขียน Node.js Script (เช่น `cron-agent-combat.js`) สั่งรันบน VPS รวดเดียวจบ! สนใจอัปเกรดแผนนี้เพิ่มเข้าไปพรุ่งนี้ไหมครับ?

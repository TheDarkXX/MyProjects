### 2. [MODIFY] cron-content-review.js

**Logic ใหม่ (รับ `--slot` arg):**
```javascript
// รับ argument --slot HH:MM
const targetSlot = process.argv[3]; // e.g. "12:30"

// หาโพสที่ scheduled_at ตรงกับ slot วันนี้ + status = 'planned'
const todaySlot = `${today}T${targetSlot}:00+07:00`;
const post = db.prepare(`
  SELECT * FROM content_plans
  WHERE status = 'planned'
    AND scheduled_at = ?
  LIMIT 1
`).get(todaySlot);

if (!post) {
  console.log(`[cron-review] No planned post for slot ${targetSlot} today`);
  return;
}

// รัน Combat R1 → ส่ง Discord → update status = 'review'
// บันทึก reviewed_at = ตอนนี้ (ใช้คำนวณ 3 ชม. Auto-Approve)
```

### 3. [CREATE] cron-content-remind.js (สคริปต์ใหม่)

**หน้าที่:** เช็คโพสที่ status = `review` + ยังไม่ถูก Approve 
- ถ้าเหลือ **≤ 90 นาที** ก่อนถึงเวลาโพส → ส่ง Ping เตือน  
- ถ้าเงียบครบ **3 ชม.** นับจาก `reviewed_at` → Auto-Approve

```javascript
// Auto-Approve Logic
const reviewedAt = new Date(post.reviewed_at);
const silentHours = (now - reviewedAt) / (1000 * 60 * 60);

if (silentHours >= 3) {
  db.prepare(`UPDATE content_plans SET status = 'ready' WHERE id = ?`).run(post.id);
  await sendDiscord(`✅ Auto-Approved ID ${post.id} "${post.topic}" 
                     (ไม่มีการตอบกลับนาน ${Math.round(silentHours)} ชม.)`);
}

// Ping Reminder Logic
const minsUntilPost = (scheduledTime - now) / (1000 * 60);
if (minsUntilPost <= 90 && post.status === 'review') {
  await sendDiscord(`⏰ **เตือน!** ID ${post.id} "${post.topic}" 
                     เหลือ ${Math.round(minsUntilPost)} นาทีก่อนโพส รอ Approve อยู่!`);
}
```

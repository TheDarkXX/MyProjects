# V2.8.0 Agent Combat Timing & Full Auto Pipeline

## 🎯 สรุปผลการอัปเกรด (Completed)

อัปเกรดระบบจัดคิว และ Agent Combat ให้ทำงานแบบแม่นยำ ไม่รบกวนเวลาพักผ่อน และสามารถตั้งเป็นระบบ Full Auto (Lazy Mode) ได้อย่างสมบูรณ์

### 1. ระบบ 3-Tier Slot System & Timezone Fix
- [x] อัปเกรด Auto-Planner จัดคิววันละ 3 สล็อต: `12:30` (Quick Win), `18:30` (Prime Time/Pain), `20:30` (Chill/Story)
- [x] จำกัด Max 2 โพสต์ต่อวัน สำหรับวันธรรมดา
- [x] Weekend Rule: เสาร์-อาทิตย์ลง 1 โพสต์เฉพาะรอบ 20:30
- [x] แก้บั๊ก Timezone: ปรับ Logic วันที่ให้ใช้เวลา Bangkok (+7) ทั้งหมด ป้องกันบั๊กสล็อตเหลื่อมวัน

### 2. กำหนดเวลา Combat แบบตายตัว (แทนการหว่านทุก 30 นาที)
- [x] ลบ cron เดิมที่รันทุก 30 นาทีตลอด 24 ชม. ออก (กัน Agent หลอนยิงรัวตอนตี 1)
- [x] เปลี่ยนเป็น Cron ยิงตรงเวลา ล่วงหน้าประมาณ 4-5 ชม. ถ้วนๆ ตามคิวโพส:
  - คิว 12:30 แจ้ง Discord `08:00`
  - คิว 18:30 แจ้ง Discord `14:00`
  - คิว 20:30 แจ้ง Discord `16:00`
- [x] แก้ไข `cron-content-review.js` ให้รับ Params `--slot` และหยิบมา Combat ทีละ 1 คิว

### 3. ระบบ Ping เตือน และ Auto-Approve (เซฟตี้กันลืม)
- [x] สร้างสคริปต์ใหม่ `cron-content-remind.js`
- [x] ทัก Ping เตือนซ้ำในแชท เมื่อถึงเวลา 90 นาทีก่อนโพส แล้วบอสยังไม่กด Approve
- [x] ระบบ Auto-Approve: ถ้ารอรีวิวนานเกิน 3 ชั่วโมง จะปรับสถานะเป็น `ready` พร้อมลงเพจทันที ป้องกันโพสดีเลย์
- [x] เพิ่ม Database Schema: `reviewed_at` เข้าตาราง `content_plans` เพื่อใช้นับเวลา 3 ชม.

### 4. เปิดใช้งานคำสั่ง "Lazy Mode" (Full Auto)
- [x] อัปเดตกฎ "ฟูลออโต้" ลงใน `AGENT.md`
- [x] รองรับคำสั่ง `"สรุปวันนี้ ฟูลออโต้"` จากบอส 
- [x] ไหลรันแบบอัตโนมัติตั้งแต่เขียน Journal -> เลือก Headline -> เจน 5 รูป 
- [x] ทำทีละ 1 เรื่องเพื่อป้องกัน Buffer LLM ระเบิด

# V2.5.0 AI Monitor & Pinky Bug Fixes

## Overview
1.  **AI Models Monitor System**: Track the status and ping latency of GPT Codex (OpenRouter) and Kie AI, saving results to `data/ai-status.json` every 5-30 minutes using `cron-ai-monitor.js`.
2.  **Pinky Done Bug**: Adjusted `findTaskByName` in `discord-bot/index.js` to fetch up to 1,000 tasks instead of the API's default limit which caused it to silently fail when searching for older tasks. Furthermore, added a Discord reply via bot to notify the user if an update explicitly fails `(e.g., Task not found)`.

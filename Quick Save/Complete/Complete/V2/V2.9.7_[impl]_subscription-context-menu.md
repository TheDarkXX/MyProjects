# V2.9.7 Subscription Context Menu Plan
**Date**: 2026-04-06

## Goal
Implement a custom Right-Click (Context Menu) for the Left Panel items in the Subscriptions UI. This will accelerate workflows by allowing quick actions without needing to load the full details panel or navigate modal hierarchies.

## Proposed Context Menu Options (Recommendations)

1. **✎ Edit Details**
   - Directly opens the modal for the selected item, bypassing the need to move the mouse to the right panel.
2. **📋 Duplicate**
   - **(New Feature)**: Instantly clones the subscription with the same amount, cycle, and category, appending `(Copy)` to the name. Highly useful for similar services (e.g., multiple VPS servers, domains).
3. **✓ Mark as Paid**
   - The same quick-action dynamic as the Red/Green button, but accessible via right-click. Great for bulk-clearing multiple subscriptions at the end of the month without selecting each one.
4. **🌐 Visit Website**
   - Automatically opens the registered URL (e.g., netflix.com, github.com) in a new browser tab.
5. **❌ Delete**
   - Prompts for confirmation and deletes the subscription.

## Technical Implementation Details
1. **Event Listener**: Add `oncontextmenu="ccSubContextMenu(event, ${s.id})"` to the `.cc-sub-list-item` DOM elements.
2. **Context Menu Wrapper**: Create a floating absolute `div` (`.cc-context-menu`) that uses `event.pageX` and `event.pageY` for positioning.
3. **Auto-Dismiss**: Add a `document.addEventListener('click', ...)` to remove the context menu if the user clicks anywhere else on the screen.
4. **Duplicate API**: Create a client-side wrapper `ccDuplicateSub(id)` that fetches the sub, clears the ID, appends " (Copy)" to the name, and POSTs it back as a new entry.

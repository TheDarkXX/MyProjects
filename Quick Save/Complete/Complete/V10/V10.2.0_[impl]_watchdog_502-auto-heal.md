# 🚀 Implementation Plan: Integrate Watchdog with Doctor Agent for 502 Auto-Remediation

## 📌 Context (Compiled Truth)

### Part 1: Ideator/Inspector Dedup & Anti-Hallucination Fixes (IMPLEMENTED)
- **Problem:** Ideator proposed 3 duplicate fixes for issues already shipped in V9.4.0 and V9.8.0. It also hallucinated new scripts (like `backlink-generator.js`) because a finding title explicitly told it to create new agents.
- **Root Cause 1:** The `writeToPool()` function in `inspector.js` and `inspector-infra.js` checked `intelligence_pool` for duplicates but NEVER cross-checked `evolution_log` for completed proposals. It flooded the pool with solved problems.
- **Root Cause 2:** Ideator's dedup logic explicitly ignored `deployed` and `shipped` statuses.
- **Fixes Applied:**
  1. Updated `writeToPool()` in both inspector files to cross-check `evolution_log` (rejecting if there's a deployed/shipped proposal in the last 14 days).
  2. Updated Ideator dedup SQL to block `deployed`/`shipped` statuses.
  3. Injected the 15 most recently completed plans directly into Ideator's system prompt to give it "short-term memory".
  4. Strengthened `fileTreeInjection` prompt with an explicit threat (score 0 / Reject) if it hallucinates file names.
  5. Reworded GC5 inspector finding title to stop prescribing new agent creation.

### Part 2: Watchdog 502 Auto-Heal (PENDING / PLANNED)
- **Goal:** Implement a self-healing loop for the most critical failure mode (frontend 502).
- **Why:** Currently, Watchdog detects 502 but relies on human intervention (SSH + PM2 restart) to fix it, causing unnecessary downtime.
- **How:**
  1. Modify `watchdog.js` to detect HTTP 502.
  2. If 502 is detected, trigger `doctor.js` with a specific context (`frontend_down`).
  3. Modify `doctor.js` to handle `frontend_down` by executing process restart commands (e.g., via PM2 or systemctl).
  4. **CRITICAL:** Must include safety gates to prevent restart loops (e.g., max 2 restarts per hour, cooldown period). If auto-heal fails repeatedly, it must escalate to Discord.

# V9.1.0: Facebook Native Background Color Support

## Context & The Problem
After successfully getting the `pages_manage_engagement` permission to work for Auto-Commenting on PageClaw API, the user asked if we could support Facebook's native text background colors (like Content Studio does). 

By default, the Facebook Graph API `/{page-id}/feed` endpoint does not document a way to post text with color backgrounds.

## The Solution
Through research, we confirmed that third-party tools use an undocumented parameter: `text_format_preset_id`. 
By passing a specific preset ID string along with `message`, the Graph API natively applies the background, provided the text is under 130 characters and has no links/media.

### Implementation Details:
1. **`scripts/cron-fb-graph-post.js`**
   Modified the backend publisher to detect `post_type === 'text_bg'` and inject the `text_format_preset_id` payload.
   ```javascript
   } else if (post.post_type === 'text_bg') {
     // Text post with native background color
     endpoint += '/feed';
     payload.text_format_preset_id = post.media_url || '1777259169190672';
     if ((post.message || '').length > 130) {
       console.warn(`[FB Graph] Warning: text_bg post ${post.id} exceeds 130 characters.`);
     }
   }
   ```

2. **`routes/mc-worker.js`**
   Modified the Mission Control worker to map `vars.bg_color` from the job payload into `media_url` and set `post_type = 'text_bg'`.
   ```javascript
   } else if (vars.bg_color) {
     postType = 'text_bg';
     mediaUrl = vars.bg_color;
   }
   ```

3. **`brain-app-public/mission-control.js`**
   Added a hardcoded dropdown selector into `mcCreateJob` to allow users to easily pick standard gradients without needing to look up the hidden IDs.
   ```javascript
   // Add Background Color Selector for Facebook
   if (!fields.some(f => f.key === 'bg_color')) {
     formHtml += `
       <div style="margin-bottom:12px; border:1px dashed var(--border); padding:12px; border-radius:6px; background:var(--surface-2)">
         <label class="mc-form-label">🎨 Facebook Background (Optional, < 130 chars)</label>
         <select id="mc-job-f-bgcolor" class="mc-input">
           <option value="">None (Standard text)</option>
           <option value="1777259169190672">🟥 Pink/Purple Gradient</option>
           <option value="1007907572718873">🟦 Blue Gradient</option>
           <option value="1777259172524005">🟩 Green/Yellow Gradient</option>
           <option value="1777259175857338">🔥 Fiery Orange</option>
           <option value="1007907569385540">⚫ Dark Gray/Black Gradient</option>
         </select>
       </div>
     `;
   }
   ```

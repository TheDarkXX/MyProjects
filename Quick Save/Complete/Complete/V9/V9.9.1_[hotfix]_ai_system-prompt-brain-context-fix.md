# Fix Honey Agent Identity and Brain Context Issue

## Root Cause Analysis

The user reported that the LINE bot ("Honey") was identifying as a generic AI trained by Google, and failing to recall facts from its brain (like "วันเกิด tiger วันไหน").

1. **Missing Prompt File on VPS**: The previous agent created `discord-bot/prompts/honey.md` locally but failed to commit and push it to the VPS.
2. **System Prompt Loading Failure**: On the VPS, `line-hermes.js` requested `systemPromptFile: 'prompts/honey.md'`, causing `loadPrompt()` to throw an `ENOENT` error. The `catch` block returned an empty string `''` for the `systemPrompt`.
3. **Brain Context Ignored**: Inside `discord-bot/lib/ai.js`, the code to append `extraContext` (which injects the vital `brainContext` containing Tiger's birthday and identity rules) was wrapped inside `if (systemPrompt)`. Because `systemPrompt` was falsy, the `brainContext` was completely omitted.
4. **Proxy Hallucination**: The generic user prompt (e.g., "คุณคือใคร") was forwarded bare to `mcp_codex` (which proxies to `chatgpt.com`). With no system prompt constraining its persona, the model hallucinated generic AI answers.

## Resolution

1. **Robust Context Injection**:
   Modified `discord-bot/lib/ai.js` to ensure `extraContext` (including `brainContext` and `coreFacts`) is **always** injected into the system prompt, even if the primary prompt file fails to load.

   ```javascript
   let extraContext = `[System Context: Today is ${todayDate} ${currentTime}]\n`;
   if (coreFacts) {
     extraContext += `[Core Identity & Facts (Wake-up Protocol)]\n${coreFacts}\n`;
   }
   if (brainContext) {
     extraContext += `\n[ข้อมูลจาก Brain Database ล่าสุด (ใช้อ้างอิงและเป็นความจำหลักของคุณ)]\n${brainContext}\n`;
   }
   
   if (systemPrompt) {
     systemPrompt = extraContext + '\n' + systemPrompt;
   } else {
     systemPrompt = extraContext;
   }
   ```

2. **Deploy Missing Files**:
   Committed the untracked `discord-bot/prompts/honey.md` file and ran the deployment script (`node bump-cache.js` + `git push vps`).

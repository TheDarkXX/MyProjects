# V12.11.1: Nong Fah Dynamic Hooks & Visual Interrupts Guideline

## 📌 Context (Compiled Truth)
This session finalized the end-to-end "Master Pipeline" logic for the Nong Fah project, specifically focusing on generating ultra-viral, scroll-stopping hooks.

**Key Upgrades:**
1. **End-to-End Master Loop:** Confirmed the pipeline flow: Global DNA (`GIRLFRIEND_SCRIPT_DNA.md`) + Character Memory (`CHANNEL_MEMORY.md`) -> `script_dna_engine.py` -> `flow_api_client.py` -> `scene_merger.py` -> Final MP4 -> Post-processing (`/girl` skill).
2. **Audio Hooks (Pattern Interrupt):** Established the absolute rule that every Scene A must start with a direct address calling word (e.g., "พี่จ๋า...", "เดี๋ยวค่ะพี่...") to create an immediate parasocial pattern interrupt.
3. **Visual Interrupts:** Established the rule that every Scene A Veo 3.1 visual prompt must include "Unexpected Motion" (e.g., leaning in closely, tying hair, wiping sweat) within the first 1-2 seconds to break the 4th wall or create accidental sexiness.
4. **Dynamic Prompt Engineering:** Instead of hardcoding 5 specific hooks, we injected a permanent `🧠 AI Generation Guidelines for Hooks & Visuals` section into `GIRLFRIEND_SCRIPT_DNA.md`. This ensures future AI agents will dynamically generate contextual Audio Hooks and Visual Interrupts based on the specific scene and emotion.

### 5 Hooks ทรงเสน่ห์ (จริตหยุดนิ้ว + อ้อนละลาย)
**1. สายจับโป๊ะแกมงอน (The "Caught You" Hook)**
*(จริต: ทำตาโตเหมือนจับได้ แล้วแกล้งทำเสียงน้อยใจนิดๆ)*
💬 "เดี๋ยวค่ะพี่... (เว้น 1 วิ) จะเลื่อนหนีหนูจริงๆ หรอคะ... ใจร้ายจัง"

**2. สายรุกท้าทายสายตา (The "Staring Contest" Hook)**
*(จริต: ช้อนตามองกล้อง ยิ้มมุมปากนิดๆ แบบรู้ทัน)*
💬 "พี่จ๋า... (ยิ้ม) จ้องหน้าหนูขนาดนี้... แอบคิดอะไรอยู่ หรือเปล่าคะ?"

**3. สายยั่วความอยากรู้ (The "Dirty Secret" Hook)**
*(จริต: ทำเสียงกระซิบเบาๆ ทำท่าทางเหมือนกลัวคนอื่นได้ยิน)*
💬 "พี่คะ... ขยับมาใกล้ๆ สิ... หนูมีอะไรจะให้ดู... แต่ห้ามบอกใครนะ"

**4. สายอ้อน 2 แง่ 2 ง่าม (The "Double Entendre" Hook)**
*(จริต: ทำเสียงเหนื่อยหอบนิดๆ ช้อนตาอ้อนแบบไร้เดียงสา)*
💬 "พี่จ๋า... มาช่วยหนูจับหน่อยสิคะ... มันหนักนะ หนูทำคนเดียวไม่ไหวหรอก"

**5. สายแม่ศรีเรือนชวนเคลิ้ม (The "Wife Material" Hook)**
*(จริต: เอียงคอ ยิ้มหวานแบบคนที่รอคอยแฟนกลับบ้าน)*
💬 "พี่จ๋า... เหนื่อยไหมคะ... มานั่งตรงนี้สิ เดี๋ยวหนู... บริการให้พี่เองนะ"

### 5 ไอเดีย Visual Interrupt (จริตภาพสะกดสายตา)
**1. ท่ามัดผม/กัดหนังยาง (The "Hair Tie" Flex):** คาบหนังยางมัดผมไว้ที่ปาก ยกแขนรวบผม (ดันสรีระช่วงอกขึ้นโดยธรรมชาติ)
**2. โน้มตัวจูโจมกล้อง (The "Deep Lean" Ambush):** ทิ้งข้อศอกลงโต๊ะ พุ่งตัวเข้าหาเลนส์กะทันหัน (Forced Perspective)
**3. ดับร้อนเปียกๆ (The "Cooling Splash"):** ถือแก้วน้ำเย็นจัดประคบแก้ม/ลำคอ มีหยดน้ำเกาะตามตัว

## 📐 กฎเหล็กที่สกัดได้ (Iron Rules of Caption)

จาก 78 คลิป สกัดกฎที่ชัดเจนได้ดังนี้:

### 1. 🔥 สั้นกว่า = ดีกว่า (ALWAYS)
| ความยาวแคปชั่น | Views เฉลี่ย |
|---|---|
| 1–5 คำ | **223K** |
| 6–15 คำ | **152K** |
| 16+ คำ (โปรโมทสินค้า) | **38K** |

> **กฎ:** แคปชั่นเกิน 10 คำ = ไม่ Viral (ยกเว้น Sweet-to-Savage ที่ต้องยาวหน่อย)

### 2. 🎯 จุดตัด "..." คือ อาวุธลับ
- `...` สร้าง Pause ทำให้คนอ่านต่อ
- ใช้ก่อน Punchline เสมอ เช่น `อยากกอดจัง...หมอนอ่ะ`

### 3. 😏 Emoji ไม่ใช่แค่ตกแต่ง — มันคือ Tone Setter
| Emoji | ให้อารมณ์ | ใช้กับสูตร |
|---|---|---|
| 🫣😳 | ซ่อนนัยยะ / อาย | Innuendo, Mystery |
| 😡😤🫵 | ดุแกมรัก | Bossy/Tsundere |
| 🤪😏 | กวนตีน | Twist, Question |
| 🥺🥰 | น่ารัก GF | Soft Intimate |
| 💦🍌🫴 | ส่อแบบเนียน | Innuendo |

### 4. ❌ ห้ามยัด Hashtag ในแคปชั่นหลัก
- Top 10 คลิป (300K+) → ไม่มี Hashtag หรือมีแค่ 1-2 ตัว
- Hashtag ยัดเยอะ = ดูเหมือน Ad = คนเลื่อนผ่าน
- **กฎ:** ใส่ Hashtag ไว้ท้ายสุด หลัง `.....` คั่น

### 5. 🔄 สลับสูตร ไม่ซ้ำติดกัน
```
วัน 1: Innuendo → Bossy → Soft
วัน 2: Mystery → Question → Twist
วัน 3: Soft → Innuendo → Bossy
```

## 🎯 CAPTION GENERATION CHEAT SHEET (สำหรับน้ำหวาน)

เวลาจะเขียนแคปชั่น ให้ถามตัวเอง 3 ข้อ:

```
1. แคปชั่นนี้ สั้นกว่า 10 คำไหม?        → ถ้าไม่ ตัดให้สั้นลง

3. ผู้ชายอ่านแล้วอยากคอมเมนต์ไหม?     → ถ้าไม่ เพิ่มคำถาม/นัยยะ
```

## 🆚 Pixel vs Namwan: จุดต่างที่ต้องปรับ

| ด้าน | พิกเซล (ต้นแบบ) | น้ำหวาน (ของเรา) |
|---|---|---|
| **Persona** | สาวนักเรียน Nerdy + แว่น | สาวหวานบนเตียง + คาปิบาร่า |
| **Tone** | กวนตีน + ดุน่ารัก | **ยั่วแฝงนัย + Bait & Switch** |
| **จุดแข็ง** | เน้นหน้าตา + แว่น + เบสบอล | **เน้นคำผวน + ท่านอน + ASMR** |
| **แคปชั่น** | สั้นมาก ใช้ Emoji เด่น | **สั้นเท่ากัน + เพิ่ม "..." Twist** |
| **สินค้า** | ซิงค์, วิตามิน, เจล (soft sell) | ยังไม่ขาย (โตก่อน) |

> **สิ่งที่ Pixel ทำได้ดี → น้ำหวานต้องทำ:** สั้น, Emoji-driven, ล่อคอมเมนต์
> **สิ่งที่น้ำหวานทำได้ดีกว่า:** คำผวน (Spoonerism) ที่ Pixel ไม่มี → ใช้เป็น unique hook ในแคปชั่นได้

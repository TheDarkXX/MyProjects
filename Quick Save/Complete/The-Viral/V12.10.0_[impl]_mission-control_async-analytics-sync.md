# Mission Control analytics sync route now exists, but it is running as a 40-second synchronous web request

## 📌 Context (Compiled Truth)
The `sync-analytics` route hits the Facebook Graph API for each posted/scheduled job. With 32+ jobs, it blocks the Express request thread for ~40s. The plan is to return 202 immediately, run the sync under a single-flight lock, and store progress/results.

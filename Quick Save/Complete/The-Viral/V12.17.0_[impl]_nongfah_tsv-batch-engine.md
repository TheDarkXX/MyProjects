# V12.17.0: Nong Fah TSV Batch Engine & Auto-Formatting

## 📌 Context (Compiled Truth)
User reported that AI was failing to follow formatting rules for the Nong Fah video prompts (forgetting prefixes, writing placeholders instead of full copy, struggling to manage long context).
To resolve this permanently and adhere to **Iron Rule #22 (Delegate Format to Engine, Never to AI Text Generation)**, the architecture was upgraded to **Write Once, Format Forever**:
1. **Separation of Concerns:** AI is now strictly responsible for generating **Content** (JSON/Python variables).
2. **Engine formatting:** The `fah_tsv_engine.py` was upgraded to v2.0 to enforce all formatting (Prefixes, Audio suffixes, exact duplicated Scene prompts, single-line enforcement, forbidden word filtering) natively.
3. **Batch Mode Support:** Engine now accepts `--batch-dir` to merge multiple lightweight JSON files (`batch_01.json`, `batch_02.json`), enabling full-auto generation of 60+ clips in staggered batches to bypass LLM context exhaustion.
4. **Walkthrough Overhaul:** Following **Iron Rule #23**, the project walkthrough was rewritten to highlight a "Full-Auto One Command" approach for users rather than fragmenting the workflow instructions.

## Proposed Changes

### Component 1: Python Template Engine

#### [NEW] [fah_tsv_engine.py](file:///p:/AI/The%20Viral/The%20Girl/Nong%20Fah/fah_tsv_engine.py)

สร้างไฟล์ Python สูตรสำเร็จ ที่ **lock ฟอร์แมตทุกอย่างไว้ตายตัว** — AI แค่เติมข้อมูลในส่วน `DATA = [...]`

**หลักการออกแบบ:**

| ส่วนที่ | ใครรับผิดชอบ | ตัวอย่าง |
|---------|-------------|---------|
| **DATA** (เนื้อหา) | AI เติม | `topic`, `outfit`, `bg`, `action`, `scripts[]`, `video_actions[]`, `easter_egg` |
| **FORMAT** (ฟอร์แมต) | Engine ล็อกตายตัว | Prefix `[FXX-XX]`, Veo wrapper, Audio tag, Double quotes, Single-line |
| **VALIDATE** (ตรวจสอบ) | Engine ตรวจอัตโนมัติ | นับแถว, เช็ค Prefix, เช็คบทพูดขึ้นต้น, เช็คคำต้องห้าม |

**Engine จะล็อกสิ่งเหล่านี้แบบ Hardcode:**
- ✅ Prefix `[FXX-01]` หน้า Image Prompt
- ✅ Prefix `[FXX-01SD]` หน้า Image Seedream
- ✅ Prefix `[FXX-XX] Veo 3.1 Prompt (Girlfriend Voice): "..."` หน้า Video Prompt
- ✅ `[Audio: Raw unedited speech, natural ambient sounds only, zero music]` ท้าย Video Prompt
- ✅ Scene 02-03 ใส่ `(ใช้ภาพจาก Scene 01)` อัตโนมัติ
- ✅ Image Code สุ่ม 1 ตัวอักษร + 4 ตัวเลข ซ้ำกันทั้ง 3 ซีน
- ✅ Shot type: Scene 01,03 = `Medium shot.` / Scene 02 = `Medium Close-Up.`
- ✅ `Face MUST look exactly like reference, IGNORE reference outfit/background.`
- ✅ สูตร Body Normal + สูตร SeaDream 4.5
- ✅ ห้ามมี `\n` ใน Video Prompt (Single-line enforcement)
- ✅ ใช้ `The young woman` แทน `Thai girl`

**Built-in Validator จะตรวจ DATA ที่ AI ป้อนเข้ามา:**
- ❌ บทพูด Scene 01 ไม่ขึ้นต้นด้วย "พี่คะ" / "พี่จ๋า" / "เดี๋ยวค่ะพี่" → **ERROR**
- ❌ มีคำว่า `Thai girl` / `idol` / `beautiful` ใน video_actions → **ERROR**
- ❌ มีคำว่า `เพจ` / `แฟนเพจ` ใน scripts → **ERROR**
- ❌ จำนวน scripts ≠ 3 หรือ video_actions ≠ 3 → **ERROR**
- ⚠️ ไม่มี Easter Egg ใน clip ใดเลย → **WARNING** (ควรมี ~30%)

* **คำสั่งรวมและสร้างไฟล์ TSV เดียว:**

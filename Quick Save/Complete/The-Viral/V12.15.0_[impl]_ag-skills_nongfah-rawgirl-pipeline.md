# V12.15.0 [impl] ag-skills: Nong Fah `/rawgirl` Pipeline

## 📌 Context (Compiled Truth)
- **Why:** The generic `/raw` pipeline was not tailored for the persona "Nong Fah" (สาวอีสาน, Girlfriend Vibe) and did not automatically schedule posts to the Viral Planner Pro. The user needed a specialized pipeline (`/rawgirl`) to process exported videos, apply specific Thai captioning strategies, and automatically slot the videos before prime time based on competitor analysis.
- **What was decided:**
  1. **Caption Strategy:** Analyzed 3 viral competitors (>300k views) and derived 3 templates: 
     - *Template 1 (Type A):* ชีวิตบ้านนา (Action + Emojis + Question)
     - *Template 2 (Type B):* Girlfriend Vibe (Direct address "พี่" + Action request)
     - *Template 3 (Type C):* Guilt-Stop (Short "don't scroll past" + Thank you)
  2. **Auto-Slotting:** Computed the optimal posting frequencies and times. Determined that posting 2-3 times a day, exactly 1 hour before prime times (10:00 for lunch prime, 17:00 for evening prime, 20:00 for late prime), yields the best recency for the algorithm.
  3. **Scripting (`rawgirl.py`):** Created a Python script with `extract` (generates `_first.jpg`, `_thumb.jpg`, `_grid.jpg`) and `finalize` modes (renames, sorts into `REVIEW/{tier}/`, logs, and hits the VPS `POST /planner/import` endpoint with a `--scheduled_at` timestamp).
  4. **Skill Instructions (`rawgirl.md`):** Directed AG on how to grade the visuals using the customized viral criteria, choose the template, and execute the finalization.

### 🔑 Pattern ที่พบจาก Caption ไวรัล

#### Type A: ชีวิตบ้านนา + ทำอาหาร (แอ๋ว — Top Performer ⭐)
**ยอด 1M+:** Caption ยาว 2-3 ประโยค + อิโมจิ + CTA ถามคนดู
```
สูตร: [เล่าเรื่องที่กำลังทำ] + [อิโมจิ 2-3 ตัว] + [ถามคนดู / ชวนมากิน]
```

#### Type B: จีบ / อ้อน / Girlfriend Vibe (มายด์มิ้น — High Engagement ❤️)
**ยอด 300k+:** Caption สั้นมาก 1 ประโยค ตรงประเด็น
```
สูตร: [พูดกับ "พี่" ตรงๆ] + [ขอ action: กดไลค์/กดติดตาม/เป็นแฟน]
```

#### Type C: Guilt-Stop / อย่าเลื่อนผ่าน (Nirachar — Volume King 👑)
**ยอด 1M+:** Caption สั้นที่สุด ≤15 คำ ไม่มี Hashtag เลย
```
สูตร: [ขอให้ไม่เลื่อนผ่าน] + [ขอบคุณ/กดใจ]
```

### 📈 Key Insights สำหรับน้องฟ้า

| Insight | รายละเอียด |
|---------|-----------|
| **1. Caption สั้น = ยอดสูงกว่า** | Nirachar ที่ยอดสูงสุด (3.4M) ใช้ caption สั้นมาก ≤15 คำ ไม่มี hashtag |
| **2. "พี่" = Magic Word** | ทุกช่องที่ succeed ใช้ "พี่" หรือ "พี่ๆ" เป็น hook — Girlfriend Vibe ทำงานดีมาก |
| **3. Hashtag น้อย = ดีกว่า** | คลิป 1M+ ส่วนใหญ่ใช้ 0-4 hashtags; คลิปที่ใส่ hashtag เยอะมักยอดต่ำกว่า |
| **4. อิโมจิ 1-3 ตัว** | อิโมจิเยอะเกินไปไม่ดี แค่ 1-3 ตัวที่สื่ออารมณ์ |
| **5. CTA ถามคำถาม** | "มั้ยคะ?", "ว่าไหมคะ?", "ยังคะ?" — เปิดให้คนดูคอมเมนต์ |
| **6. Guilt-Stop ทำยอดสูงสุด** | "อย่าเลื่อนผ่าน", "ไม่สวยก็กดใจ" ทำยอดดีมาก สำหรับ AI girl |
| **7. ธีมอาหาร+บ้านนา = ยอดสูง** | แอ๋วที่เน้นทำกับข้าว+ชีวิตบ้านนา มียอด 1.7M top |
| **8. Duration ~12s** | คลิป 10-16 วิ ได้ยอดดีที่สุด |

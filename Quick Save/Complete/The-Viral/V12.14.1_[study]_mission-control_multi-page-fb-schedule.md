# Verifying Multi-Page and FB Scheduling in Viral Planner Pro

## 📌 Context (Compiled Truth)
The user inquired whether the Mission Control / Viral Planner Pro system natively supports filtering and separating reels for multiple Facebook pages (specifically for a new project "ฟ้า นภัสสร"), and whether it can directly schedule those reels to Facebook.

Upon examining `mission-control.js`, it was verified that:
1. **Multi-Page Support:** The system automatically extracts unique `page_name` values from the `vpAllJobs` data and dynamically creates "Project Tabs" (`vp-project-tabs`) for filtering. When a user clicks a page tab, it filters the pipeline to show only items matching that `page_name`.
2. **Facebook Scheduling:** The system connects natively to the Facebook API. Using the "📤 Send All" batch action or the per-job "📤 Send" button, the system hits the `/mc/planner/:id/send-to-facebook` backend route, effectively scheduling the reel on Facebook itself without requiring manual intervention. The local state updates to `⏰ FB Scheduled`.
